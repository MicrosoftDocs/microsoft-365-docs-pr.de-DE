# <a name="create-the-microsoft-365-defender-evaluation-environment"></a>Erstellen der Microsoft 365 Defender-Evaluierungsumgebung

Es gibt zwei gängige Möglichkeiten, diesen nächsten Schritt bei der Evaluierung auszuführen. In diesem Dokument wird davon ausgegangen, dass Sie bereits über einen M365-Produktionsmandanten verfügen und E5-Testlizenzen aktivieren, um M365 Defender in *der aktuellen Umgebung* auszuwerten. Mit einer direkten Auswertung können Sie alle Sicherheitsmethoden beim Erwerb von Lizenzen nach dem Auswertungszeitraum beibehalten.

Die zweite besteht darin, [Ihre Microsoft 365 Defender Testumgebung](setup-m365deval.md) für die Auswertung einzurichten. Da es möglicherweise nicht viele echte Signale vom Unternehmen gibt, sollten Sie sich dieser Warnung bewusst sein.

## <a name="to-activate-e5-trial-licenses-to-evaluate-microsoft-365-defender"></a>So aktivieren Sie E5-Testlizenzen, um Microsoft 365 Defender 
1. Melden Sie sich bei Ihrem vorhandenen M365-Mandantenverwaltungsportal an.
2. Wählen Sie im Navigationsmenü *"Dienste kaufen"* aus.
3. Scrollen Sie nach unten zum *Abschnitt Office 365,* und wählen Sie unter Office 365 E5 Lizenz die Schaltfläche "Details" aus.

:::image type="content" source="../../media/mdo-eval/2_mdo-eval-license-details.png" alt-text="Der Office 365 Abschnitt verfügt über eine Schaltfläche &quot;Details&quot;, auf die geklickt werden kann.":::

4. Wählen Sie den *Kostenlosen Testlink starten* aus.

:::image type="content" source="../../media/mdo-eval/3-m365-purchase-button.png" alt-text="Klicken Sie auf &quot;Kostenlose Testversion starten&quot; (es gibt eine Gebühr von 35 USD).":::

5. Bestätigen Sie Ihre Anforderung, und klicken Sie auf die Schaltfläche *"Jetzt testen".*

:::image type="content" source="../../media/mdo-eval/4_mdo-trial-order.png" alt-text="Im Bereich &quot;Auschecken, Bestellung bestätigen&quot; befindet sich die Schaltfläche &quot;Jetzt testen&quot; (für eine Office 365 E5 Testversion eines Monats für 25 Benutzer).":::
