# <a name="contact-microsoft-defender-for-endpoint-support-for-us-government-customers"></a>Kontaktieren des Microsoft Defender for Endpoint-Support für Us Government-Kunden

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Gilt für:**
- [Microsoft Defender für Endpunkt](https://go.microsoft.com/fwlink/?linkid=2154037)

Defender for Endpoint hat den Supportprozess kürzlich aktualisiert, um eine modernere und erweiterte Supporterfahrung zu bieten.

## <a name="using-the-right-portal"></a>Verwenden des richtigen Portals
Um einen Supportfall zu öffnen, müssen Sie sich bei Ihrem Microsoft Defender for Endpoint-Portal anmelden:

Umgebung | Portal-URL
:---|:---
GCC-M on Commercial | [https://securitycenter.microsoft.com](https://securitycenter.microsoft.com)
GCC-M | [https://gcc.securitycenter.microsoft.us](https://gcc.securitycenter.microsoft.us)
GCC-H | [https://securitycenter.microsoft.us](https://securitycenter.microsoft.us)
DoD | [https://securitycenter.microsoft.us](https://securitycenter.microsoft.us)

Wenn Sie sich nicht beim Portal anmelden können, können Sie auch einen Supportfall über das Telefon [öffnen.](../../business-video/get-help-support.md)

## <a name="opening-a-support-case"></a>Öffnen eines Supportfalles
Voraussetzungen und Anweisungen finden Sie unter [Kontaktieren des Microsoft Defender for Endpoint-Support](contact-support.md).
