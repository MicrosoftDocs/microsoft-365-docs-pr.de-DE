## <a name="installation-failed"></a><span data-ttu-id="a61ad-111">Installation fehlgeschlagen</span><span class="sxs-lookup"><span data-stu-id="a61ad-111">Installation failed</span></span>

<span data-ttu-id="a61ad-116">Wir zeigen dem Endbenutzer zwar keinen genauen Fehler an, aber wir behalten eine Protokolldatei mit dem Installationsfortschritt in `/Library/Logs/Microsoft/mdatp/install.log` bei.</span><span class="sxs-lookup"><span data-stu-id="a61ad-116">While we do not display an exact error to the end user, we keep a log file with installation progress in `/Library/Logs/Microsoft/mdatp/install.log`.</span></span> <span data-ttu-id="a61ad-117">Jede Installationssitzung wird an diese Protokolldatei angefügt.</span><span class="sxs-lookup"><span data-stu-id="a61ad-117">Each installation session appends to this log file.</span></span> <span data-ttu-id="a61ad-118">Sie können die `sed` letzte Installationssitzung nur verwenden:</span><span class="sxs-lookup"><span data-stu-id="a61ad-118">You can use `sed` to output the last installation session only:</span></span>

