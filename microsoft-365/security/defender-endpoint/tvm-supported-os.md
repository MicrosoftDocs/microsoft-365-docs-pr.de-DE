Bevor Sie beginnen, stellen Sie sicher, dass Sie die folgenden Betriebssystem- oder Plattformanforderungen für Bedrohungs- und Sicherheitsrisikomanagement, damit die Aktivitäten auf Ihren Geräten ordnungsgemäß berücksichtigt werden.
