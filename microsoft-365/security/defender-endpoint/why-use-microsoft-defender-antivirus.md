# <a name="better-together-microsoft-defender-antivirus-and-microsoft-defender-for-endpoint"></a>Noch besser zusammen: Microsoft Defender Antivirus und Microsoft Defender für Endpunkt

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Gilt für:**

- [Microsoft Defender für Endpunkt](/microsoft-365/security/defender-endpoint)

Microsoft Defender Antivirus ist die Schutzkomponente der nächsten Generation von [Microsoft Defender for Endpoint](/microsoft-365/security/defender-endpoint/microsoft-defender-endpoint) (Microsoft Defender for Endpoint). 

Sie können zwar eine Nicht-Microsoft-Antivirenlösung mit Microsoft Defender for Endpoint verwenden, es gibt jedoch Vorteile, Microsoft Defender Antivirus defender for Endpoint zu verwenden. Es handelt sich Microsoft Defender Antivirus nicht nur um eine hervorragende Antivirenlösung der nächsten Generation, sondern in Kombination mit anderen Defender for Endpoint-Funktionen wie [EDR](/microsoft-365/security/defender-endpoint/overview-endpoint-detection-response) und automatisierten Untersuchungen und Korrekturen [erhalten](/microsoft-365/security/defender-endpoint/automated-investigations)Sie einen besseren Schutz, der zwischen Produkten und Diensten koordiniert wird. 

## <a name="11-reasons-to-use-microsoft-defender-antivirus-together-with-microsoft-defender-for-endpoint"></a>11 Gründe für die Verwendung Microsoft Defender Antivirus zusammen mit Microsoft Defender for Endpoint

|# |Vorteil  |Warum dies wichtig ist |
|--|--|--|
|1|Freigabe von Antivirussignalen |Microsoft-Anwendungen und -Dienste teilen Signale in Ihrer Unternehmensorganisation und stellen eine stärkere einzelne Plattform zur Verfügung. Weitere [Informationen finden Sie unter Insights from the MITRE ATT&CK-based evaluation of Microsoft Defender for Endpoint](https://www.microsoft.com/security/blog/2018/12/03/insights-from-the-mitre-attack-based-evaluation-of-windows-defender-atp/). |
|2|Bedrohungsanalyse und Ihre Bewertung für Geräte |Microsoft Defender Antivirus sammelt zugrunde liegende Systemdaten, [](/microsoft-365/security/defender-endpoint/threat-analytics) die von der Bedrohungsanalyse und [der Microsoft Secure Score für Geräte verwendet werden.](/microsoft-365/security/defender-endpoint/tvm-microsoft-secure-score-devices) Dies bietet dem Sicherheitsteam Ihrer Organisation aussagekräftigere Informationen, z. B. Empfehlungen und Möglichkeiten zur Verbesserung der Sicherheitslage Ihrer Organisation. |
|3|Leistung |Microsoft Defender for Endpoint ist für die Zusammenarbeit mit Microsoft Defender Antivirus konzipiert, sodass Sie eine bessere Leistung erzielen, wenn Sie diese Angebote gemeinsam verwenden. [Bewerten Microsoft Defender Antivirus](evaluate-microsoft-defender-antivirus.md) und [Microsoft Defender for Endpoint](/microsoft-365/security/defender-endpoint/evaluate-mde).|
|4 |Details zu blockierter Schadsoftware |Weitere Details und Aktionen für blockierte Schadsoftware finden Sie unter Microsoft Defender Antivirus und Microsoft Defender for Endpoint. [Verstehen von Schadsoftware & anderen Bedrohungen](/windows/security/threat-protection/intelligence/understanding-malware).|
|5 |Netzwerkschutz |Das Sicherheitsteam Ihrer Organisation kann Ihr Netzwerk schützen, indem bestimmte URLs und IP-Adressen blockiert werden. [Schützen Sie Ihr Netzwerk](/microsoft-365/security/defender-endpoint/network-protection).|
|6 |Blockieren von Dateien |Das Sicherheitsteam Ihrer Organisation kann bestimmte Dateien blockieren. [Beenden und Isolieren von Dateien in Ihrem Netzwerk](/microsoft-365/security/defender-endpoint/respond-file-alerts#stop-and-quarantine-files-in-your-network).|
|7 |Attack Surface Reduction |Das Sicherheitsteam Ihrer Organisation kann Ihre Sicherheitsrisiken (Angriffsoberflächen) verringern und Angreifern weniger Möglichkeiten zum Ausführen von Angriffen geben. Bei der Reduzierung der Angriffsfläche wird cloudbasierter Schutz für eine Reihe von Regeln verwendet. [Verschaffen Sie sich einen Überblick über die Reduzierung der Angriffsfläche.](/microsoft-365/security/defender-endpoint/overview-attack-surface-reduction)|
|8 |Überwachungsereignisse |Überwachungsereignissignale sind in den [EDR verfügbar.](/microsoft-365/security/defender-endpoint/overview-endpoint-detection-response) (Diese Signale sind bei Nicht-Microsoft-Antivirenlösungen nicht verfügbar.) |
|9 |Geografische Daten |Gemäß ISO 270001 und Datenaufbewahrung werden geografische Daten gemäß der ausgewählten geografischen Souveränität Ihrer Organisation bereitgestellt. Siehe [Complianceangebote: ISO/IEC 27001:2013 Information Security Management Standards](/microsoft-365/compliance/offering-iso-27001). |
|10|Dateiwiederherstellung über OneDrive |Wenn Sie Microsoft Defender Antivirus mit Office 365 [verwenden](/Office365/Enterprise)und Ihr Gerät von Ransomware angegriffen wird, sind Ihre Dateien geschützt und wiederherstellbar. [OneDrive Dateien wiederherstellen und Windows Defender Ransomware-Schutz einen Schritt weiter .](https://techcommunity.microsoft.com/t5/Microsoft-OneDrive-Blog/OneDrive-Files-Restore-and-Windows-Defender-takes-ransomware/ba-p/188001)|
|11|Technischer Support |Wenn Sie Microsoft Defender for Endpoint zusammen mit Microsoft Defender Antivirus verwenden, haben Sie ein Unternehmen, um technischen Support zu fordern. [Behandeln von Dienstproblemen und](/microsoft-365/security/defender-endpoint/troubleshoot-mde) [Überprüfen von Ereignisprotokollen und Fehlercodes mit Microsoft Defender Antivirus](troubleshoot-microsoft-defender-antivirus.md). |
