## <a name="microsoft-defender-antivirus-your-next-generation-protection"></a>Microsoft Defender Antivirus: Ihr Schutz der nächsten Generation

Microsoft Defender Antivirus ist die Schutzkomponente der nächsten Generation von Microsoft Defender for Endpoint. Dieser Schutz vereint maschinelles Lernen, Big-Data-Analyse, eingehende Untersuchungen zur Bedrohungsabwehr und die Microsoft-Cloud-Infrastruktur zum Schutz von Geräten in Ihrer Unternehmensorganisation. Ihre Schutzdienste der nächsten Generation umfassen die folgenden Funktionen:

- [Verhaltensbasierter, heuristischer](configure-protection-features-microsoft-defender-antivirus.md)und Echtzeit-Antivirusschutz, der die kontinuierliche Überprüfung mithilfe der Datei- und Prozessverhaltensüberwachung und anderer Heuristiken (auch als Echtzeitschutz bekannt) umfasst.  Es umfasst auch das Erkennen und Blockieren von Apps, die als unsicher, aber möglicherweise nicht als Schadsoftware erkannt werden.
- [In der Cloud zugestellter Schutz,](cloud-protection-microsoft-defender-antivirus.md)der nahezu sofortige Erkennung und Blockierung neuer und neuer Bedrohungen umfasst.
- [Dedizierte Schutz- und Produktupdates,](manage-updates-baselines-microsoft-defender-antivirus.md)die Updates im Zusammenhang mit der Aktualisierung von Microsoft Defender Antivirus enthalten.

## <a name="try-a-demo"></a>Testen Sie eine Demo!

- Block at first sight (BAFS)-Schutz
- Schutz potenziell unerwünschter Anwendungen (PUA)
