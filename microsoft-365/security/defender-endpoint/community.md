# <a name="access-the-microsoft-defender-for-endpoint-community-center"></a>Zugreifen auf das Microsoft Defender for Endpoint Community Center

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**Gilt für:**
- [Microsoft Defender für Endpunkt](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

>Möchten Sie Defender for Endpoint erleben? [Registrieren Sie sich für eine kostenlose Testversion.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-assignaccess-abovefoldlink)

Das Defender for Endpoint Community Center ist ein Ort, an dem Communitymitglieder Erfahrungen über das Produkt lernen, zusammenarbeiten und teilen können. 

Es gibt mehrere Bereiche, die Sie erkunden können, um mehr über bestimmte Informationen zu erfahren:
- Ankündigungen 
- Neuerungen
- Informationen zu Bedrohungen


Es gibt verschiedene Möglichkeiten, wie Sie auf das Community Center zugreifen können:
- Wählen Sie im Navigationsbereich Microsoft Defender Security Center die Option **Community Center aus.**  Eine neue Browserregisterkarte wird geöffnet und führt Sie zur Defender for Endpoint Tech Community-Seite. 
- Zugreifen auf die Community über die [Microsoft Defender for Endpoint Tech Community-Seite](https://techcommunity.microsoft.com/t5/Windows-Defender-Advanced-Threat/ct-p/WindowsDefenderAdvanced)


Sie können Unterhaltungen, die in der Community veröffentlicht wurden, sofort anzeigen und lesen. 

Um die vollständige Erfahrung in der Community zu erhalten, z. B. Beiträge kommentieren zu können, müssen Sie der Community beitreten. Weitere Informationen zum Einstieg in die Microsoft Tech Community finden Sie unter [Microsoft Tech Community: Getting Started](https://techcommunity.microsoft.com/t5/Getting-Started/Microsoft-Tech-Community-Getting-Started-Guide/m-p/77888#M15).
