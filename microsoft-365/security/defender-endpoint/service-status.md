# <a name="check-the-microsoft-defender-for-endpoint-service-health"></a>Überprüfen der Integrität des Microsoft Defender for Endpoint-Diensts

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Gilt für:**
- [Microsoft Defender für Endpunkt](https://go.microsoft.com/fwlink/?linkid=2154037)



>Möchten Sie Microsoft Defender for Endpoint erleben? [Registrieren Sie sich für eine kostenlose Testversion.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-servicestatus-abovefoldlink)

**Der Dienststatus** enthält Informationen zum aktuellen Status des Defender for Endpoint-Diensts. Sie können überprüfen, ob der Dienstzustand fehlerfrei ist oder aktuelle Probleme auftreten. Wenn Probleme auftreten, werden Informationen wie der Zeitpunkt des Auftretens des Problems, die vorläufige Ursache und die erwartete Lösungszeit ermittelt.

Außerdem werden Informationen zu behobenen verlaufshistorischen Problemen sowie Details wie Datum und Uhrzeit der Problem behoben. Wenn keine Probleme mit dem Dienst auftreten, wird ein fehlerfreier Status angezeigt.

Sie können Details zum Dienstzustand anzeigen, indem  Sie im Dashboard für Sicherheitsvorgänge auf die Kachel klicken oder im Navigationsbereich das Menü Dienstintehzustand auswählen. 

Auf der Seite Dienstintegtezustandsdetails sind die folgenden Registerkarten verfügbar: 

- **Aktueller Status**
- **Statusverlauf**

## <a name="current-status"></a>Aktueller Status
Auf **der Registerkarte Aktueller Status** wird der aktuelle Status des Defender for Endpoint-Diensts angezeigt. Wenn der Dienst reibungslos ausgeführt wird, wird ein fehlerfreier Dienstzustand angezeigt. Wenn Probleme auftreten, werden die folgenden Dienstdetails angezeigt, um ihnen einen besseren Einblick in das Problem zu verschaffen:

- Datum und Uhrzeit, zu dem das Problem erkannt wurde
- Eine kurze Beschreibung des Problems
- Updatezeit
- Zusammenfassung der Auswirkungen
- Vorläufige Ursache
- Nächste Schritte
- Erwartete Auflösungszeit

Updates zum Fortschritt eines Problems werden auf der Seite angezeigt, sobald das Problem behoben wird. Es werden Updates zu Informationen wie einer aktualisierten Schätzungsauflösungszeit oder den nächsten Schritten zu sehen sein.

Wenn ein Problem behoben wird, wird es auf der Registerkarte **Statusverlauf** aufgezeichnet.

## <a name="status-history"></a>Statusverlauf
Die **Registerkarte Statusverlauf** gibt alle historischen Probleme wieder, die angezeigt und behoben wurden. Sie sehen Details zu den behobenen Problemen zusammen mit den anderen Informationen, die während der Lösung enthalten waren.
