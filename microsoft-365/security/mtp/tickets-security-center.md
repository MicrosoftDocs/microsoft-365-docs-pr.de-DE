Das [Microsoft 365 Security Center](overview-security-center.md) wurde um die Möglichkeit erweitert, Tickets in ServiceNow nativ zu erstellen und nachverfolgt zu werden. [Weitere Informationen zu ServiceNow](https://www.servicenow.com/)
