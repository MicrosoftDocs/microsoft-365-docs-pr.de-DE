### <a name="original-query"></a><span data-ttu-id="6ee9b-175">Ursprüngliche Abfrage</span><span class="sxs-lookup"><span data-stu-id="6ee9b-175">Original query</span></span>
<span data-ttu-id="6ee9b-176">Die folgende Abfrage verwendet `DeviceAlertEvents` in Microsoft Defender ATP, um die Warnungen zu erhalten, die _powershell.exe_beinhalten:</span><span class="sxs-lookup"><span data-stu-id="6ee9b-176">The following query uses `DeviceAlertEvents` in Microsoft Defender ATP to get the alerts that involve _powershell.exe_:</span></span>

### <a name="modified-query"></a><span data-ttu-id="6ee9b-177">Geänderte Abfrage</span><span class="sxs-lookup"><span data-stu-id="6ee9b-177">Modified query</span></span>
