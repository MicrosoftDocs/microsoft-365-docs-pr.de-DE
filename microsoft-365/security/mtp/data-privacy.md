# <a name="microsoft-threat-protection-data-security-and-privacy"></a>Microsoft Threat Protection Datensicherheit und Datenschutz

**Gilt für:**
- Microsoft Threat Protection

[!INCLUDE [Prerelease information](../includes/prerelease.md)]

Durch die Verwendung der Microsoft Threat Protection Preview stimmen Sie den folgenden Bedingungen zu:

Ihre anwendbaren Kundendaten (wie in den Online Service-Nutzungsbedingungen ("OST") definiert) werden von anderen Microsoft-Diensten in Microsoft Threat Protection und von Microsoft Threat Protection zurück in anwendbare Microsoft-Dienste übertragen. Für die Dauer dieser Preview befolgt die Verwendung Ihrer Kundendaten in Microsoft Threat Protection die Datenverarbeitungsstandards und -verpflichtungen für Microsoft Defender Advanced Threat Protection (Microsoft Defender ATP). Sie akzeptieren, dass solche Verpflichtungen von den Diensten abweichen können, von denen diese Kundendaten übertragen werden. Darüber hinaus werden Kundendaten, die in Microsoft Threat Protection gespeichert sind, in dem Geo gespeichert, das Sie für die Speicherung Ihrer Microsoft Defender ATP-Kundendaten ausgewählt haben. Diese können von den Geo-Daten abweichen, die Sie in Verbindung mit anderen Diensten ausgewählt haben. Microsoft wird die Kundendaten nicht außerhalb eines solchen Geo übertragen, es sei denn, dies ist im Abschnitt "Datenstandort" des Microsoft Trust Centers im [Microsoft Trust Center](https://www.microsoft.com/trust-center) angegeben.

Während Sie festlegen, welche Nutzer auf Microsoft Threat Protection zugreifen dürfen, lässt Microsoft Threat Protection derzeit keine Role Based Access Control (RBAC) zu. Sie akzeptieren, dass wenn Microsoft Threat Protection von einem Microsoft-Dienst, der RBAC unterstützt, Daten erhält, die Zugriffsebenen in diesem Dienst nicht in Microsoft Threat Protection gelten.


Weitere Informationen zur Datenspeicherung und zum Datenschutz der jeweiligen Produkte finden Sie unter:

- [Microsoft Cloud App Security Datensicherheit und Datenschutz](https://docs.microsoft.com/cloud-app-security/cas-compliance-trust)
- [Office 365 Datenschutz, Sicherheit und Transparenz](https://docs.microsoft.com/office365/servicedescriptions/office-365-platform-service-description/privacy-security-and-transparency#advanced-threat-protection)
