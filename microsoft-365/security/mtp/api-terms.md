## <a name="apis"></a>APIs

 Microsoft 365 Defender-APIs unterliegen [der Microsoft-API-Lizenz und den Nutzungsbedingungen](https://docs.microsoft.com/legal/microsoft-apis/terms-of-use).

### <a name="throttling-limits"></a>Einschränkungsgrenzwerte

Jede API verfügt über eigene Einschränkungs Grenzwerte.<br>
Es gibt Einschränkungs Grenzwerte nach Stunden und Minuten.<br>
Beispielsweise sind die Einschränkungs Grenzwerte für die Vorfall-API: 50 Aufrufe pro Minute und 1500 pro Stunde.<br>
Der HTTP-Antwortstatuscode für den Fall einer Drosselung wird 429.
