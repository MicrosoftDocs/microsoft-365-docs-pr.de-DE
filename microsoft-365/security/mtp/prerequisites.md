# <a name="microsoft-threat-protection-prerequisites"></a>Voraussetzungen für Microsoft Threat Protection

**Gilt für:**
- Microsoft Threat Protection

[!INCLUDE [Prerelease information](../includes/prerelease.md)]

Hier lernen Sie die Lizenz-, Hardware- und Softwareanforderungen sowie andere Konfigurationseinstellungen zum Ausführen und Verwenden von Microsoft 365 Security kennen.

## <a name="licensing-requirements"></a>Lizenzierungsanforderungen
Für Microsoft 365 Security ist eine der folgenden Lizenzen erforderlich:

- Microsoft 365 E5 
- Office 365 E5, Enterprise Mobility + Security E5 und Windows E5

Sie können diese Lizenzen auf der Seite [Microsoft 365 für Unternehmen](https://www.microsoft.com/en-us/microsoft-365/enterprise) erwerben.

### <a name="check-your-existing--licenses"></a>Überprüfen vorhandener Lizenzen
Im Microsoft 365 Admin Center unter [admin.microsoft.com](https://admin.microsoft.com/) können Sie Ihre vorhandenen Lizenzen anzeigen. Navigieren Sie im Admin Center zu **Abrechnung** > **Lizenzen**.

Sie müssen entweder **Abrechnungsadministrator**- oder der **Globaler Leser**-[Rolle in Azure AD](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#available-roles) angehören, um Lizenzierungsinformationen anzeigen zu können. Wenn Sie Probleme beim Zugriff haben, wenden Sie sich an einen globalen Administrator.  

## <a name="browser-requirements"></a>Browseranforderungen
Der Zugriff auf das Microsoft 365 Security Center erfolgt über einen Browser. Internet Explorer und Microsoft Edge werden unterstützt. Alle HTML5-kompatiblen Browser werden ebenfalls unterstützt.
