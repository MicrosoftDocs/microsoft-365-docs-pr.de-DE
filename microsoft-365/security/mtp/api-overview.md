# <a name="overview-of--microsoft-threat-protection-apis"></a>Übersicht über Microsoft Threat Protection-APIs

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Gilt für:**
- Microsoft Threat Protection


>[!IMPORTANT] 
>Einige Informationen beziehen sich auf Vorabversionen von Produkten, die vor der kommerziellen Veröffentlichung noch erheblich geändert werden können. Microsoft übernimmt mit diesen Informationen keinerlei Gewährleistung, sei sie ausdrücklich oder konkludent.

Die Microsoft Threat Protection-Lösung ist auf einer integrationsfähigen Plattform aufgebaut. 

Die Microsoft Threat Protection-APIs auf Lop-Ebene ermöglichen es Ihnen, Workflows basierend auf den freigegebenen Vorfällen und erweiterten Jagd Tabellen zu automatisieren.

- **Kombinierte Vorfälle-Warteschlange** – unterstützt Sicherheitsexperten bei der Fokussierung auf das, was kritisch ist, indem sichergestellt wird, dass der vollständige Angriffsbereich und die betroffenen Objekte zusammengefasst werden und mit der Vorfall-API rechtzeitig aufgetaucht sind.

- **Produktübergreifende Bedrohungs Jagd** – Sicherheitsteams können Ihr einzigartiges organisatorisches Wissen nutzen, um nach Zeichen von Kompromissen zu suchen, indem Sie Ihre eigenen benutzerdefinierten Abfragen über APIs über die von den verschiedenen Schutzprodukten gesammelten Rohdaten erstellen. 

Zusätzlich zu diesen APIs stehen für jedes der verschiedenen Schutzprodukte zusätzliche APIs zur Verfügung, die Ihnen bei der Innovation auf der Grundlage der einzelnen Produktfunktionen helfen.
