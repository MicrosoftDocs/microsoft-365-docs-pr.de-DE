## <a name="how-to-use-the-csl-file"></a><span data-ttu-id="24b6e-137">Verwenden der CSL-Datei</span><span class="sxs-lookup"><span data-stu-id="24b6e-137">How to use the CSL file</span></span>
<span data-ttu-id="24b6e-138">Bevor Sie eine Folge starten, greifen Sie auf gitHub auf die entsprechende [Kusto-CSL-Datei](https://github.com/microsoft/Microsoft-threat-protection-Hunting-Queries/tree/master/Webcasts/TrackingTheAdversary) zu, und kopieren Sie den Inhalt in den Abfrageeditor für die erweiterte Suche.</span><span class="sxs-lookup"><span data-stu-id="24b6e-138">Before starting an episode, access the corresponding [Kusto CSL file on GitHub](https://github.com/microsoft/Microsoft-threat-protection-Hunting-Queries/tree/master/Webcasts/TrackingTheAdversary) and copy its contents to the advanced hunting query editor.</span></span> <span data-ttu-id="24b6e-139">Während Sie sich eine Folge ansehen, können Sie die kopierten Inhalte verwenden, um dem Sprecher zu folgen und Abfragen ausführen.</span><span class="sxs-lookup"><span data-stu-id="24b6e-139">As you watch an episode, you can use the copied contents to follow the speaker and run queries.</span></span> 

<span data-ttu-id="24b6e-140">Der folgende Auszug aus einer CSL-Datei zeigt einen umfassenden Satz von Anleitungen, die als Kommentare mit gekennzeichnet `//` sind.</span><span class="sxs-lookup"><span data-stu-id="24b6e-140">The following excerpt from a CSL file shows a comprehensive set of guidance marked as comments with `//`.</span></span>

```kusto

// Contains
// - Account information associated with the logon
// - The device which the account logged onto
// - The process which performed the logon
// - Network information (for network logons)
// - Timestamp
```

// DeviceLogonEvents
