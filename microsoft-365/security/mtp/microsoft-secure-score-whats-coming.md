# <a name="whats-coming-to-microsoft-secure-score"></a>What's coming to Microsoft Secure Score

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Die Microsoft Secure Score finden Sie https://security.microsoft.com/securescore im [Microsoft 365 Security Center.](overview-security-center.md)

## <a name="proposed-changes"></a>Vorgeschlagene Änderungen

We're making some changes in the near future to make [Microsoft Secure Score](microsoft-secure-score.md) a better representative of your security posture and improve usability. Ihre Bewertung und die maximal mögliche Punktzahl können sich ändern.

Derzeit sind keine Änderungen geplant.
