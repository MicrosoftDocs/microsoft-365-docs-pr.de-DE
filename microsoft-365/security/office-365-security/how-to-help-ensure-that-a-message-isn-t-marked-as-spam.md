# <a name="how-to-help-ensure-that-a-message-isnt-marked-as-spam"></a>Sicherstellen, dass eine Nachricht nicht als Spam gekennzeichnet wird

Als Exchange Online- oder Exchange Online Protection-Administrator (EOP) können Sie sicherstellen, dass eine E-Mail-Nachricht, die den Dienst durchläuft, nicht als Spam gekennzeichnet wird. Lesen Sie die aktualisierte Version der Tipps und Verfahren, um dies zu erreichen, indem Sie [sichere Absenderlisten in Office 365 erstellen](create-safe-sender-lists-in-office-365.md).
