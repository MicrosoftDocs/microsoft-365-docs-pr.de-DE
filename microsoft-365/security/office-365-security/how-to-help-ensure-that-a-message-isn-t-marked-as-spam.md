# <a name="how-to-help-ensure-that-a-message-isnt-marked-as-spam"></a>Sicherstellen, dass eine Nachricht nicht als Spam gekennzeichnet wird

Als Exchange Online- oder Exchange Online Protection-Administrator (EOP) können Sie sicherstellen, dass eine E-Mail-Nachricht, die den Dienst durchläuft, nicht als Spam gekennzeichnet wird. Lesen Sie die aktualisierte Version der Tipps und Verfahren, um dies zu erreichen, [indem Sie verhindern, dass gute e-Mails in Office 365 als Spam markiert werden](https://docs.microsoft.com/microsoft-365/compliance/prevent-email-from-being-marked-as-spam).
