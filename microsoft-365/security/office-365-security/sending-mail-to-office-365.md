# <a name="sending-mail-to-microsoft-365"></a>Senden von E-Mails an Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]


Diese Artikel helfen externen Absendern, ihren Ruf zu verbessern und ihre Fähigkeit zu erhöhen, E-Mails an Benutzer in Microsoft 365 zu senden. Sie enthalten auch einige Informationen dazu, wie Sie Junk-E-Mails und Phishingversuche melden können, auch wenn Sie selbst kein Microsoft 365-Benutzer sind.

Wenn Sie kein Kunde sind, aber versuchen, E-Mails an eine Person zu senden, bei der es sich um einen Kunden handelt, sind Sie am richtigen Ort. Wenn Sie administratorisch sind und Hilfe bei der Bekämpfung von Spam benötigen, ist dies nicht der richtige Abschnitt für Sie. Wechseln Sie stattdessen zu [Antispam- und Ansoftwareschutz in Microsoft 365.](anti-spam-and-anti-malware-protection.md)
