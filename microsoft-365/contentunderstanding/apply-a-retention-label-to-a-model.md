# <a name="apply-a-retention-label-to-a-document-understanding-model-preview"></a>Anwenden einer Aufbewahrungs Bezeichnung auf ein Dokument grundlegendes Modell (Vorschau)

> [!Note] 
> Der Inhalt dieses Artikels ist für Project Cortex private Preview. [Erfahren Sie mehr über Project Cortex](https://aka.ms/projectcortex).

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSoL]

</br>

Sie können eine [Aufbewahrungs Bezeichnung](https://docs.microsoft.com/microsoft-365/compliance/retention?view=o365-worldwide#retention-labels) ganz einfach auf ein Dokument Verständnis Modell in Project Cortex anwenden.

Mit Aufbewahrungs Bezeichnungen können Sie Aufbewahrungseinstellungen auf die Dokumente anwenden, die Ihr Dokument Verständnis Modell identifiziert.  Möglicherweise möchten Sie, dass Ihr Modell nicht nur Vertrags Erneuerungs Dokumente identifiziert, die in Ihre Dokumentbibliothek hochgeladen werden, sondern auch ein Aufbewahrungs auf diese anwenden, damit diese Dokumente niemals gelöscht werden können.

Sie können eine Aufbewahrungs Bezeichnung auf Ihr Dokument Verständnis Modell über Ihre Modelleinstellungen anwenden. 

> [!Note]
> Damit Aufbewahrungs Bezeichnungen für Ihr Modell zur Verfügung stehen, müssen Sie [im Microsoft 365 Compliance Center erstellt und veröffentlicht](https://docs.microsoft.com/microsoft-365/compliance/create-apply-retention-labels?view=o365-worldwide#how-to-create-and-publish-retention-labels)werden.
