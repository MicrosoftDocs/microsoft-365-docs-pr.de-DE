> [!NOTE]
> <span data-ttu-id="cdb7d-110">Ein SharePoint-Administrator kann eine Inhaltscenter-Website erstellen, wie er auch eine beliebige andere [SharePoint-Website](https://docs.microsoft.com/sharepoint/create-site-collection) über den Bereitstellungsbereich der Admin Center-Website erstellen würde.</span><span class="sxs-lookup"><span data-stu-id="cdb7d-110">A SharePoint admin can create a content center site like they would [create any other SharePoint site](https://docs.microsoft.com/sharepoint/create-site-collection) through the admin center site provisioning panel.</span></span>

<span data-ttu-id="cdb7d-111">So erstellen Sie ein neues Inhaltscenter:</span><span class="sxs-lookup"><span data-stu-id="cdb7d-111">To create a new content center:</span></span>

> [!NOTE] 
> <span data-ttu-id="cdb7d-117">Sie können eine inhaltscenter-Website für die Darstellung in einer der verfügbaren Sprachen auswählen.
