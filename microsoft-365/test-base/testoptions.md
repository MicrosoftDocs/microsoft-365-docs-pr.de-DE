# <a name="step-5-choose-your-test-options"></a>Schritt 5: Wählen Sie Ihre Testoptionen aus. 

Die ```Test Options``` Registerkarte ist für Benutzer, die Funktionstests durchführen möchten, um anzugeben, wann der Windows Update-Patch in der Abfolge der Ausführung ihrer funktionalen Testskripts angewendet werden soll.

![Abbildung der Testoptionen. Out-of-Box- oder Funktionstests](Media/testoptions.png)

Wählen Sie _**"Überprüfen"**_ aus, um zur nächsten Registerkarte zu navigieren und die ausgewählten Testoptionen zu überprüfen.
