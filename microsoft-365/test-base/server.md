# <a name="windows-server-application-testing"></a>Windows Serveranwendungstests 

Mit Test Base for Microsoft 365 können Sie jetzt Ihre Anwendungen mit Windows Server 2016 und 2019 überprüfen, einschließlich Server Core!

To get started with validating your uploaded applications against pre-release updates for Windows Server 2016 and 2019 operating systems on Test Base for Microsoft 365, kindly be to the following steps:

1.   Melden Sie sich bei unserem Self-Service-Onboarding-Portal an. Wählen Sie im linken Navigationsmenü ```Upload new package``` unter und füllen Sie die ```Package catalogue``` Testdetails aus.

2.  Wählen Sie ```Security updates``` als Betriebssystemupdatetyp aus:

![Auswählen von Sicherheitsupdates](Media/selecting-security-updates.png)

3. Wählen Sie unter zu testenden Betriebssystemversionen die entsprechenden Betriebssystemversionen aus. Sie können Windows Versionen des Serverbetriebssystems oder eine Kombination aus Server- und Clientbetriebssystemversionen auswählen.

![Auswählen der Betriebssystemversion](Media/selecting-OS-versions.png)

4. Geben Sie weitere erforderliche Informationen an, überprüfen Sie die bereitgestellten Details, und laden Sie Ihr Anwendungspaket hoch. Nach dem Hochladen können Sie den Paketstatus auf der Menüregisterkarte "Pakete verwalten" anzeigen.


5. Wenn Sie Testergebnisse und Einblicke aus der Überprüfung Ihrer Anwendung für Sicherheitsupdates vor der Veröffentlichung für Windows Server 2016 und 2019 anzeigen möchten, wechseln Sie zur Seite "Testzusammenfassung" oder zur Seite mit den Ergebnissen des Sicherheitsupdates.

![Anzeigen von Testergebnissen](Media/access-test-results.png)

Wechseln Sie zum nächsten Artikel, um mit **Funktionstests** zu beginnen.
