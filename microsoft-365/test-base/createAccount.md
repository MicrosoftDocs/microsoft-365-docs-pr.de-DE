# <a name="step-1-create-a-test-base-account"></a>Schritt 1: Erstellen eines Testbasiskontos

Wenn Sie kein Azure-Abonnement haben, erstellen Sie ein [kostenloses Konto,](https://azure.microsoft.com/en-us/free/) bevor Sie beginnen.

## <a name="enter-details-for-test-base-account"></a>Geben Sie Details für das Testbasiskonto ein.
 
1. Suchen Sie im Azure-Portal nach **"Test Base".**

![Erstellen eines Suchbilds für ein Testbasiskonto](Media/CreateTestAccount1.png)

2. Klicken Sie auf **"Hinzufügen",** um ein Testbasiskonto zu erstellen.

![Klicken auf "Hinzufügen", um das Konto zu erstellen](Media/CreateTestAccount2.png)

3.  Lesen Sie das Kontrollkästchen durch, ```Terms of Use``` um Ihre Zufriedenheit mit der zu ```Terms of Use``` bestätigen.

![Überprüfen der Nutzungsbedingungen](Media/CreateTestAccount3.png)

4.  Geben Sie die richtigen Informationen unter den folgenden Anforderungen ein: 
    -   Abonnement: Ressourcengruppe
    -   Instanzdetails: Name.

**Derzeit unterstützt "Test Base" nur die Standardpreisstufe.**

![Wählen Sie "Abonnement", "Ressourcengruppe" und "Typ" in den Details aus.](Media/CreateTestAccount4.png)

5.  Klicken Sie abschließend ```Review + Create``` auf, um Ihr neu erstelltes Konto zu überprüfen und zu aktivieren.
