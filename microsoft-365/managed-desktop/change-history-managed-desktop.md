Sie können den Verlauf bestimmter Commits (einschließlich Details zu änderungen) jederzeit anzeigen, indem Sie das [Repository auf GitHub](https://github.com/MicrosoftDocs/microsoft-365-docs/tree/public/microsoft-365/managed-desktop)besuchen.
