# <a name="microsoft-managed-desktop-archived-devices"></a>Von Microsoft Managed Desktop archivierte Geräte

Um eine optimale Benutzererfahrung zu gewährleisten, wird die Unterstützung für Geräte von Microsoft Managed Desktop zu einem bestimmten Zeitpunkt zurückziehen. In diesem Artikel werden Geräte aufgeführt, die sich in den nächsten Jahren dem Ausstand nähern, die wir als "archiviert" bezeichnen. Wenn Sie mit Microsoft Managed Desktop beginnen, indem Sie Bereits installierte Geräte verwenden, sollten Sie diese Liste sorgfältig überprüfen.

>[!IMPORTANT]
>Sie können diese Geräte noch heute registrieren, aber sie haben eine kürzere Lebensdauer mit Microsoft Managed Desktop. Wenn Sie neue Geräte kaufen, sollten Sie Modelle aus der Liste [der aktiven Geräte verwenden.](./device-list.md)
