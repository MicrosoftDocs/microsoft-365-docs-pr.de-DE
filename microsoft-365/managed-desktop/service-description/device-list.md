# <a name="microsoft-managed-desktop-devices"></a>Von Microsoft verwaltete Desktop Geräte 

Microsoft Managed Desktop wertet regelmäßig Geräte aus, die in den Dienst eingeschlossen werden sollen. In diesem Thema werden die Anforderungen für ein Gerät beschrieben, das in Microsoft Managed Desktop registriert werden muss.

Weitere Informationen zu Microsoft Managed Desktop finden Sie unter [Dokumentation und Ressourcen für Microsoft Managed Desktop](https://docs.microsoft.com/microsoft-365/managed-desktop/).

Außerdem muss das Gerät Folgendes tun:     

- Haben Sie ein Bild aus der Factory oder durch erneutes Imaging, das:
    - Hat die neueste Version von Windows 10 pro oder Enterprise oder unmittelbar vorhergehende Version;
    - Hat die neueste Version von Microsoft Office 365, klicken Sie auf Ausführen oder unmittelbar vorhergehender Version;
    - Hat die neuesten OEM-bereitgestellten Treiber installiert; und
    - Keine zusätzliche vorinstallierte Software
    - HP-Geräte mit 5VS04AV müssen über ein Microsoft Managed Devices-Abbild verfügen.
- Die Windows-Erstausführung ist nicht abgeschlossen.
- Sie müssen mit [Windows Autopilot](https://docs.microsoft.com/en-us/windows/deployment/windows-autopilot/user-driven-aad) registriert sein und das Microsoft Managed Desktop Autopilot-Profil zugewiesen haben.
