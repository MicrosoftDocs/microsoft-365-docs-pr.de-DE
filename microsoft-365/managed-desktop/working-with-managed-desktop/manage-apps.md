# <a name="manage-apps-in-microsoft-managed-desktop"></a>Verwalten von Apps in Microsoft Managed Desktop

<!--Application management -->

Sobald Applications Onboarded sind, deren Verwaltung Kunden, Microsoft verwalteter Desktop Vorgänge und Support von Microsoft verwalteter Desktop gemeinsam verwendet. Kunden:

- Stellen Sie sicher, dass die entsprechende Lizenzen für die gesamte Software über Intune bereitgestellt werden 
- Überwachen des Status des ihrer Anwendung
- Verwalten von Application Settings Profile für die Bereitstellung

Microsoft verwalteten Desktops Vorgänge Team prüft app Bereitstellung Benachrichtigungen, um sicherzustellen, dass keine Anwendungen aus die Microsoft verwalteter Desktop Begriffe verletzen bereitgestellt werden. 

Unterstützung von Microsoft verwalteten Desktops bietet Unterstützung für Microsoft verfasste Applications (beispielsweise Office).
