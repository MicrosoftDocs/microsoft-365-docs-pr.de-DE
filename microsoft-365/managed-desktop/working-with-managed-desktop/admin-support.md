# <a name="admin-support-for-microsoft-managed-desktop"></a>Administratorunterstützung für Microsoft Managed Desktop

Sie können Supporttickets oder Feedbackanfragen über das Microsoft Managed Desktop Administrative Portal an Microsoft senden. Supportanforderungen werden immer vor Feedbackübermittlungen priorisiert. Supportanforderungen werden nach Schweregrad triaged und verwaltet, wie in der [Schweregraddefinitionstabelle beschrieben.](#sev) Feedback wird überprüft, und es wird eine Antwort bereitgestellt, sofern angefordert. 

>[!IMPORTANT]
>Stellen Sie sicher, [dass Sie einen Administratorkontakt für](../get-started/add-admin-contacts.md) App-Packen, Geräte, Sicherheit und andere einrichten. Sie können in keinem dieser Bereiche eine Supportanfrage übermitteln, wenn kein Administratorkontakt konfiguriert ist.

**So übermitteln Sie eine Supportanfrage**
1. Melden Sie sich bei [Microsoft Endpoint Manager an,](https://endpoint.microsoft.com/) und navigieren Sie zum **Menü Problembehandlung + Support.**
2. Suchen Sie nach dem Abschnitt Microsoft Managed Desktop, wählen Sie **Dienstanforderung aus.**
3. Wählen **Sie auf Supportanfragen** **+ Neues Supportticket aus.**
4. Wählen Sie **den Supportanforderungstyp aus,** der der von Ihnen benötigten Hilfe entspricht. In der folgenden Tabelle sind die Optionen aufgeführt. 
5. Wählen Sie **den Schweregrad aus.** Weitere Informationen finden Sie unter [Support request severity definitions](#sev). 

Supportanforderungstyp | Geeignet in folgender Situation
--- | ---
Incident | Sie müssen das Microsoft Managed Desktop Operations Team untersuchen, z. B. die weit verbreiteten Auswirkungen einer Änderung oder eines Dienstausfalls.
Anfordern von Informationen | Sie planen eine Änderung für Netzwerk, Proxykonfiguration, VPN-Systeme, Zertifikatablauf oder benötigen nur einige Informationen zum Dienst. Eine Antwort des Microsoft Managed Desktop Operations Teams wird bei der Übermittlung einer Änderung in Ihrer Organisation empfohlen.
Änderungsanforderung | Sie benötigen vom Microsoft Managed Desktop Operations Team eine Änderung, z. B. das Verschieben von Geräten zwischen Updategruppen.

## <a name="support-request-severity-definitions"></a>Unterstützung von Definitionen für den Schweregrad von Anfragen

Die erste Antwortzeit ist der Zeitraum, ab dem Sie Ihre Supportanfrage übermitteln, an den sie von einem Microsoft Managed Desktop-Techniker kontaktiert werden und mit der Arbeit an Ihrer Supportanfrage beginnen. Die anfängliche Antwortzeit variiert je nach dem Schweregrad der Anforderung von den geschäftlichen Auswirkungen der Anforderung.

Schweregrad  | Kundensituation |  Anfängliche Antwortzeit   | Erwartete Kundenantwort
--- | --- | --- | ---
**Schweregrad A – Kritische Auswirkungen** |  **Wichtige Geschäftliche Auswirkungen**<br><br>Das Unternehmen des Kunden hat einen erheblichen Verlust oder eine Verschlechterung der Dienstleistungen und erfordert sofortige Aufmerksamkeit.<br><br>**Wichtige Auswirkungen auf die Anwendungskompatibilität**<br><br>Das gesamte Unternehmen des Kunden hat finanzielle Auswirkungen aufgrund des Absturzes oder des Verlusts kritischer Funktionen | Initial: < 1 Stunde<br>Update: 60 Minuten<br>24 x 7 verfügbar | Wenn Sie Schweregrad A auswählen, bestätigen Sie, dass das Problem wichtige geschäftliche Auswirkungen hat, mit schwerwiegenden Verlusten und Beeinträchtigungen von Diensten. <br><br>Das Problem erfordert eine sofortige Antwort, und Sie verpflichten sich, den 24 x 7-Betrieb täglich mit dem Microsoft-Team bis zur Lösung zu gewährleisten, andernfalls kann Microsoft den Schweregrad auf Stufe B verringern.<br><br> Sie stellen außerdem sicher, dass Microsoft über Ihre genauen Kontaktinformationen verfügt. 
**Schweregrad B – Moderate Auswirkungen** |  **Moderate Auswirkungen auf das Unternehmen**<br><br>Das Unternehmen des Kunden hat einen moderaten Verlust oder eine Verschlechterung der Dienstleistungen, aber die Arbeit kann in einem vernünftigen Maß beeinträchtigt fortgesetzt werden.<br><br>**Moderate Auswirkungen auf die Anwendungskompatibilität**<br><br>Eine bestimmte Geschäftsgruppe ist aufgrund des Absturzverhaltens oder des Verlusts kritischer Funktionen nicht mehr produktiv. |  Initial: < 4 Stunden<br>Update: 12 Stunden<br>Geschäftszeiten (24 x 7 verfügbar) | Wenn Sie Schweregrad B auswählen, bestätigen Sie, dass das Problem moderate Auswirkungen auf Ihr Unternehmen mit Verlust und Beeinträchtigung von Diensten hat, aber Problemumgehungen ermöglichen eine angemessene, wenn auch temporäre Geschäftskontinuität. <br><br>Das Problem erfordert eine dringende Antwort. Wenn Sie sich für 24 x 7 entschieden haben, wenn Sie die Supportanfrage übermitteln, verpflichten Sie sich täglich zu einem kontinuierlichen 24 x 7-Vorgang mit dem #A0 bis zur Lösung, andernfalls kann Microsoft den Schweregrad auf Stufe C verringern. Wenn Sie sich für den Support während der Geschäftszeiten entschieden haben, wenn Sie einen Schweregrad B-Vorfall übermitteln, kontaktiert Microsoft Sie nur während der Geschäftszeiten.<br><br>Sie stellen außerdem sicher, dass Microsoft über Ihre genauen Kontaktinformationen verfügt.
**Schweregrad C – Minimale Auswirkung** |   **Minimale Geschäftliche Auswirkungen**<br><br> Das Unternehmen des Kunden funktioniert mit geringfügigen Dienstleistungen.<br><br>**Geringfügige Auswirkungen auf die Anwendungskompatibilität**<br><br>Potenziell nicht zusammenhängende Benutzer haben geringfügige Kompatibilitätsprobleme, die die Produktivität nicht verhindern |    Initial: < 8 Stunden<br>Update: 24 Stunden<br>Geschäftszeiten  | Wenn Sie den Schweregrad C auswählen, bestätigen Sie, dass das Problem minimale Auswirkungen auf Ihr Unternehmen hat, wenn geringfügige Dienstbeschwerde vor sich geht.<br><br>Bei einem Vorfall mit dem Schweregrad C kontaktiert Microsoft Sie nur während der Geschäftszeiten.<br><br>Sie stellen außerdem sicher, dass Microsoft über Ihre genauen Kontaktinformationen verfügt.

Weitere Details:
- **Supportsprachen** – Der support wird auf Englisch bereitgestellt.
- **Änderungen des Schweregrads** – Microsoft kann den Schweregrad herabstufen, wenn der Kunde nicht in der Lage ist, ausreichende Ressourcen oder Antworten zur Verfügung zu stellen, damit Microsoft die Problemlösung fortsetzen kann. 
- **Geschäftszeiten** – In den meisten Ländern sind die Geschäftszeiten von 09:00 Uhr bis 17:00 Uhr( Pacific Standard Time).
- **Anwendungskompatibilität** – Damit ein Anwendungskompatibilitätsproblem berücksichtigt werden kann, muss zwischen der vorherigen und aktuellen Version von Windows oder Office ein reproduzierbarer Fehler derselben Version der Anwendung aufgetreten sein. Zum Beheben von Anwendungskompatibilitätsproblemen benötigt Microsoft eine Kundenkontaktstelle, mit der sie zusammenarbeiten kann. Die Person muss direkt mit unserem Fast Track-Team zusammenarbeiten, um das Problem zu untersuchen und zu beheben.
- **Kundenantwortzeit** Wenn ein Kunde die erwarteten Antwortanforderungen nicht erfüllen kann, stuft Microsoft die Anforderung um einen Schweregrad auf ein Minimum von Schweregrad C herab. Wenn ein Kunde nicht auf Aktionsanforderungen reagiert, wird microsoft die Supportanfrage innerhalb von 48 Stunden nach der letzten Anforderung mindern und schließen.

## <a name="provide-feedback"></a>Feedback geben

Wir freuen uns über Ihr Feedback und verwenden es, um die Administratorunterstützung zu verbessern.

Sobald sich ein Ticket im **Status "Entschärft"** oder **"Aufgelöst"** befindet, können Sie Ihr Feedback zu Ihrer Erfahrung mit diesem bestimmten Problem teilen. Um Feedback zu teilen, wechseln Sie zur Seite **Dienstanforderungen** im Menü **Problembehandlung + Support** des MEM-Portals. Wählen Sie das bestimmte Ticket aus. Die Ticketdetails werden im Fly-In auf der rechten Seite angezeigt, wählen Sie die Registerkarte **Feedback** aus, und geben Sie die angeforderten Informationen an. Achten Sie darauf, keine persönlichen Informationen in das Feedbackformular zu geben. Weitere Informationen zum Datenschutz finden Sie unter [Microsoft Privacy Statement](https://privacy.microsoft.com/privacystatement).
