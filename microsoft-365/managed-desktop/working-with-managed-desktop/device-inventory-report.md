# <a name="device-inventory-report"></a>Gerätebestandsbericht

In der Ansicht **"Geräte"** können Sie die Registerkarte **"Alle exportieren"** auswählen, um eine durch Trennzeichen getrennte Datei herunterzuladen, einschließlich der folgenden Informationen:

- Seriennummer
- Gerätename
- Letztes Einchecken des Geräts
- Verwaltet von
- Hersteller
- Modell
- Os
- Betriebssystemlizenz
- Betriebssystemversion
- Besitz
- Registrierungsdatum
- Ringname
- Storage – Kostenlos in GB
- Storage – Gesamtanzahl in GB
- Registriert durch Benutzer-UPN
- Zugewiesener Benutzer
- Status
- Alter (Monate)
- Profil
- Angemeldete Benutzer
- TPM-Version
- Sicherer Start aktiviert
- Primärer Datenträgertyp
- Gesamter physischer Speicher
