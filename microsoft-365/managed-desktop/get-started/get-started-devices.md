# <a name="get-your-users-ready-to-use-devices"></a>Vorbereiten Ihrer Benutzer für die Verwendung von Geräten

Sobald ein Microsoft Managed Desktop-Gerät in den Händen Ihres Benutzers ist, sind die ersten Schritte mit diesem Gerät schnell und einfach. Geräte sind mit der aktuellen Version von Windows vorkonfiguriert, und Konfigurationen und Apps werden aus der Cloud installiert, sobald der Benutzer die Einrichtung abgeschlossen hat. 
 
Um dies noch einfacher zu machen, bieten wir einen Leitfaden an, der Ihre Benutzer durch die anfängliche Einrichtung führt und Hilferessourcen sowohl für das Setup als auch für die spätere Verwendung zur Verfügung stellt, falls erforderlich. Sie können dieses Handbuch so anpassen, dass bestimmte Für Ihre Organisation spezifische Details enthalten sind. Anschließend verteilen Sie das Handbuch direkt an Ihre Benutzer zusammen mit ihrem Gerät. Es gibt zwei Versionen, eine für Organisationen, die das Supportsystem von Microsoft verwenden (die App "Hilfe holen"), eine für Organisationen, die alternative Unterstützung für ihre Benutzer bereitstellen.

## <a name="to-prepare-the-guide"></a>So bereiten Sie das Handbuch vor

1. Laden Sie die für Ihre Organisation geeignete Version herunter:
- Microsoft Managed Desktop – Erste Schritte mit Ihrem Gerät [(Hilfeversion erhalten)](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-custom-v3.pdf) – Verwenden Sie diese Version, wenn Ihre Benutzer das Microsoft-Supportsystem verwenden.
- [Microsoft Managed Desktop – Erste Schritte mit Ihrem](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-custom-v2.pdf) Gerät  – Verwenden Sie diese Version, wenn Ihre Benutzer das Supportsystem von Microsoft nicht verwenden.
2. Verwenden Sie jede App, die PDF-Dateien öffnen kann, um details zu füllen, die für Ihre Organisation relevant sind:
    - Der Name des Netzwerks, mit dem Ihre Benutzer eine Verbindung herstellen sollten, um das Setup fortsetzen zu können (Schritt 3 in der Anleitung)
    - Der Name des Azure-Mandantenkontos Ihrer Organisation (Schritt 4 im Handbuch)
    - Kontaktinformationen für den internen IT-Support Ihrer Organisation (oben auf der zweiten Seite)
3. Speichern Sie die bearbeitete PDF, und verteilen Sie sie dann an Ihre Benutzer.
