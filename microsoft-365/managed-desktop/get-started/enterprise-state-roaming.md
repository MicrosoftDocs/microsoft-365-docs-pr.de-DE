# <a name="enable-enterprise-state-roaming"></a>Aktivieren von Enterprise State Roaming

[Enterprise Statusroaming](/azure/active-directory/devices/enterprise-state-roaming-overview) ermöglicht Benutzern die sichere Synchronisierung von Benutzer- und Anwendungseinstellungsdaten mit der Cloud. Dies bedeutet, dass sie die gleiche Erfahrung haben, unabhängig davon, auf welchem Windows Gerät sie sich anmelden. Wenn Sie beispielsweise eines ihrer Microsoft Managed Desktop-Geräte durch ein neues ersetzen, sieht es genauso aus und verhält sich genauso wie das letzte Gerät. Enterprise Statusroaming ist ein optionales Feature für den Microsoft Managed Desktop Dienst, das Sie für Ihre Benutzer konfigurieren können und nicht im Rahmen Microsoft Managed Desktop enthalten oder verwaltet wird.

Um Enterprise Statusroaming zu aktivieren, führen Sie die Schritte unter [Aktivieren Enterprise Zustandsroaming in Azure Active Directory](/azure/active-directory/devices/enterprise-state-roaming-enable)aus.

>[!NOTE]
>Wenn Sie Enterprise Statusroaming aktivieren, überschreibt Ihre bevorzugte Sprachenliste die während der Geräteeinrichtung ausgewählte Sprache. Obwohl Benutzer dies problemlos beheben können, kann dies zunächst zu einer inkonsistenten Lokalisierung führen. Ermitteln Sie vor dem Einrichten von Geräten, ob Enterprise Statusroaming für Ihre Benutzer geeignet ist.
