# <a name="enable-enterprise-state-roaming"></a>Aktivieren von Enterprise State Roaming

Um eine optimale Erfahrung mit Microsoft Managed Desktop, aktivieren Sie [Enterprise State Roaming](/azure/active-directory/devices/enterprise-state-roaming-overview), mit dem Benutzer Benutzer- und Anwendungseinstellungendaten sicher mit der Cloud synchronisieren können. Dies bedeutet, dass sie die gleiche Erfahrung haben, unabhängig davon, Windows sie sich anmelden. Wenn Sie beispielsweise eines ihrer Microsoft Managed Desktop durch ein neues ersetzen, sieht es genauso aus und verhält sich genauso wie das letzte.

Führen Sie Enterprise schritte aktivieren Enterprise [Statusroaming in Azure Active Directory](/azure/active-directory/devices/enterprise-state-roaming-enable)aus, und kehren Sie dann zu dieser Dokumentation zurück.

Wenn Sie Probleme mit Enterprise Statusroaming haben, wenden Sie sich an den [Administratorsupport](../working-with-managed-desktop/admin-support.md).
