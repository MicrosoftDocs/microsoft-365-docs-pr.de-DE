# <a name="enable-enterprise-state-roaming"></a>Aktivieren von Enterprise State Roaming

Um die besten Erfahrungen mit Microsoft Managed Desktop zu erzielen, aktivieren Sie das [Enterprise State Roaming](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-overview), sodass Benutzer und Anwendungseinstellungsdaten von Benutzern auf sichere Weise mit der Cloud synchronisiert werden können. Dies bedeutet, dass Sie unabhängig von dem Windows-Gerät, an dem Sie sich anmelden, dieselbe Erfahrung haben. Wenn Sie beispielsweiseeines Ihrer Microsoft Managed Desktop-Geräte durch ein neues ersetzen, wird es genau wie das letzte Aussehen und sich so Verhalten.

Zum Aktivieren des Enterprise-Status-Roaming führen Sie die Schritte unter [enable Enterprise State Roaming in Azure Active Directory](https://docs.microsoft.com/azure/active-directory/devices/enterprise-state-roaming-enable)aus, und kehren Sie dann zu dieser Dokumentation zurück.

Wenn Sie Probleme mit dem Enterprise State Roaming haben, wenden Sie sich an den Administrator [Support](../working-with-managed-desktop/admin-support.md).
