# <a name="set-up-microsoft-managed-desktop-devices"></a>Richten Sie Microsoft Managed Desktop-Geräte ein

Sie können sowohl neue als auch vorhandene Geräte in Microsoft Managed Desktop.

## <a name="to-obtain-new-devices"></a>So erhalten Sie neue Geräte

Wir empfehlen die Zusammenarbeit mit einem unserer genehmigten Gerätepartner. Sie können mit Ihrem Microsoft-Kontokontakt zusammenarbeiten, um weitere Hilfe beim Einrichten einer Gerätepartnerschaft zu erhalten. Im Wesentlichen ist der Vorgang jedoch wie dies:

1. Überprüfen Sie die Liste der derzeit empfohlenen Geräte, indem Sie Microsoft Managed Desktop auf der Website [Geschäft Windows 10 Pro Geschäftsgeräte](https://www.microsoft.com/windowsforbusiness/view-all-devices) filtern.
2. Bestellen Sie ein oder mehrere Beispiele der Geräte, die Sie mit einem kompatiblen Bild verwenden möchten. Für die Sortierung sind [möglicherweise bestimmte Bestellschritte erforderlich.](../service-description/device-images.md)
3. [Überprüfen](validate-device.md) Sie die Beispielgeräte.
5. Nach erfolgreicher Überprüfung müssen Sie die Geräte in Zusammenarbeit mit einem genehmigten Gerätepartner bestellen.
6. Nachdem sie angekommen sind, gehen Sie wie folgt vor:
    - [Selbstregistrieren neuer Geräte](register-devices-self.md)
    - Zusammenarbeit mit einem Partner zum Registrieren der Geräte
7. [Machen Sie Ihre Benutzer bereit,](get-started-devices.md) Microsoft Managed Desktop verwenden

## <a name="to-reuse-existing-devices"></a>So verwenden Sie vorhandene Geräte wiederverwenden

1. Überprüfen Sie, ob Ihre vorhandenen Geräte unsere [Geräteanforderungen erfüllen.](../service-description/device-requirements.md) Sie können auch die Überprüfung der [Herunterladbaren](../get-ready/readiness-assessment-downloadable.md) Bereitschaftsbewertung verwenden, um zu überprüfen, ob ein bestimmtes Gerät die erforderlichen Anforderungen erfüllt. 
2. Wenn Sie ein vorhandenes Gerät wiederverwenden, müssen Sie es möglicherweise erneut abbilden. Bildoptionen finden Sie unter [Gerätebilder](../service-description/device-images.md).
3. Wählen Sie ein oder mehrere Beispiele für die Geräte aus, die Sie wiederverwenden möchten, und überprüfen Sie [sie dann.](validate-device.md)
4. Gehen Sie nach erfolgreicher Überprüfung wie folgt vor:
    - [Selbstregistrieren vorhandener Geräte](register-reused-devices-self.md)
    - Zusammenarbeit mit einem Partner zum Registrieren der Geräte
5. [Machen Sie Ihre Benutzer bereit,](get-started-devices.md) Microsoft Managed Desktop verwenden
