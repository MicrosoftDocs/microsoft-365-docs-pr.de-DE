# <a name="host-online-meetings-for-your-business"></a>Hosten von Onlinebesprechungen für Ihr Unternehmen

Sie müssen sich mit einem Kunden, Kunden oder Partner treffen, aber nicht alle an einem Ort abrufen? Keine Sorge, Sie haben eine Onlinebesprechung. Wenn Sie über ein Smartphone, Tablet oder Laptop verfügen, können Sie sich mit Microsoft Teams verbinden.

Sie müssen die [﻿Kostenlose Version](https://support.microsoft.com/office/6d79a648-6913-4696-9237-ed13de64ae3c) herunterladen oder sich [Online in einem Browser anmelden](https://support.microsoft.com/office/1613bb53-f3fa-431e-85a9-d6a91e3468c9) , um Video-und Bildschirm Freigaben zu erhalten.

Wenn Sie häufig eng an Projekten mit einem Kunden, einem Kunden oder einem Partner zusammenarbeiten, sollten Sie das [Erstellen eines Teams für Ihre Kunden](https://support.microsoft.com/office/11fbb083-52ee-434d-8c6e-63711fdafac7) in die Hand nehmen, um Sie problemlos in Besprechungen einzuladen und auch Unterhaltungen zu führen, Dateien freizugeben und alle Projekte in Microsoft Teams nachzuverfolgen.

## <a name="1-schedule-a-meeting"></a>1. Planen einer Besprechung

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FOhP?autoplay=false]

Verwenden Sie Microsoft Teams, um Besprechungen mit ihren Mitarbeitern, Kunden und anderen Gästen zu planen. Versuch es:

1. Wählen Sie in Microsoft Teams im linken Navigationsbereich **Besprechungen** aus.
1. Wählen Sie **Besprechungstermin planen** aus.
1. Geben Sie im **Feld neue Besprechung** einen **Titel** und einen **Ort** für die Besprechung ein.
1. Geben Sie eine **Startzeit** und ein **Endzeit** Datum ein.
1. Geben Sie im Feld **Details** eine Beschreibung der Besprechung und andere Details ein, die Sie hinzufügen möchten, beispielsweise eine Besprechungsagenda.
1. Geben Sie unter **Personen einladen** die Namen der Mitarbeiter oder Clients ein, die Sie einladen möchten.
1. Wenn Sie unter "mit **Vorbehalt** " oder " **beschäftigt** " unterhalb von Namen sehen, wählen Sie eine der bereitgestellten **freien** Zeiten aus, oder klicken Sie auf **Zeit Plan Assistent** , um weitere Optionen
1. Wählen Sie **Besprechungstermin planen** aus.

## <a name="2-join-a-meeting"></a>2. an einer Besprechung teilnehmen

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FYWn?autoplay=false]

Verwenden Sie Microsoft Teams, um an Besprechungen mit beiden Mitarbeitern in Ihrem Unternehmen und an Kunden außerhalb Ihres Unternehmens teilzunehmen. Versuch es:

1. Wählen Sie in Microsoft Teams im linken Navigationsbereich **Besprechungen** aus.
1. Öffnen Sie die Besprechung, der Sie beitreten möchten, und klicken Sie auf **Microsoft Teams-Besprechung beitreten**.
1. Wenn die Besprechung geöffnet wird, wählen Sie **jetzt beitreten** aus.
1. Alle zu Ihrer Besprechung eingeladenen Clients öffnen die Besprechung in Ihrem Kalender, wählen den Besprechungslink aus, laden die Teams-APP herunter oder öffnen Sie im Internet, geben den Namen ein und klicken dann auf **beitreten**.
1. Wenn Sie sehen, dass Ihr Client in der Lobby angezeigt wird, wählen Sie **zulassen** aus, um Sie einzulassen.
1. Sobald alle Benutzer beitreten, können Sie die Besprechung starten.
 
## <a name="3-have-an-impromptu-meeting"></a>3. eine spontane Besprechung haben

Sie können einen Chat in Microsoft Teams ganz einfach in eine Besprechung umwandeln. Wählen Sie einfach **Video Anruf** :::image type="icon" source="../media/video-call-icon.png" border="false"::: Taste oder **Audioanruf-** :::image type="icon" source="../media/audio-call-icon.png" border="false"::: Schaltfläche, um einen Anruf zu starten. Sie können auch Personen zu dem Anruf hinzufügen, wenn er in eine Gruppendiskussion umgewandelt werden muss.

Benötigen Sie ein wenig Hilfe? Weitere Informationen finden Sie unter [Starten eines Anrufs von einem Chat in Microsoft Teams](https://support.microsoft.com/office/f5138c9d-df4c-43d8-9cf6-53400c1a7798).
