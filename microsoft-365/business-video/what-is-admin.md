# <a name="what-is-a-microsoft-365-admin"></a>Was ist ein Microsoft 365-Administrator?

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1SRc0?autoplay=false]

In jedem Microsoft 365 Business-Abonnement muss mindestens ein Administratorkonto vorhanden sein. Der Administrator verwaltet den Dienst, Benutzerkonten und Geräte. Wenn Sie der Abonnent sind, der sich für das Abonnement angemeldet hat, sind Sie der globale Administrator für Ihr Abonnement. Sie können auch Administrator werden, wenn Sie von einem anderen Administrator erstellt werden.

Sie benötigen zwar immer mindestens eine Administratorrolle für Ihr Abonnement, es kann jedoch ein Sicherheitsrisiko darstellen, wenn das Administratorkonto durch eine externe Partei kompromittiert wird. Es wird empfohlen, die Administratorrolle nur denjenigen zuzuweisen, die diese wirklich benötigen, und die Benutzerrolle den meisten Personen zuzuweisen.

## <a name="try-it"></a>Probieren Sie es aus!

### <a name="change-a-user-role-to-an-admin-role"></a>Ändern einer Benutzerrolle in eine Administratorrolle

Hier erfahren Sie, wie Sie eine Benutzerrolle in eine Administratorrolle ändern:

1. Wenn Sie bei Microsoft 365 angemeldet sind, wählen Sie das App-Startprogramm aus. Wenn die Schaltfläche admin angezeigt wird, sind Sie Administrator.
1. Wählen Sie **Administrator** aus, um zum Microsoft 365 Admin Center zu wechseln.
1. Wählen Sie im linken Navigationsbereich **Benutzer** und dann **aktive Benutzer** aus.
1. Wählen Sie die Person aus, die Sie als Administrator festlegen möchten. Die Details des Benutzers werden im rechten Dialogfeld angezeigt.
1. Wählen Sie **Rollen verwalten** aus.
1. Deaktivieren Sie das Kontrollkästchen **Benutzer (kein Administratorzugriff)** , und wählen Sie dann eine der Administratorrollen aus. Wenn Sie mehr über eine Rolle erfahren möchten, bewegen Sie den Mauszeiger über das Ausrufezeichen (!), und lesen Sie die QuickInfo. Um dem Benutzer die gleiche Rolle wie Ihnen zu geben, wählen Sie **globaler Administrator** aus.
1. Wählen Sie **Änderungen speichern** aus.

### <a name="create-an-admin-account"></a>Erstellen eines Administratorkontos 

Sie können auch ein dediziertes Administratorkonto für eine IT-Person oder einen IT-Dienst erstellen:

1. Wenn Sie bei Microsoft 365 angemeldet sind, wählen Sie das App-Startprogramm aus.
1. Wählen Sie **Administrator** aus, um zum Microsoft 365 Admin Center zu wechseln.
1. Wählen Sie im linken Navigationsbereich **Benutzer** und dann **aktive Benutzer** aus.
1. Wählen Sie **Benutzer hinzufügen** aus.
1. Geben Sie den **Vornamen** und Nachnamen, den **Anzeigenamen** und den **Benutzernamen des** Benutzers **ein, und** wählen Sie dann **weiter** aus.
1. Wählen Sie **Benutzer ohne Produktlizenz erstellen** aus. Dadurch wird ein neuer Benutzer hinzugefügt, aber Sie müssen nicht für ein monatliches Abonnement bezahlen. Wählen Sie **Weiter** aus.
1. **Rollen** öffnen.
1. Deaktivieren Sie das Kontrollkästchen, und wählen Sie dann eine der Administratorrollen aus. Wählen Sie **Weiter** aus.
1. Überprüfen Sie die Informationen, und wählen Sie dann **Hinzufügen fertig stellen** aus.
1. Wählen Sie **Schließen** aus.
