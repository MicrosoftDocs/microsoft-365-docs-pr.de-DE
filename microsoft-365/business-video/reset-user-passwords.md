# <a name="reset-passwords-for-your-users"></a>Zurücksetzen von Kennwörtern für Ihre Benutzer

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FVVP?autoplay=false]

In Microsoft 365 for Business können Sie Kennwörter für Benutzer zurücksetzen. Sie können Mitarbeiter auch Ihre eigenen Kennwörter zurücksetzen lassen. Weitere Informationen finden Sie unter [Einrichten von Self-Service Password Reset](set-up-self-serve-password-reset.md) .

## <a name="try-it"></a>Probieren Sie es aus!

1. Wenn ein Benutzer ein neues Kennwort anfordert, erhalten Sie eine Anforderung zum Zurücksetzen von Kennwörtern in e-Mails. Um das Kennwort zurückzusetzen, öffnen Sie das App-Startfeld und wählen Sie **Admin** aus.
1. Wählen Sie im Microsoft 365 Admin Center die Option **Users**,  **Active Users** aus, und wählen Sie dann das Schlüsselsymbol neben dem Benutzer aus, der das Zurücksetzen angefordert hat.
1. Wählen Sie **Kennwort automatisch generieren** aus, damit ein zufälliges Kennwort automatisch erstellt wird.
1. Wählen Sie **Zurücksetzen** aus.
1. Aktivieren Sie das Kontrollkästchen **Kennwort in e-Mail senden** .
1. Ihre e-Mail-Adresse wird im Feld **e-Mail an das neue Kennwort an folgende Empfänger** angezeigt. Fügen Sie die Alternative e-Mail-Adresse des Benutzers oder eine beliebige von Ihnen ausgewählte Adresse hinzu.
1. Wählen Sie **e-Mail senden und schließen** aus.

    Der Benutzer erhält eine e-Mail mit den Anweisungen zum Zurücksetzen des Kennworts.
