# <a name="work-from-anywhere---overview"></a>Work from Anywhere – Übersicht

Weg von Ihrem Schreibtisch? Keine Laptop Batterie mehr? Kein Problem! Erledigen Sie Ihre Arbeit von überall mit Office auf Ihrem Telefon oder Tablet.

**Pro-Tipp:** Halten Sie Ihre Work-Life-Balance in Schach, indem Sie Benachrichtigungen für Ihre geschäftlichen apps auf Ihren persönlichen Geräten deaktivieren.

Verwenden Sie die folgenden vier Tipps, um überall zu arbeiten: Laden Sie eine Infografik herunter, um Tipps für die Arbeit von überall zu erhalten: [PDF](https://go.microsoft.com/fwlink/?linkid=2079451), [PowerPoint](https://go.microsoft.com/fwlink/?linkid=2079455)

**Sie denken daran, regelmäßig von zu Hause zu arbeiten?**

Verwenden Sie diese Vorlage zum Analysieren ihrer Gewohnheiten und der Art und Weise, wie Sie arbeiten, um einen Plan zu erstellen, der beim Übergang zu einem Home Office funktioniert. Sammeln Sie die Materialien und Einrichtungen, um Ihren Raum zu optimieren und Sie effizient und auf dem Arbeitsplatz zu halten. [Herunterladen der Vorlage](https://templates.office.com/EN-US/work-from-home-checklist-TM77989015)

## <a name="install-outlook-and-keep-in-touch"></a>Installieren von Outlook und Keep in Touch

Sie können über e-Mails und Termine mit Outlook auf Ihrem Telefon hinaus bleiben. Laden Sie die APP herunter, starten Sie, melden Sie sich an, und richten Sie Ihr Konto ein.

Hier finden Sie weitere Informationen zum [Installieren von Outlook auf einem Telefon](https://support.microsoft.com/office/647909af-560f-4977-ae45-3b45ad9e8236).

## <a name="save-your-files-to-the-cloud"></a>Speichern Ihrer Dateien in der Cloud

Speichern Sie Ihre Geschäftsdateien in Microsoft Teams oder SharePoint. Auf diese Weise können Sie und die anderen Personen in Ihrem Unternehmen auch dann darauf zugreifen, wenn Sie nicht im Büro sind.

**Einrichten der SharePoint-und Microsoft Teams-Mobile Apps**

- Rufen Sie die [SharePoint-Mobile App](https://support.microsoft.com/office/539608ac-4725-455e-aea0-9ca1f769849f)ab.
- Rufen Sie die [Microsoft Teams-Mobile App](https://support.microsoft.com/office/set-up-your-teams-mobile-apps-1ba8dce3-1122-47f4-8db6-00a4f93117e8)ab.

Sie können Ihre persönlichen Dateien in OneDrive speichern und von Ihrem Telefon aus synchronisieren.

**Synchronisieren von Dateien mit OneDrive**

1. Öffnen Sie OneDrive, und melden Sie sich an.
1. Befolgen Sie die Anweisungen, um das Setup abzuschließen und die Synchronisierung zu starten.

Benötigen Sie ein wenig Hilfe? Erhalten Sie Ratschläge darüber, [wo Dateien gespeichert](https://support.microsoft.com/office/c7c20284-bc94-47f4-9728-d28e9daf0790)werden sollen, wie Sie [die OneDrive Mobile App](https://support.microsoft.com/office/448d4051-3a43-4d2e-b1d8-de0aa03c069e) und wie Sie [Dateien und Ordner synchronisieren](https://support.microsoft.com/office/d9262485-9bf8-4ceb-bac2-e83f68cb6a97).

### <a name="work-in-a-browser"></a>Arbeiten in einem Browser

Sie können Dokumente auch direkt in Ihrem Browser erstellen und bearbeiten. Öffnen Sie Dateien, bearbeiten Sie Text, passen Sie die Formatierung an, fügen Sie Bilder hinzu, und erstellen Sie Diagramme in Word, Excel und PowerPoint. Sie können sogar gleichzeitig mit einem Kollegen oder Kunden in einem Dokument arbeiten, wenn Sie [die Datei mit diesen freigegeben](https://support.microsoft.com/office/6725104a-6df7-4778-99c4-c06217dffecc) haben.

1. Melden Sie sich bei [Office.com](https://office.com) mit Ihrem Microsoft 365-Konto an.
1. Starten Sie eine Office-App, um die Arbeit zu erhalten.

**Arbeiten in einem Browser mit Office Online**

### <a name="join-online-meetings-remotely"></a>Remote an Onlinebesprechungen teilnehmen

Von Ihrem Laptop:

- Klicken Sie in Microsoft Teams  :::image type="icon" source="../media/meetings-icon.png" border="false"::: auf der linken Seite der APP auf Besprechungs Schaltfläche, und es wird eine Liste aller Besprechungen für die Woche angezeigt. Suchen Sie die gewünschte Besprechung, und klicken Sie dann auf beitreten.

Auf dem Telefon:

- Sowohl vor als auch während der Besprechung wird der Besprechungs Hinweis in Ihrer Chat Liste oder in einer Kanal Unterhaltung in Microsoft Teams angezeigt.

    Sie können von dieser Besprechungsnotiz oder aus dem Kalender der **Besprechung** teilnehmen. :::image type="icon" source="../media/meetings-icon.png" border="false":::
