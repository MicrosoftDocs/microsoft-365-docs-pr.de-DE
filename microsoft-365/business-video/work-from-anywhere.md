# <a name="work-from-anywhere---overview"></a>Arbeiten von überall aus – Übersicht

Weg vom Schreibtisch? Kein Laptop mehr akku? Kein Problem! Behandeln Sie Ihre Arbeit von überall mit Office auf Ihrem Smartphone oder Tablet.

**Pro-Tipp:** Halten Sie Ihre Work-Life-Balance im Gleichgewicht, indem Sie Benachrichtigungen für Ihre Arbeits-Apps auf Ihren persönlichen Geräten deaktivieren.

Verwenden Sie die vier unten aufgeführten Tipps, um überall zu arbeiten: Laden Sie eine Infografik herunter, um Tipps zum Arbeiten von überall aus zu erhalten: [PDF](https://go.microsoft.com/fwlink/?linkid=2079451), [PowerPoint](https://go.microsoft.com/fwlink/?linkid=2079455)

**Denken Sie daran, regelmäßig von zu Hause aus zu arbeiten?**

Verwenden Sie diese Vorlage, um Ihre Gewohnheiten und die Art und Weise zu analysieren, wie Sie am Erstellen eines Plans arbeiten möchten, der beim Übergang in ein Heimbüro funktioniert. Sammeln Sie die Materialien und Ausstattungen, um Ihren Platz zu optimieren und Effizient und Einsatz zu halten. [Herunterladen der Vorlage](https://templates.office.com/EN-US/work-from-home-checklist-TM77989015)

## <a name="install-outlook-and-keep-in-touch"></a>Installieren von Outlook und In-Kontakt-Bleiben

Sie können mit Outlook auf Ihrem Telefon über E-Mails und Termine auf dem Neuanrufen auf dem Computer bleiben. Laden Sie die App herunter, starten Sie sie, melden Sie sich an, und richten Sie Ihr Konto ein.

Erhalten Sie weitere Details zum Installieren [von Outlook auf einem Telefon.](https://support.microsoft.com/office/647909af-560f-4977-ae45-3b45ad9e8236)

## <a name="save-your-files-to-the-cloud"></a>Speichern Ihrer Dateien in der Cloud

Speichern Sie Ihre Geschäftsdateien in Microsoft Teams oder SharePoint. Auf diese Weise können Sie und die anderen Personen in Ihrem Unternehmen darauf zugreifen, auch wenn Sie nicht im Büro sind.

**Einrichten der mobilen SharePoint- und Microsoft Teams-Apps**

- Holen Sie sich [die mobile SharePoint-App.](https://support.microsoft.com/office/539608ac-4725-455e-aea0-9ca1f769849f)
- Holen Sie sich [die mobile Microsoft Teams-App.](https://support.microsoft.com/office/set-up-your-teams-mobile-apps-1ba8dce3-1122-47f4-8db6-00a4f93117e8)

Sie können Ihre persönlichen Dateien in OneDrive speichern und von Ihrem Telefon aus synchronisieren.

**Synchronisieren von Dateien mit OneDrive**

1. Öffnen Sie OneDrive, und melden Sie sich an.
1. Folgen Sie den Anweisungen, um das Setup abzuschließen und mit der Synchronisierung zu beginnen.

Benötigen Sie etwas Hilfe? Erhalten Sie Tipps [zum Speichern von Dateien,](https://support.microsoft.com/office/c7c20284-bc94-47f4-9728-d28e9daf0790)zur Verwendung der [mobilen #A0](https://support.microsoft.com/office/448d4051-3a43-4d2e-b1d8-de0aa03c069e) und zum Synchronisieren von [Dateien und Ordnern.](https://support.microsoft.com/office/d9262485-9bf8-4ceb-bac2-e83f68cb6a97)

### <a name="work-in-a-browser"></a>Arbeiten in einem Browser

Sie können Dokumente auch direkt in Ihrem Browser erstellen und bearbeiten. Öffnen Sie Dateien, bearbeiten Sie Text, passen Sie die Formatierung an, fügen Sie Bilder hinzu, und erstellen Sie Diagramme in Word, Excel und PowerPoint. Sie können sogar gleichzeitig mit einem Kollegen oder Kunden in einem Dokument arbeiten, wenn Sie Ihre [Datei für](https://support.microsoft.com/office/6725104a-6df7-4778-99c4-c06217dffecc) sie freigegeben haben.

1. Melden Sie sich [bei Office.com](https://office.com) mit Ihrem Microsoft 365-Konto an.
1. Starten Sie eine Office-App, um zu arbeiten.

### <a name="join-online-meetings-remotely"></a>Remotebesprechungen an Onlinebesprechungen teilnehmen

Von Ihrem Laptop aus:

- Klicken Sie in  Microsoft Teams auf der linken Seite der App auf die Schaltfläche "Besprechungen", und es wird eine Liste aller Besprechungen für :::image type="icon" source="../media/meetings-icon.png" border="false"::: die Woche angezeigt. Suchen Sie die Besprechung, die Sie wünschen, und klicken Sie dann auf "Teilnehmen".

Von Ihrem Telefon aus:

- Sowohl vor als auch während der Besprechung wird der Besprechungsbenachrichtigung in Ihrer Chatliste oder einer Kanal unterhaltung in Microsoft Teams angezeigt.

    Sie können aus diesem Besprechungsbenachrichtigungs- oder aus dem **Besprechungskalender** :::image type="icon" source="../media/meetings-icon.png" border="false"::: teilnehmen.
