# <a name="use-microsoft-planner-to-plan-an-event"></a>Planen eines Ereignisses mithilfe von Microsoft Planner

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c8a41e1a-c30d-4adb-ab32-6eac922d0ec0?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Verwenden Sie Microsoft Planner zum Planen eines Ereignisses, z. B. zum Nachverfolgen eines Produktstarts.

### <a name="work-with-buckets"></a>Arbeiten mit Buckets

Verwenden Sie Buckets, um Aufgaben nach Typ zu sortieren, z. B. Ereignisplanung, Marketing- und Anzeigensicherheit und PR.

1. Wählen  **Sie "Neuen Bucket hinzufügen"**  aus, und geben Sie eine neue Aufgabenkategorie ein.
2. Wählen  **+**  Sie diese Option aus, um einem Bucket Aufgaben hinzuzufügen, und geben Sie einen Namen ein.
3. Select  **Add Task**.

### <a name="work-with-labels"></a>Arbeiten mit Bezeichnungen

Verwenden Sie Bezeichnungen, um zu sehen, welche Aufgaben Ihre Aufmerksamkeit erfordern.

1. Geben Sie Namen für Ihre Bezeichnungen ein.
2. Wählen Sie ein farbiges Feld aus, um eine Bezeichnung zu einer Aufgabe zu entfernen oder hinzuzufügen.

## <a name="add-a-comment"></a>Kommentar hinzufügen

Posten Sie einen Kommentar, um eine Aufgabe mit den Personen zu besprechen, denen sie zugewiesen ist.

1. Geben Sie eine Nachricht in das Feld  **"Kommentare"**  ein.
2. Wählen Sie  **"Senden"** aus.

### <a name="keep-track-of-tasks"></a>Nachverfolgen von Aufgaben

Verwenden  **Sie Diagramme**  und  **Boardansicht,**  um den Fortschritt für jedes Teammitglied und das Gesamtprojekt zu überwachen oder Aufgaben zu sortieren und zu organisieren.

- Wählen **Sie Diagramme,** dann **Gruppieren nach** und schließlich Fälligkeitsdatum aus, um zu sehen, welche Aufgabe als Nächstes fällig ist. 
- Select  **Board**, then **Group by**, and finally **Progress**  to sort tasks by progress.

**Hinweis:**  Aufgaben werden nach  **"Nicht gestartet",**  **"In** Bearbeitung" und  **"Abgeschlossen"**  organisiert, damit Sie wissen, wann Sie bereit sind, Ihr Produkt zu starten.
