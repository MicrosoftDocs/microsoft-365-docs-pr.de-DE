# <a name="use-microsoft-planner-to-plan-an-event"></a>Planen eines Ereignisses mithilfe von Microsoft Planner

> [!VIDEO https://www.microsoft.com/videoplayer/embed/c8a41e1a-c30d-4adb-ab32-6eac922d0ec0?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Verwenden Sie Microsoft Planner, um ein Ereignis wie das Verfolgen eines Produktstarts zu planen.

### <a name="work-with-buckets"></a>Arbeiten mit Buckets

Verwenden Sie Buckets, um Aufgaben nach Typ zu sortieren, wie beispielsweise die Planung von Ereignissen, Marketing-und Werbematerial sowie PR.

1. Wählen Sie  **neuen Bucket hinzufügen**  aus, und geben Sie eine neue Aufgabenkategorie ein.
2. Wählen Sie aus,  **+**  um einem Bucket Aufgaben hinzuzufügen, und geben Sie einen Namen ein.
3. Wählen Sie  **Aufgabe hinzufügen** aus.

### <a name="work-with-labels"></a>Arbeiten mit Beschriftungen

Verwenden Sie Bezeichnungen, um zu sehen, welche Aufgaben Ihre Aufmerksamkeit benötigen.

1. Geben Sie Namen für ihre Beschriftungen ein.
2. Wählen Sie ein farbiges Feld aus, um einer Aufgabe eine Beschriftung zu entfernen oder hinzuzufügen.

## <a name="add-a-comment"></a>Kommentar hinzufügen

Posten Sie einen Kommentar, um eine Aufgabe mit den Personen zu besprechen, denen Sie zugewiesen ist.

1. Geben Sie im Feld  **Kommentare**  eine Nachricht ein.
2. Wählen Sie  **senden** aus.

### <a name="keep-track-of-tasks"></a>Verwalten von Aufgaben

Verwenden Sie  **Diagramme**  und  **Brett**  Ansichten, um den Fortschritt der einzelnen Teammitglieder und des Gesamtprojekts zu überwachen oder Aufgaben zu sortieren und zu organisieren.

- Wählen Sie  **Diagramme**, dann **Gruppieren nach** und schließlich **Fälligkeitsdatum**  aus, um zu sehen, welche Aufgabe als nächstes fällig ist.
- Wählen Sie  **Brett**, dann **Gruppieren nach** und schließlich **Fortschritt**  aus, um Aufgaben nach dem Fortschritt zu sortieren.

**Hinweis:**  Aufgaben werden durch  **nicht gestartet**,  **in Bearbeitung** und  **abgeschlossen**  organisiert, sodass Sie wissen, wann Sie Ihr Produkt starten können.
