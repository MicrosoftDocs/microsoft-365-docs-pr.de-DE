# <a name="add-a-new-user"></a>Hinzufügen eines neuen Benutzers

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FOfN?autoplay=false]

Wenn eine neue Person Ihrem Unternehmen Beitritt, müssen Sie Sie Ihrem Microsoft 365 Business Premium-Abonnement als neuen Benutzer hinzufügen.

## <a name="try-it"></a>Probieren Sie es aus!

1. Wechseln Sie im Microsoft 365 Admin Center zu **Benutzerverwaltung**, und wählen Sie Benutzer hinzufügen aus.
1. Geben Sie den **Vornamen** und den **Namen** des neuen Benutzers ein. Der **Anzeigename** wird automatisch ausgefüllt, aber Sie können ihn ändern.
1. Geben Sie einen **Benutzernamen** ein. Dies ist die e-Mail-Adresse des neuen Benutzers. Wenn Sie über mehr als eine Domäne verfügen, wählen Sie in der Dropdownliste die richtige Domäne aus.
1. Wenn Sie ein Kennwort automatisch erstellen möchten, wählen Sie **Kennwort automatisch generieren** aus.
1. Wenn Sie festlegen möchten, dass der Benutzer sein Kennwort ändern muss, aktivieren Sie das Kontrollkästchen **diesen Benutzer zum Ändern des Kennworts bei der ersten Anmeldung anfordern** .
1. Wenn Sie möchten, dass die Kontodetails per e-Mail gesendet werden, aktivieren Sie das Kontrollkästchen **Kennwort in e-Mail nach Abschluss senden** . Ihr e-Mail-Name wird automatisch hinzugefügt, aber Sie können die Alternative e-Mail-Adresse des neuen Benutzers hinzufügen, damit Sie auch die Kontodetails erhalten. Wählen Sie **Weiter** aus.
1. Wählen Sie in der Dropdownliste Ihren Standort aus, und aktivieren Sie dann das Kontrollkästchen neben dem Abonnement, das der Benutzer verwenden soll. Wählen Sie **Nex** t aus.
1. Wenn der Benutzer Administrator sein soll, erweitern Sie **Rolle** s, deaktivieren Sie das Kontrollkästchen **Benutzer (kein Administratorzugriff)** , und wählen Sie in der Liste eine Administratorrolle aus. Weitere Informationen zu Administratorrollen und deren Einrichtung finden Sie unter [Was ist ein Administrator](what-is-admin.md)?
1. Geben Sie unter **Profilinformationen** Details zum Auftrag des Benutzers, zur Abteilung, zum Büro und so weiter ein. (Sie oder der neue Benutzer können diese Informationen später jederzeit hinzufügen.) Wählen Sie **weiter** aus.
1. Überprüfen Sie die Benutzer Details. Wenn Sie eine zusätzliche Lizenz für diesen Benutzer erwerben müssen, wird auf der Seite Überprüfung eine Benachrichtigung angezeigt. Wählen Sie **Hinzufügen beenden** aus, und wählen Sie dann **Schließen** aus.
1. Stellen Sie sicher, dass der Benutzer erfolgreich erstellt wurde: Wählen Sie im Microsoft 365 Admin Center im linken Navigationsbereich **Benutzer** und dann **aktive Benutzer** aus. Der neue Benutzer sollte zusammen mit der Ihnen zugewiesenen Lizenz aufgeführt werden.
