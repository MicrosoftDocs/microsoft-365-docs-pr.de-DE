# <a name="set-up-multifactor-authentication-on-your-phone"></a>Einrichten der mehrstufigen Authentifizierung auf Ihrem Telefon

## <a name="watch-set-up-multifactor-authentication-with-a-mobile-device"></a>Überwachung: Einrichten der mehrstufigen Authentifizierung mit einem mobilen Gerät

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2MmQR?autoplay=false]

Die mehrstufige Authentifizierung bietet mehr Sicherheit für Ihr Unternehmen. Führen Sie die folgenden Schritte aus, um sie einzurichten.

## <a name="try-it"></a>Probieren Sie es aus!

1. Wenn Sie sich [https://office.com](https://office.com) anmelden, wird die Eingabeaufforderung **"Weitere Informationen" angezeigt.** Wählen Sie **Weiter** aus.
1. Wählen Sie unter **Schritt 1** die **mobile App** aus der Dropdownliste **"Authentifizierungstelefon"** aus.
1. Wählen Sie unter "Wie soll die mobile App verwendet werden?" die Methode aus, mit der Sie sich anmelden möchten:
    - Wählen Sie **"Empfangen von Benachrichtigungen zur Überprüfung"** aus, um sich direkt von Ihrer mobilen App aus zu authentifizieren, was die Authentifizierung per Fingerabdruck umfassen kann.
    - Wählen Sie **"Überprüfungscode verwenden"** aus, um bei jeder Authentifizierung einen neuen Überprüfungscode einzugeben.
1. Wählen Sie **Einrichten** aus.
1. Lassen Sie das Fenster **"Mobile App konfigurieren"** auf Ihrem Computer geöffnet.
1. Wechseln Sie auf Ihrem mobilen Gerät zum App Store, suchen Sie nach "Microsoft Authenticator", wählen Sie es aus, installieren Sie es, und öffnen Sie es.
1. Befolgen Sie in Authenticator die Eingabeaufforderungen, um den Assistenten abzuschließen, und wählen Sie dann das Pluszeichen (+) aus, um Ihr Konto hinzuzufügen.
1. Wählen Sie **"Geschäfts-, Schul- oder Unikonto" aus,** wählen Sie **"Zulassen"** aus, wenn Sie zur Berechtigung zum Aufnehmen von Bildern und Aufzeichnen von Videos aufgefordert werden, und folgen Sie dann den Anweisungen, um den QR-Code im geöffneten Fenster auf Ihrem Computer zu scannen.
1. Nachdem das Konto hinzugefügt wurde, wählen Sie **"Got It"** aus.
1. Wählen Sie auf Ihrem Computer **"Weiter"** und dann erneut **"Weiter"** aus.
1. Notieren Sie sich den Code, der auf Ihrem mobilen Gerät angezeigt wird, geben Sie den Code auf Ihrem Computer in das Feld unter **Schritt 2** ein, und wählen Sie dann **"Überprüfen"** aus.
1. Geben Sie in den Feldern unter **Schritt 3** eine Sicherungstelefonnummer ein, z. B. Ihre Büronummer, klicken Sie auf **"Weiter"** und dann auf **"Fertig".**
