# <a name="move-files-to-onedrive"></a>Verschieben von Dateien in OneDrive

> [!VIDEO https://www.microsoft.com/videoplayer/embed/d74b083c-1f44-43ea-8a14-2e1fc600b341?autoplay=false]

Nachdem Sie sich für Microsoft 365 Business Premium registrieren,&#39;Sie Ihre vorhandenen Arbeitsdateien in die OneDrive for Business. OneDrive ist ein guter Ort, um persönliche Arbeitsdateien oder vertrauliche Geschäftsdateien zu speichern, auf die Sie den Zugriff steuern möchten.

## <a name="try-it"></a>Probieren Sie es aus!

1. Wählen Windows die Schaltfläche **Start** aus.
2. Suchen Sie nach OneDrive, und wählen Sie sie aus.
3. Geben Sie Ihre E-Mail-Adresse für Ihre Arbeit ein.
4. Wählen **Sie Anmelden** aus, und wählen Sie dann Weiter **aus.**
5. Gehen Sie durch das kurze Lernprogramm, und wählen Sie **dann Öffnen sie OneDrive Ordner aus.** Wenn Sie auch die persönliche Version von OneDrive verwenden,&#39;sie hier sehen.
6. Wechseln Sie im Datei-Explorer zu den Dateien, die Sie kopieren möchten.
7. Wählen Sie die Dateien aus, und ziehen Sie sie dann in den Ordner OneDrive ablegen. Blaue kreisförmige Pfeile neben Ihren Dateien bedeuten, dass&#39;mit der Cloud synchronisiert werden können. Wenn sie&#39;Synchronisierung erneut durchgeführt haben, werden grüne Häkungen angezeigt.

Zusätzlich zum lokalen Arbeiten OneDrive Dateien können Sie über Ihren Webbrowser darauf zugreifen:

1. Melden Sie sich in Ihrem Webbrowser bei office.com E-Mail-Adresse an.
2. Wählen Sie **OneDrive** aus. Alle Ihre Dateien sind aufgelistet, und Sie können sie auf jedem Beliebigen Computer im Browser bearbeiten.
