# <a name="overview-of-sharing-business-files"></a>Übersicht über die Freigabe von Geschäftsdateien

Sie müssen Dateien für Kunden, Kunden, Partner, Lieferanten und andere Personen in Ihrem Unternehmen freigeben. Wie können Sie Dateien freigeben, damit jeder Sie finden, öffnen und bearbeiten kann?

|--|--|--|

## <a name="share-a-file-with-someone-outside-of-your-company"></a>Freigeben einer Datei für eine Person außerhalb Ihres Unternehmens

- Geben Sie eine Datei für eine Person außerhalb Ihres Unternehmens (beispielsweise einen Client oder einen Kunden) per e-Mail frei, wenn Sie Sie nur auf eine Weise wechseln müssen.
- Wenn Sie eine Antwort benötigen oder diese Informationen ausfüllen und zurücksenden möchten, können Sie SharePoint oder Microsoft Teams verwenden, um Sie mit einem Bearbeitungslink freizugeben.

Erfahren Sie mehr darüber, wie Sie [Dateien außerhalb Ihrer Organisation](securely-share-files-externally.md) mit sicheren Links freigeben und wie Sie [Dateien für einen beliebigen Link freigeben](share-files-externally.md).

## <a name="share-and-collaborate-with-a-client-or-business-partner"></a>Freigeben und Zusammenarbeit mit einem Client oder Geschäftspartner

Freigabe im Rahmen eines Projekts? Oder als Teil einer laufenden Beziehung mit einem Geschäftspartner oder Lieferanten? Sie können nur ein wenig mehr Zeit einlegen und ein Team in Microsoft Teams einrichten und eine fortlaufende Freigabe und Zusammenarbeit tätigen. Es ist ganz einfach, mit Personen außerhalb Ihres Unternehmens zu arbeiten, und Sie können sogar separate Kanäle für zusätzliche Lieferanten oder Partner einrichten, damit alles direkt bleibt.

1. Erstellen Sie das Team.
1. Laden Sie Ihre Dateien hoch.
1. Laden Sie Ihre Partner oder Zulieferer in das Team als "Gast"-Benutzer ein, um Ihnen Zugriff zu gewähren.

Erfahren Sie mehr darüber, wie Sie [Dateien außerhalb Ihrer Organisation mit Secure Link s freigeben](https://support.microsoft.com/office/7266f44e-3e06-4736-b9d3-0580c24bba34).

## <a name="share-inside-your-business"></a>Freigeben innerhalb Ihres Unternehmens

Wenn Sie mit Dateien arbeiten, die Sie innerhalb Ihres Unternehmens freigeben, empfiehlt es sich auch, ein Team in Microsoft Teams einzurichten. Geben Sie Ihre Dateien dort frei, und jeder in Ihrem Unternehmen kann Sie leicht finden und bearbeiten, und Sie wissen, dass Sie alle auf derselben Seite arbeiten.

1. Erstellen Sie das Team, und laden Sie Ihr Team dazu ein.
1. Laden Sie Ihre Dateien hoch.

Erfahren Sie mehr über die Speicherung von [Unternehmensdateien](https://support.microsoft.com/office/e4d98e10-3532-4eed-85d6-92728454e32b) und die [Speicherung von Mitarbeitern](https://support.microsoft.com/office/12dbe3e4-dbef-48f8-a90e-87f1bc607073).
