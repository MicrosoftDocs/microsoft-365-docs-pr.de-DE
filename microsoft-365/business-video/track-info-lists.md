# <a name="use-microsoft-lists-to-track-business-info"></a>Verwenden von Microsoft-Listen zum Nachverfolgen von Geschäftsinformationen

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4MnGM?autoplay=false]

## <a name="try-it"></a>Ausprobieren

Nachverfolgen von Informationen und Organisieren der Arbeit mit Microsoft Lists. Erstellen Sie eine Liste von Grund auf neu, aus Excel, aus einer vorhandenen Liste oder aus einer Vorlage. Sie können mit Microsoft 365, Microsoft Teams oder SharePoint beginnen.

### <a name="add-an-item-to-the-list"></a>Hinzufügen eines Elements zur Liste

1. Wählen Sie **Neu** aus.
1. Fügen Sie Details zur Bestellung hinzu.
1. Klicken Sie auf **Speichern**.

### <a name="share-the-list-with-partners-or-coworkers"></a>Freigeben der Liste für Partner oder Kollegen

1. Wählen Sie über der Liste Freigeben **aus.**
1. Geben Sie ein, für wen Sie freigeben möchten. Es kann sich um eine Person oder eine Gruppe sein.
1. Wählen Sie die Berechtigungen aus.
1. Wählen Sie **Zugriff gewähren aus.**

### <a name="add-formatting-to-your-list"></a>Hinzufügen von Formatierungen zu Ihrer Liste

Formatieren Sie eine Spalte, um die Details in der Spalte zu markieren:

1. Wählen Sie die Spaltenüberschrift aus, wählen Sie die von Ihnen ausgewählte Option aus, z. B. eine Hintergrundfarbe oder abgerundete Ecken.
1. Wählen **Sie Speichern** aus, um die Änderungen anzuwenden.

### <a name="add-rules-to-alert-coworkers-about-list-updates"></a>Hinzufügen von Regeln zum Warnen von Kollegen über Listenupdates

1. Wählen Sie oberhalb der Liste **Automatisieren aus.**
1. Wählen **Sie Regel erstellen aus.**
1. Wählen Sie die entsprechende Bedingung aus, um die Regel auszulösen, die Sie erstellen möchten.
1. Passen Sie die Regel mit den spezifischen Informationen an, die Sie benötigen, z. B. die Namen der Personen, die beim Aktualisieren der Liste benachrichtigt werden sollen.
1. Wählen **Sie Erstellen** aus, um die Regel zu speichern.

### <a name="sort-and-group-list-items"></a>Sortieren und Gruppieren von Listenelementen

Sie können eine Liste von Bestellungen auf verschiedene Weise priorisieren:

- Um nach Stichtag zu sortieren, wählen Sie die Bestellung nach Spalte erfüllen aus, und wählen Sie dann Ältere bis Neuer aus.
- Wählen Sie zum Gruppieren von Bestellungen nach Kunden die Spalte Kundentyp aus, und wählen Sie Gruppe nach Kundentyp aus.

### <a name="update-an-items-status"></a>Aktualisieren des Status eines Elements

Sie können den Fortschritt sofort nachverfolgen:

- Doppelklicken Sie auf ein Element, um es zu öffnen, und aktualisieren Sie den Status der Bestellung.
