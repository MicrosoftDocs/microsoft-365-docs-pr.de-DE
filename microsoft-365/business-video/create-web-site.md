# <a name="create-a-website"></a>Erstellen einer Website

> [!VIDEO https://www.microsoft.com/videoplayer/embed/4839abc6-9323-4cbf-a79d-2907235f9ebb?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Microsoft 365 enthält keine öffentliche Website für Ihr Unternehmen. Um eine Einrichtung zu erstellen, erwägen Sie die Verwendung eines Webhostingunternehmens wie GoDaddy, Wix oder WordPress.

1. Suchen Sie nach einem Webhostinganbieter, in dem Sie Ihre Website erstellen können.

     Möglicherweise ist es am einfachsten, den Website-Generator im selben Unternehmen zu wählen, in dem Ihr Domänenname registriert ist.
1. Entwerfen Sie Ihre Unternehmenswebsite, und stellen Sie sicher, dass Sie Ihr Unternehmenslogo, Ihren Geschäftszweck, Text, Bilder und Links enthalten.
1. Um die Website in Microsoft 365 zu integrieren, können Sie Ihre Geschäftliche E-Mail zur Website und einen Link zu Microsoft Teams, SharePoint oder einem anderen Tool für die Zusammenarbeit hinzufügen.
1. Stellen Sie sicher, dass Sie Ihre Website mit Ihrem Domänennamen verbinden, den&#39;auch für E-Mails auf Microsoft 365 verwenden möchten.
