# <a name="create-a-website"></a>Erstellen einer Website

> [!VIDEO https://www.microsoft.com/videoplayer/embed/4839abc6-9323-4cbf-a79d-2907235f9ebb?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Microsoft 365 enthält keine öffentliche Website für Ihr Unternehmen. Um einen einzurichten, sollten Sie eine Webhosting-Firma wie GoDaddy, WiX oder WordPress verwenden.

1. Suchen Sie nach einem Webhosting-Anbieter, in dem Sie Ihre Website erstellen können.

     Möglicherweise ist es am einfachsten, den Website-Generator bei demselben Unternehmen zu wählen, in dem Ihr Domänenname registriert ist.
1. Entwerfen Sie Ihre Business-Website, und stellen Sie sicher, dass Sie Ihr Firmen Logo, Geschäftszweck, Text, Bilder und Links einschließen.
1. Um die Website mit Microsoft 365 zu integrieren, können Sie Ihre geschäftlichen e-Mail-Adresse zur Website hinzufügen und einen Link zu Microsoft Teams, SharePoint oder andere Tools für die Zusammenarbeit.
1. Stellen Sie sicher, dass Sie Ihre Website mit Ihrem Domänennamen verbinden, den Sie&#39;auch für e-Mail unter Microsoft 365 verwenden.
