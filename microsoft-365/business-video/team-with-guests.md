# <a name="create-a-team-with-guests"></a>Erstellen eines Teams mit Gästen

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FQMp?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Sie können Microsoft Teams verwenden, um gemeinsam an einem Projekt mit ihren Mitarbeitern und Personen außerhalb des Unternehmens wie Ihren Kunden zu arbeiten.

1. Wählen Sie in Microsoft  **Teams in der**  linken Navigationsleiste aus, und wählen Sie dann  **Team erstellen** aus.
2. Geben Sie im Feld  **Team erstellen**  einen Teamnamen ein, geben Sie eine kurze Beschreibung ein, wählen Sie eine Datenschutzeinstellung aus, und klicken Sie dann auf  **weiter**.
3. Geben Sie im Feld  **Mitglieder hinzufügen**  die Namen anderer Mitarbeiter in Ihrer Organisation ein. Für externe Gäste geben Sie Ihre e-Mail-Adresse ein.

    Wenn Sie eine Nachricht erhalten, dass Sie keinen Gast hinzufügen können, besuchen Sie die Teams und das Skype Admin Center, um den Gastzugriff zu aktivieren.

1. Wählen Sie  **Hinzufügen** und dann  **Schließen** aus.
2. Geben Sie unten auf der Seite eine kurze Willkommensnachricht für Ihr neues Team ein, und wählen Sie dann **senden** aus :::image type="icon" source="../media/teams-send-icon.png" border="false"::: .   

Ihr Client erhält eine e-Mail-Einladung von Microsoft Teams, in der Sie darüber informiert werden, dass ein kostenloses Microsoft-Konto für den Beitritt zum Team benötigt wird. Sie können ein Konto mithilfe der vorhandenen e-Mail-Adresse erstellen, indem Sie die angegebenen Anweisungen befolgen. Anschließend können Sie Microsoft Teams im Internet verwenden oder die Teams-App auf Ihrem Computer installieren.
