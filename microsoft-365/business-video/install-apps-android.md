# <a name="install-outlook-and-office-apps-on-android-devices"></a>Installieren von Outlook- und Office-Apps auf Android-Geräten

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfquk?autoplay=false]

Wenn Ihr Android-Gerät von Microsoft 365 Business Premium verwaltet wird, werden Sie möglicherweise aufgefordert, beim ersten Installieren einer Office-App zusätzliche Sicherheit zu einrichten. 

## <a name="try-it"></a>Probieren Sie es aus!

1. Wählen Sie auf Ihrem Gerät den Google Play Store aus.
2. Suchen Sie nach der Office-App, die Sie installieren möchten. Wir&#39;verwenden Outlook für dieses Beispiel.
3. Wählen Sie auf dem Willkommensbildschirm von Outlook die Option **Erste Schritte aus.**
4. Geben Sie **auf der** Seite Konto hinzufügen Ihre Microsoft 365-E-Mail-Adresse ein, und wählen Sie dann **Weiter aus.**
5. Geben **Sie unter Kennwort eingeben** Ihr Kennwort ein, und wählen Sie dann Anmelden **aus.**
6. Wählen Sie auf der nächsten Seite  **Weiter**  aus, wenn Sie ein anderes Konto hinzufügen möchten. Wählen Sie andernfalls  **Überspringen** aus.
7. Wenn Sie zum Hinzufügen der Intune-Unternehmensportal-App aufgefordert werden, wählen **Sie Zum Speichern wechseln aus.**
8. Installieren und öffnen Sie die Intune-Unternehmensportal-App, und wählen Sie dann  **OK**  aus, wenn die Meldung angezeigt wird, dass Ihre Organisation Daten in dieser App schützt.
9. Wenn Sie nach einer PIN gefragt werden, geben Sie eine 4-stellige PIN ein, die Sie&#39;zum Entsperren Ihres Telefons verwenden möchten, wählen Sie Senden **aus,** bestätigen Sie Ihre PIN, und wählen Sie dann erneut Senden **aus.**
