# <a name="create-a-company-wide-email-signature"></a>Erstellen einer unternehmensweiten e-Mail-Signatur

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1IEWf?autoplay=false]

Eine unternehmensweite e-Mail-Signatur wird in jeder e-Mail angezeigt, die von Personen in Ihrer Organisation gesendet wird. Sie können damit wichtige Details wie die Kontaktinformationen Ihres Unternehmens oder einen rechtlichen Haftungsausschluss anzeigen. 

## <a name="try-it"></a>Probieren Sie es aus!

1. Wählen Sie im Microsoft 365 Admin Center die Option **Exchange** aus.
1. Wählen Sie **Nachrichtenfluss** aus.
1. Wählen Sie **Add +** aus, und klicken Sie dann auf **Haftungsausschlüsse anwenden**.
1. Auf der Seite **neue Regel** :
    1. Geben Sie einen Namen für die Regel ein.
    1. Wählen Sie in der Dropdownliste **diese Regel anwenden** aus die Option **auf alle Nachrichten anwenden** aus.
    1. Überprüfen Sie in der Dropdownliste **ausführen die folgende** , ob **die Haftungsausschluss anfügen** angezeigt wird.
    1. Wählen Sie rechts auf der Seite **Text eingeben** aus, und geben Sie im Textfeld **Disclaimer angeben** den Text für Ihre e-Mail-Signatur ein. Sie können das Aussehen Ihrer Signatur verbessern, indem Sie den Text mit HTML formatieren.
    1. Wenn ein Bild in Ihrer Signatur angezeigt werden soll, müssen Sie eine öffentlich verfügbare URL für das Bild verwenden. Navigieren Sie zu dem Bild im Internet, klicken Sie mit der rechten Maustaste darauf, und wählen Sie **Bildadresse kopieren** aus. Fügen Sie die Adresse in das Textfeld **Haftungsausschluss angeben** ein. Wählen Sie **OK** aus, und Scrollen Sie nach unten.
    1. Um sicherzustellen, dass die Signatur mit verschlüsselten e-Mails funktioniert, fügen Sie eine Fallback-Option hinzu. Klicken Sie rechts auf der Seite auf **auswählen**, dann auf **Umbruch**, und wählen Sie dann **OK** aus.
    1. Scrollen Sie nach unten, und lassen Sie den Modus auf **erzwingen** festgelegt, und wählen Sie dann **Speichern** aus.
1. Eine Warnmeldung wird angezeigt. Wählen Sie **Ja** aus, um die Regel auf alle zukünftigen Nachrichten anzuwenden.

    Ihre Signatur wurde erstellt. Wenn Sie Ihre nächste e-Mail senden, wird die soeben erstellte Signatur nicht angezeigt, aber die e-Mail-Empfänger werden Sie sehen.
