# <a name="create-a-company-wide-email-signature"></a>Erstellen einer unternehmensweiten E-Mail-Signatur

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1IEWf?autoplay=false]

Eine unternehmensweite E-Mail-Signatur wird in jeder E-Mail angezeigt, die von Personen in Ihrer Organisation gesendet wird. Sie können es verwenden, um wichtige Details anzuzeigen, z. B. Ihre Unternehmenskontaktinformationen oder einen Haftungsausschluss. 

## <a name="try-it"></a>Probieren Sie es aus!

1. Wählen Sie im Microsoft 365 Admin Center Die Option **Exchange aus.**
1. Wählen **Sie Nachrichtenfluss aus.**
1. Wählen **Sie Hinzufügen +** aus, und wählen Sie dann **Haftungsausschlüsse anwenden aus.**
1. Auf der **Seite Neue Regel:**
    1. Geben Sie einen Namen für die Regel ein.
    1. Wählen Sie **in der Dropdownliste Diese Regel anwenden** auf alle Nachrichten anwenden **aus.**
    1. Überprüfen Sie **in der Dropdownliste** Gehen Sie wie folgt vor, ob Der Haftungsausschluss **anfügen** angezeigt wird.
    1. Wählen Sie rechts auf der Seite Text eingeben **aus,** und geben Sie dann den Text für Ihre E-Mail-Signatur in das Textfeld **Haftungsausschluss** angeben ein. Sie können das Aussehen Ihrer Signatur verbessern, indem Sie den Text mit HTML formatieren.
    1. Wenn ein Bild in Ihrer Signatur angezeigt werden soll, müssen Sie eine öffentlich verfügbare URL für dieses Bild verwenden. Navigieren Sie zum Bild im Web, klicken Sie mit der rechten Maustaste darauf, und wählen Sie **Bildadresse kopieren aus.** Fügen Sie die Adresse in das Textfeld **Haftungsausschluss** angeben ein. Wählen **Sie OK** aus, und scrollen Sie dann nach unten.
    1. Um sicherzustellen, dass die Signatur mit verschlüsselten E-Mails funktioniert, fügen Sie eine Ausweichoption hinzu. Wählen Sie rechts auf der Seite Auswählen **aus,** wählen Sie **Umschließen** aus, und wählen Sie dann **OK aus.**
    1. Scrollen Sie nach unten, und lassen Sie den Modus auf **Erzwingen** festgelegt, und wählen Sie dann **Speichern aus.**
1. Es wird eine Warnmeldung angezeigt. Wählen **Sie Ja** aus, um die Regel auf alle zukünftigen Nachrichten anzuwenden.

    Ihre Signatur wurde erstellt. Wenn Sie Ihre nächste E-Mail senden, wird die gerade erstellte Signatur nicht mehr zu sehen sein, aber die E-Mail-Empfänger sehen sie.
