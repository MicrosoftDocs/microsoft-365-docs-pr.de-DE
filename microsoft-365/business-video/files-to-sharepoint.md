# <a name="move-files-to-sharepoint"></a>Verschieben von Dateien zu SharePoint

> [!VIDEO https://www.microsoft.com/videoplayer/embed/65ec460e-296c-458f-b004-906756dfb3d0?autoplay=false]

Nachdem Sie sich für Microsoft 365 Business Premium registrieren,&#39;Ihre Unternehmensarbeitsdateien nach SharePoint kopieren. SharePoint ist ein guter Ort zum Speichern freigegebener Unternehmensdateien, auf die jeder Zugriff benötigt. Dadurch wird in der Regel die Verwendung einer Dateifreigabe oder eines Netzlaufwerks ersetzt.

## <a name="try-it"></a>Probieren Sie es aus!

1. Öffnen Sie Microsoft Teams, und öffnen Sie dann ein Team, auf das jeder in Ihrem Unternehmen Zugriff hat.
2. Wählen **Sie "Dateien"** und dann **"In SharePoint öffnen" aus.**
3. Wählen  **Sie "Dokumente"** aus, um zu dem Speicherort zu wechseln, an dem alles auf der Website gespeichert ist.
4. Wählen Sie **"Synchronisieren"** und dann **"Ja" aus.** Dadurch werden die Dateien der SharePoint-Website mit Ihrem Computer synchronisiert.
5. Im Datei-Explorer sehen&#39;nun den Ordner synchronisierter Dokumente von Ihrem Unternehmen&#39;SharePoint-Website. Wenn Sie zusätzliche SharePoint-Websites synchronisieren, werden diese oberhalb oder unterhalb dieser angezeigt. Klicken Sie mit der rechten Maustaste auf den Ordner synchronisierter Dokumente, und öffnen Sie ihn in einem neuen Fenster, um ihn neben den Unternehmensdateien anzuzeigen, die Sie verschieben möchten.
6. Wählen Sie alle Dateien aus, die Sie verschieben möchten, und ziehen Sie sie dann in den synchronisierten SharePoint-Ordner. Blaue kreisförmige Pfeile neben Ihren Dateien bedeuten, dass&#39;mit der SharePoint-Website synchronisiert werden können. Wenn sie&#39;erneut synchronisiert haben, werden grüne Häkungen angezeigt.
7. Kehren Sie zu Ihrer SharePoint-Website zurück, um die Dateien zu sehen, die mit der Cloud synchronisiert wurden.

Sie können nicht nur direkt von Ihrem Computer aus an Ihren Dateien arbeiten, sondern auch in einem Webbrowser auf einem beliebigen Computer darauf zugreifen.

Sie können jetzt über Ihre SharePoint-Website oder Teams auf Ihre Dateien zugreifen.
