# <a name="move-files-to-sharepoint"></a>Verschieben von Dateien in SharePoint

> [!VIDEO https://www.microsoft.com/videoplayer/embed/65ec460e-296c-458f-b004-906756dfb3d0?autoplay=false]

Nachdem Sie sich für Microsoft 365 Business Premium registrieren,&#39;Sie Ihre Unternehmensarbeitsdateien in die SharePoint. SharePoint ist ein guter Ort, um freigegebene Unternehmensdateien zu speichern, auf die jeder Zugriff benötigt. Dadurch wird in der Regel die Verwendung einer Dateifreigabe oder eines Netzwerklaufwerks ersetzt.

## <a name="try-it"></a>Probieren Sie es aus!

1. Öffnen Microsoft Teams, und öffnen Sie dann ein Team, auf das jeder in Ihrem Unternehmen Zugriff hat.
2. Wählen **Sie Dateien** aus, und wählen Sie dann Öffnen in **SharePoint**.
3. Wählen  **Sie Dokumente** aus, um zu dem Speicherort zu wechseln, an dem alles auf der Website gespeichert ist.
4. Wählen **Sie Synchronisieren** aus, und wählen Sie dann Ja **aus.** Dadurch werden die Dateien des SharePoint mit Ihrem Computer synchronisiert.
5. Im Datei-Explorer&#39;sie nun den Ordner synchronisierte Dokumente von Ihrem Unternehmen&#39;der SharePoint sehen. Wenn Sie zusätzliche websites SharePoint synchronisieren, werden diese oberhalb oder unterhalb dieser angezeigt. Klicken Sie mit der rechten Maustaste auf den Ordner synchronisierte Dokumente, und öffnen Sie ihn in einem neuen Fenster, um ihn nebeneinander mit den Unternehmensdateien anzuzeigen, die Sie verschieben möchten.
6. Wählen Sie alle Dateien aus, die Sie verschieben möchten, und ziehen Sie sie dann in den synchronisierten SharePoint Ordner. Blaue kreisförmige Pfeile neben Ihren Dateien bedeuten, dass&#39;erneut mit dem SharePoint synchronisiert werden. Wenn sie&#39;Synchronisierung erneut durchgeführt haben, werden grüne Häkungen angezeigt.
7. Kehren Sie zu SharePoint, um die Dateien zu sehen, die mit der Cloud synchronisiert wurden.

Sie können nicht nur direkt von Ihrem Computer aus an Ihren Dateien arbeiten, sondern auch in einem Webbrowser auf jedem Beliebigen Computer darauf zugreifen.

Sie können jetzt über Ihre Website oder SharePoint auf Ihre Dateien Teams.
