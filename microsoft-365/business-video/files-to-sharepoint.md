# <a name="move-files-to-sharepoint"></a>Dateien in SharePoint migrieren

> [!VIDEO https://www.microsoft.com/videoplayer/embed/65ec460e-296c-458f-b004-906756dfb3d0?autoplay=false]

Nachdem Sie sich für Microsoft 365 Business Premium angemeldet haben, möchten Sie Ihre Unternehmens Arbeitsdateien in SharePoint kopieren&#39;. SharePoint ist ein guter Ort zum Speichern freigegebener Unternehmensdateien, auf die jeder Zugriff benötigt. Dies ersetzt normalerweise die Verwendung einer Dateifreigabe oder eines Netzlaufwerks.

## <a name="try-it"></a>Probieren Sie es aus!

1. Öffnen Sie Microsoft Teams, und öffnen Sie ein Team, auf das jeder in Ihrem Unternehmen Zugriff hat.
2. Wählen Sie  **Dateien** aus, und wählen Sie dann  **in SharePoint öffnen** aus.
3. Wählen Sie  **Dokumente** aus, um zu dem Speicherort zu wechseln, an dem alles auf der Website gespeichert ist.
4. Wählen Sie  **Synchronisieren** aus, und klicken Sie dann auf  **Ja**. Dadurch werden die Dateien der SharePoint-Website mit Ihrem Computer synchronisiert.
5. Im Datei-Explorer&#39;Sie nun den Ordner "synchronisierte Dokumente" von der SharePoint-Website Ihres unter&#39;nehmens anzeigen. Wenn Sie zusätzliche SharePoint-Websites synchronisieren, werden diese oberhalb oder unterhalb dieser angezeigt. Klicken Sie mit der rechten Maustaste auf den Ordner synchronisierte Dokumente, und öffnen Sie ihn in einem neuen Fenster, um ihn nebeneinander mit den zu verschiebenden Unternehmensdateien anzuzeigen.
6. Wählen Sie alle Dateien aus, die Sie verschieben möchten, und ziehen Sie Sie dann in den synchronisierten SharePoint-Ordner. Blaue Kreis Pfeile neben ihren Dateien bedeuten, dass Sie&#39;erneut mit der SharePoint-Website synchronisieren. Wenn Sie die Synchronisierung&#39;erneut durchgeführt haben, werden grüne Häkchen angezeigt.
7. Kehren Sie zu Ihrer SharePoint-Website zurück, um die Dateien anzuzeigen, die mit der Cloud synchronisiert wurden.

Sie können nicht nur an Ihren Dateien direkt von Ihrem Computer arbeiten, sondern auch auf einem beliebigen Computer in einem Webbrowser darauf zugreifen.

Sie können nun von Ihrer SharePoint-Website oder ihren Teams aus auf Ihre Dateien zugreifen.
