# <a name="switch-from-google-workspace---overview"></a>Wechseln von Google Workspace – Übersicht

Führen Sie die folgenden Schritte aus, um Ihre Daten, E-Mails und Benutzer von Google Workspace zu Microsoft 365 Business zu verschieben.


| Schritt  |Beschreibung  |
|---------|---------|
|Schritt 1 |  [Registrieren für](../sign-up.md) Microsoft 365 Business Premium       |
|Schritt 2 |   [Richten Sie die Migration von Microsoft 365 für Google Workspace ein.](set-up-microsoft-365-forgoogle.md) </br> In diesem Schritt überprüfen Sie, ob Sie die Domäne besitzen, die Sie in Google Workspace verwenden, Benutzer hinzufügen und grundlegende Sicherheit für Geräte einrichten. |
|Schritt 3 | [Richten Sie Sicherheitsrichtlinien für Windows-Geräte ein.](../secure-win10-pcs.md)</br> Die Sicherheit von Windows wird separat auf der Seite "Setup" im Microsoft 365 Admin Center eingerichtet. |
|Schritt 4|[Hinzufügen Ihrer Google Workspace-Domäne zu Microsoft 365](add-google-domain.md) </br> Nachdem Sie überprüft haben, dass Sie derEn Domäne besitzen, die Sie für E-Mails verwenden, können Sie und alle anderen Benutzer ihre alte E-Mail verwenden, um sich bei Microsoft 365 Business Premium zu anmelden. |
|Schritt 5 | [Installieren Sie Office-Apps und Microsoft Teams.](../install-office.md)</br> Alle Personen, die über Microsoft 365-Lizenzen verfügen, sollten die Office-Apps auf ihren Arbeitsgeräten installieren.|
|Schritt 6 | [Migrieren Sie die E-Mail- und Kalenderelemente aller Benutzer.](migrate-email.md)</br> In diesem Schritt führen Sie eine Exchange Online-Migration aus, um alle E-Mails, Kalender und Kontakte von Google Workspace zu verschieben.  |
|Schritt 7 | [Verbinden Sie die Domäne mit Microsoft 365.](connect-domain-tom365.md) </br> Nachdem Sie die Verbindung hergestellt haben, wird die Domänen-E-Mail an Microsoft 365 gesendet, und alle Ihre Microsoft 365-Dienste funktionieren.|
|Schritt 8|Verwenden [Sie Mover, um alle Daten](mover-migrate-files.md) von Laufwerk zu OneDrive und von freigegebenen Laufwerken auf Teamwebsites zu verschieben.</br> In diesem Schritt werden alle Daten auf persönlichen und freigegebenen Laufwerken kopiert und nach Microsoft 365 verschoben.|
|Schritt 9| [Stellen Sie Google Workspace ein,](cancel-google.md) aber behalten Sie Ihre Domäne bei. </br> Wenn Google Ihre Domäne verwaltet, können Sie sie dort auch behalten, nachdem Sie Ihr Google Workspace-Abonnement eingestellt haben. Sie können ihn auch auf einen anderen DNS-Host verschieben, wenn Sie möchten.|
