# <a name="switch-from-google-workspace---overview"></a>Wechseln von Google Workspace – Übersicht

Führen Sie die folgenden Schritte aus, um Ihre Daten, E-Mails und Benutzer von Google Workspace in Microsoft 365 zu verschieben.


| Schritt  |Beschreibung  |
|---------|---------|
|Schritt 1 |  [Registrieren sie](../sign-up.md) sich für Microsoft 365 Business Premium       |
|Schritt 2 |   [Richten Sie Microsoft 365 für die Migration von Google Workspace ein.](set-up-microsoft-365-forgoogle.md) </br> In diesem Schritt überprüfen Sie, ob Sie die Domäne besitzen, die Sie in Google Workspace verwenden, Benutzer hinzufügen und grundlegende Sicherheit für Geräte einrichten. |
|Schritt 3 | [Richten Sie Sicherheitsrichtlinien für Windows ein.](../secure-win10-pcs.md)</br> Windows Sicherheit wird separat auf der Setupseite von Microsoft 365 Admin Center eingerichtet. |
|Schritt 4|[Hinzufügen Ihrer Google Workspace-Domäne zu Microsoft 365](add-google-domain.md) </br> Nachdem Sie überprüft haben, dass Sie derEnt der Domäne sind, die Sie für E-Mails verwenden, können Sie und alle anderen Benutzer ihre alten E-Mails verwenden, um sich bei Microsoft 365 Business anmelden. |
|Schritt 5 | [Installieren Office Apps und Microsoft Teams](../install-office.md).</br> Alle Personen, die über Microsoft 365 verfügen, sollten die Office auf ihren Arbeitsgeräten installieren.|
|Schritt 6 | [Migrieren Sie die E-Mail- und Kalenderelemente aller Benutzer.](migrate-email.md)</br> In diesem Schritt führen Sie eine Exchange Online aus, um alle E-Mails, Kalender und Kontakte von Google Workspace zu verschieben.  |
|Schritt 7 | [Verbinden Domäne, die Microsoft 365](connect-domain-tom365.md). </br> Nachdem Sie eine Verbindung hergestellt haben, wird die Domänen-E-Mail Microsoft 365, und alle Ihre Microsoft 365 funktionieren.|
|Schritt 8|Verwenden [Sie Mover, um alle Daten](mover-migrate-files.md) von Drive in OneDrive und von freigegebenen Laufwerken zu Teamwebsites zu verschieben.</br> In diesem Schritt werden alle Daten in persönlichen und freigegebenen Laufwerken kopiert und in Microsoft 365.|
|Schritt 9| [Beenden Sie Google Workspace,](cancel-google.md) behalten Sie jedoch Ihre Domäne bei. </br> Wenn Google Ihre Domäne verwaltet, können Sie sie auch dort behalten, nachdem Sie Ihr Google Workspace-Abonnement eingestellt haben. Sie können sie auch auf einen anderen DNS-Host verschieben, wenn Sie möchten.|
