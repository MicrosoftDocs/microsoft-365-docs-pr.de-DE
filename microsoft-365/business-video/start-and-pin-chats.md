# <a name="chat-with-employees-and-customers"></a>Chatten Sie mit Mitarbeitern und Kunden

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWedV6?autoplay=false]

Sie können intern mit Ihren Kollegen chatten oder mit einem Gastbenutzer, z. B. einem Kunden oder einem Partner, chatten. Wenn [der Gastzugriff für Ihr](https://docs.microsoft.com/microsoftteams/set-up-guests) Unternehmen aktiviert ist, können Sie einfach eine E-Mail-Adresse eingeben, um einen Gast zu einem Chat hinzuzufügen.

## <a name="try-it"></a>Probieren Sie es aus!

Starten Sie 1:1- oder Gruppenchats, um mit einer kleineren Gruppe zusammenzuarbeiten oder privat zu sprechen, und pinnen Sie Chats für schnellen Zugriff.

1. In  **Chats** können Sie 1:1 oder Gruppenchats mit Personen haben.
2. Wählen Sie **Neuer Chat aus.** :::image type="icon" source="../media/new-chat-icon.png" border="false":::  
3. Geben Sie den Namen der Person oder der Personen ein, die Sie kontaktieren möchten.
4. Wählen Sie den Pfeil nach unten aus, um Ihrer Chatgruppe einen Namen hinzuzufügen.

So nehmen Sie einen Video- oder Audioanruf aus einem Chat vor

- Wählen  **Sie Videoanruf** :::image type="icon" source="../media/video-call-icon.png" border="false"::: oder **Audioanruf aus,** :::image type="icon" source="../media/audio-call-icon.png" border="false"::: um jemanden direkt aus einem Chat an anruft.
