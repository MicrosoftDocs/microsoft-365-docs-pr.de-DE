# <a name="overview-of-the-microsoft-365-admin-center"></a>Übersicht über das Microsoft 365 Admin Center

- [Das Admin Center in vereinfachter Ansicht](#watch-the-admin-center-in-simplified-view)
- [Das Admin Center in der Dashboardansicht](#watch-the-admin-center-in-dashboard-view)

Das Microsoft 365 Admin Center hat zwei Ansichten: die vereinfachte Ansicht hilft kleineren Organisationen bei der Verwaltung ihrer am häufigsten verwendeten Aufgaben. Die Dashboardansicht enthält komplexere Einstellungen und Aufgaben. Sie können von einer Schaltfläche oben im Admin Center zwischen diesen wechseln.

## <a name="watch-the-admin-center-in-simplified-view"></a>Überwachung: Das Admin Center in der vereinfachten Ansicht

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWD3sq?autoplay=false]

Mit dem Microsoft 365 Admin Center können Sie Kennwörter zurücksetzen, Ihre Rechnung anzeigen, Benutzer hinzufügen oder entfernen und vieles mehr an einem Ort.

Melden Sie sich mit Ihrem Geschäftskonto bei Office.com an, und wählen Sie das App-Startfeld aus.

Wenn Sie über die Berechtigung zum Zugriff auf das Admin Center verfügen, wird der **Administrator** in der Liste angezeigt. Wählen Sie es aus.

Überprüfen Sie oben im Admin Center die wichtigsten Aktionen für Sie. Je nachdem, was Sie bereits eingerichtet haben, werden möglicherweise unterschiedliche Aktionen angezeigt, z. B. das Erstellen neuer Konten, das Verwenden von Teams, das Einrichten von E-Mails und das Installieren Office Apps.

Unter **"Ihre Organisation"** auf der Registerkarte **"Benutzer"** befindet sich eine Liste von Personen, die auf Apps und Dienste zugreifen, neue Benutzer hinzufügen, Kennwörter zurücksetzen oder das Menü mit drei Punkten (weitere Aktionen) verwenden können. Wählen Sie eine Person aus, um ihre Informationen und Einstellungen anzuzeigen oder zu bearbeiten.

Erstellen **Sie** auf der Registerkarte Teams ein neues Team, oder verwalten Sie vorhandene Teams. Sie können die Mitglieder eines Teams verwalten oder die drei Punkte (weitere Aktionen) auswählen, um andere Teams Einstellungen zu ändern.

Fügen Sie auf der Registerkarte **"Abonnements"** weitere Produkte hinzu, fügen Sie Lizenzen hinzu, oder verwenden Sie das Menü mit drei Punkten (weitere Aktionen), um Lizenzen oder Zahlungsmethoden zu ändern.

Durchsuchen Sie auf der Registerkarte **"Lernen"** Videos und Artikel über das Admin Center und andere Microsoft 365 Features. Um erweiterte Features des Admin Centers zu erkunden, öffnen Sie das Navigationsmenü, und erweitern Sie die Überschriften, um mehr zu sehen. Wählen Sie **"Alle anzeigen"** aus, um alles im Navigationsmenü anzuzeigen, oder verwenden Sie die Suchleiste, um schnell zu finden, wonach Sie suchen.

Wenn Sie Hilfe benötigen, wählen Sie **Hilfe & Support** aus. Suchen Sie nach einem Thema, zu dem Sie Hilfe benötigen, und zeigen Sie die empfohlene Lösung an, oder wählen Sie das Headset aus, um sich an den Support zu wenden, und geben Sie dann Ihre Frage und Kontaktinformationen ein.

## <a name="watch-the-admin-center-in-dashboard-view"></a>Überwachung: Das Admin Center in der Dashboardansicht

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfvDL?autoplay=false]

Die Microsoft 365 Admin Center ist der Ort, an dem Sie Ihr Unternehmen in der Cloud verwalten. Sie können Aufgaben wie das Hinzufügen und Entfernen von Benutzern, das Ändern von Lizenzen und das Zurücksetzen von Kennwörtern ausführen.

Spezielle Arbeitsbereiche wie Sicherheit oder Geräteverwaltung ermöglichen eine präzisere Steuerung. Weitere Informationen dazu, wie die Admin Center zusammenarbeiten, finden Sie unter ["Was ist mit den spezifischen Typen von IT-Rollen und anderen Arbeitsbereichen wie Sicherheit, Geräteverwaltung oder Exchange?"](#what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange) in diesem Artikel.

To get to the Microsoft 365 Admin Center, go to [admin.microsoft.com](https://admin.microsoft.com) or, if you're already signed in, select the app launcher, and choose **Admin**.

Auf der Startseite können Sie Karten für Aufgaben erstellen, die Sie häufig ausführen. Um eine neue Karte hinzuzufügen, wählen Sie **"Karte hinzufügen"** und dann das Pluszeichen neben der Karte aus, die Sie hinzufügen möchten. Schließen Sie nach Abschluss des Vorgangs das Fenster. Sie können die Karten neu anordnen, indem Sie sie auswählen und dann an den gewünschten Ort ziehen. Um eine Karte zu entfernen, wählen Sie die drei Punkte (weitere Aktionen) und dann **entfernen** aus.

Um weitere Administratoraufgaben anzuzeigen, erweitern Sie das Navigationsmenü. Erweiterte Konfigurationseinstellungen finden Sie unten in den zusätzlichen Admin Centern.

Eine häufige Aufgabe, die Sie im Admin Center ausführen können, ist das Hinzufügen eines Benutzers. Wählen Sie dazu **"Benutzer",** **"Aktive Benutzer"** und dann **"Benutzer hinzufügen"** aus. Geben Sie den Namen des Benutzers und andere Informationen ein, und wählen Sie dann **"Weiter"** aus. Folgen Sie den Aufforderungen, um das Hinzufügen des Benutzers abzuschließen. Wenn Sie fertig sind, wählen Sie **"Hinzufügen fertig stellen"** und dann **"Schließen"** aus.

Sie können Ihre aktiven Benutzer nach Spalten sortieren, z. B. **Anzeigename** oder **Lizenzen.** Wenn Sie weitere Spalten hinzufügen möchten, wählen **Sie "Spalten auswählen"** aus, wählen Sie die Spalten aus, die Sie hinzufügen möchten, und wählen Sie dann **"Speichern"** aus.

Wählen Sie einen Benutzer aus, um weitere Optionen anzuzeigen, z. B. die Verwaltung seiner Produktlizenzen.

Um weitere Features zu aktivieren, die in Ihrem Abonnement enthalten sind, wählen Sie **Setup** aus. Hier können Sie die Anmeldesicherheit, den Schutz mobiler Apps, DLP und andere Features aktivieren, die in Ihrem Abonnement enthalten sind.

Wenn Sie jederzeit Support benötigen, wählen Sie **"Hilfe benötigen"** aus. Geben Sie Ihre Frage ein, und sehen Sie sich die angezeigten Links an. Wenn Sie hier keine Antwort erhalten, wählen Sie **"Support kontaktieren"** aus, um eine Serviceanfrage zu öffnen.

Weitere Informationen zum Verwalten von Abrechnung, Kennwörtern, Benutzern und Administratoren finden Sie in den anderen Lektionen in diesem Kurs.

## <a name="who-is-an-admin"></a>Wer ist ein Administrator?

Standardmäßig erhält die Person, die sich registriert und ein Microsoft 365 für Unternehmen-Abonnement erwirbt, die Administratorberechtigungen. Diese Person kann anderen Personen Administratorberechtigungen zuweisen, damit diese Microsoft 365 für ihre Organisation verwalten können.

Wird die Meldung „**Sie haben keine Berechtigung für den Zugriff auf diese Seite oder das Ausführen dieser Aktion**“ angezeigt, dann sind Sie kein Administrator.

### <a name="who-has-admin-permissions-in-my-business"></a>Wer hat in meinem Unternehmen Administratorberechtigungen?
<a name="bkmk_admin"> </a>

Sie suchen Ihren Administrator, damit dieser Ihr Kennwort zurücksetzt, ein Konto löscht oder andere Aufgaben ausführt? So können Sie ihn kontaktieren:

- **Universitäten und Schulen**: Wenden Sie sich an Ihr technisches Supportteam. Normalerweise finden Sie einen Link auf der Website Ihrer Universität. Bei kleineren Schulen gibt es möglicherweise nur ein paar technisch geschulte Benutzer, die über Administratorberechtigungen verfügen.

- **Große Unternehmen**: Wenden Sie sich an das interne Helpdesk bzw. den technischen Support Ihres Unternehmens.

- **Kleine Unternehmen**: Wenden Sie sich an den Geschäftsinhaber/-mitbesitzer. Häufig vergeben diese Personen die Administratorberechtigungen an den IT-Berater, der sämtliche Computerwartungsaufgaben für das Unternehmen erledigt.

Wenn Sie keine Ahnung haben, wen Sie an Ihrem Arbeitsplatz bzw. in Ihrer Schule oder Universität um Hilfe bitten sollen, wenden Sie sich an die Person, von der Sie Ihr Benutzerkonto und Ihr Kennwort erhalten haben.

> [!NOTE]
> Administratoren gezielter Versionen haben zuerst Zugriff auf neue Features. Neue Features werden später für alle Administratoren bereitgestellt. Dies bedeutet, dass das Admin Center möglicherweise nicht angezeigt wird oder anders aussieht als in den Hilfeartikeln beschrieben. Um zu den ersten zu gehören, die neue Features sehen, lesen Sie "Am Admin Center teilnehmen" weiter unten.

## <a name="turn-on-targeted-release"></a>Aktivieren der zielgerichteten Version

1. Melden Sie sich bei [admin.microsoft.com](https://admin.microsoft.com)an, wechseln Sie zum Navigationsbereich, und wählen Sie **Einstellungen** \> **Organisationsprofil** aus.

2. Wechseln Sie zur Karte **"Versionseinstellungen",** und wählen Sie dann **"Bearbeiten"** aus.

3. Wählen Sie entweder **"Zielversion für alle"** oder **"Zielversion" für ausgewählte Benutzer** aus. Wenn Sie die benutzerorientierte Version für ausgewählte Benutzer auswählen, stellen Sie sicher, dass Sie Ihr Administratorkonto (und alle anderen Administratoren in Ihrer Organisation, die teilnehmen möchten) zur Liste der ausgewählten Benutzer hinzufügen.

## <a name="admin-center-feedback"></a>Admin Center-Feedback

Im Admin Center können Sie Microsoft Feedback zu Ihrer Erfahrung geben, indem Sie direkt neben der Schaltfläche **"Hilfe benötigen"** am unteren Rand jeder Seite **"Feedback geben"** auswählen. Teilen Sie uns mit, was Ihnen gefällt und was wir besser machen können. Darüber hinaus können Sie von Zeit zu Zeit Popupumfragen erhalten, in denen Sie nach Ihren Gesamtaufrufen oder einer bestimmten, neu veröffentlichten Erfahrung gefragt werden. Sie können am Ende dieses Artikels auch Feedback geben, indem Sie auswählen, **ob diese Informationen hilfreich waren.**

### <a name="which-microsoft-365-plans-are-available-to-trial-or-buy"></a>Welche Microsoft 365 Pläne stehen zur Testversion oder zum Kauf zur Verfügung?

Microsoft 365 ist eine vollständige, intelligente Lösung, die Office 365, Windows 10 und Enterprise Mobility + Security umfasst, mit der jeder kreativ sein und sicher zusammenarbeiten kann. Die folgenden Microsoft 365 Abonnements sind im Admin Center verfügbar, damit Sie jetzt versuchen oder kaufen können:

- Microsoft 365 Business
- Microsoft 365 Enterprise E3
- Microsoft 365 Enterprise E5

Weitere Informationen finden Sie unter ["Testen oder Kaufen eines Microsoft 365 Abonnements".](../commerce/try-or-buy-microsoft-365.md)

### <a name="i-found-a-bug-or-i-want-to-request-a-feature-enhancement-how-do-i-let-microsoft-know"></a>Ich habe einen Fehler gefunden, oder ich möchte eine Funktionserweiterung anfordern. Wie kann ich Microsoft mitteilen?

Wir freuen uns, von Ihnen zu hören! Das Melden von Fehlern und das Teilen von Feedback hilft uns, die Microsoft 365 Admin Center zu verbessern. Um Feedback zu geben, wählen Sie die Schaltfläche **"Feedback"** am unteren Rand der Seite aus, und verwenden Sie das Formular, um uns Ihre Meinung zu senden. Aktivieren Sie das Kontrollkästchen, und bestätigen Sie Ihre E-Mail-Adresse, wenn Sie möchten, dass jemand aus dem Microsoft 365 Admin Center Team Ihre Kommentare nachverfolgung. Wir können nicht versprechen, jedes Feedback weiterzuverwenden, aber wir werden es versuchen!

Sie können auch Feedback von außerhalb des Admin Centers in unserem UserVoice-Forum geben. Sie können diese Seite verwenden, um Featurevorschläge zu machen, über die andere Forumbenutzer abstimmen können: [UserVoice-Forum für das neue Admin Center.](https://go.microsoft.com/fwlink/?linkid=2024994)

### <a name="what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange"></a>Was ist mit den spezifischen Typen von IT-Rollen und anderen Arbeitsbereichen wie Sicherheit, Geräteverwaltung oder Exchange?

Die Microsoft 365 Admin Center ist der allgemeine Einstiegspunkt für alle Teams und Rollen, die Microsoft 365 verwalten. Die Oberfläche, Informationen und Steuerelemente sind für jeden Administrator und jede Rolle zugeschnitten und anpassbar. Darüber hinaus ermöglichen spezielle Arbeitsbereiche eine tiefe, präzise Kontrolle. Zu diesen Speziellen Arbeitsbereichen gehören SharePoint, Teams &amp; Skype, Exchange, Sicherheit, Compliance, Geräteverwaltung und Azure Active Directory. Die spezialisierten Arbeitsbereiche finden Sie im Navigationsbereich im Microsoft 365 Admin Center unter [https://admin.microsoft.com](https://go.microsoft.com/fwlink/?linkid=2024339) .
