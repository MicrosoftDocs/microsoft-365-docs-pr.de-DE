# <a name="set-up-microsoft-365-business-premium"></a>Einrichten von Microsoft 365 Business Premium

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE471FJ?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

1. Melden Sie sich beim Microsoft 365 Admin Center an, und wählen Sie  **Gehe zu Setup** aus. Der Setup-Assistent wird gestartet.
2.  Auf der Seite  **Office-Apps installieren**  können Sie auswählen, ob Sie Office-Apps auf Ihrem Computer installieren möchten, oder wählen Sie  **weiter** aus.
3.  Geben Sie auf der Seite  **Ihre Domäne ist Ihre Geschäftsidentität**  den Domänennamen Ihres Unternehmens ein, und wählen Sie dann  **diese Domäne verwenden** aus. Wenn Sie eine allgemeine Registrierungsstelle wie WordPress oder GoDaddy verwenden, werden Ihre Einträge automatisch auf dieser Seite hinzugefügt.
4. Wählen Sie auf der Seite  **Domäne über**  prüfen die Option  **über** prüfen aus, um den Besitz Ihrer Domäne zu überprüfen, und melden Sie sich dann bei Ihrem Domänen registrierungskonto an. Wenn Sie Microsoft einen TXT-Eintrag für die Kontoüberprüfung hinzufügen lassen möchten, wählen Sie  **bestätigen** aus.
5. Geben Sie auf der Seite  **Benutzer hinzufügen und Lizenzen zuweisen**  Benutzer Vornamen, Nachname und Benutzernamen ein. Wählen Sie dann  **Benutzer hinzufügen und Lizenzen zuweisen** aus.
6. Geben Sie auf der Seite  **Anmeldeinformationen freigeben**  Alternative e-Mail-Adressen für Ihre Benutzer ein, und wählen Sie  **e-Mail senden und Fortfahren** aus.
7.  Klicken Sie auf der Seite  **Datensätze aktivieren**  auf  **weiter**.
8.  Auf der Seite  **Hinzufügen von DNS-Einträgen**  wird empfohlen, die Dienste für e-Mail, Teams und Sicherheit für mobile Geräte aktiviert zu lassen. Wählen Sie  **configure** aus, und wählen Sie dann  **Connect** aus, um die Änderungen mit Ihrer Domänenregistrierungsstelle zu bestätigen.
9.  Lassen Sie auf der Seite **Schutz vor fortgeschrittenen Cyber-Bedrohungen verbessern** **Office Advanced Threat Protection** auf, überprüfen Sie, was gescannt wird, und wählen Sie dann **Erweiterte Bedrohungsschutz**-**Richtlinien** erstellen aus.   
10.  Wenn Sie möchten, können Sie optional Outlook auf Ihren mobilen Geräten installieren. Wählen Sie  **weiter** aus.
11. Wählen Sie auf der Seite "Personen, die  **Microsoft Teams verwenden**  " aus, ob Sie Benutzern in Ihrer Organisation eine e-Mail über Teams senden möchten, und wählen Sie  **weiter** aus.
12. Lassen Sie Mobile App Verwaltung auf der Seite  **Daten in Office für Mobile schützen**  auf, erweitern Sie die Einstellungen, und überprüfen Sie Sie, und wählen Sie dann  **Mobile App Verwaltungsrichtlinie erstellen** aus.
13. Kehren Sie nach Abschluss des Setups zum Microsoft Admin Center zurück. Im Admin Center können Sie auf der Seite  **Setup**  fortfahren, Features wie Windows 10-Richtlinien, DLP usw. einzurichten.

Das Setup ist abgeschlossen.
