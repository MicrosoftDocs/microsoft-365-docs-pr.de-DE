# <a name="upgrade-from-windows-10-home-to-windows-10-pro"></a>Upgrade von Windows 10 Home auf Windows 10 pro

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Microsoft 365 Business erfordert Windows 10 pro, um Ihre Daten zu schützen. Wenn Sie über einen Computer mit Windows 10 Home verfügen, führen Sie die folgenden Schritte aus, um ein Upgrade auf Windows 10 pro auszuführen.

1. Wählen Sie auf dem Desktop das  **Startmenü**  aus, suchen Sie nach  **Microsoft Store** , und wählen Sie es aus.
2. Klicken Sie im Microsoft Store auf die Suchleiste, geben Sie  **Windows 10 pro** ein, und wählen Sie dann  **Upgrade auf Windows 10 pro** aus.
3. Wählen Sie unter  **Upgrade auf Windows 10 pro die** Option Weitere  **Informationen** aus, und wählen Sie dann  **kaufen** aus.
4. Geben Sie in das Feld  **Windows-Sicherheit**  Ihre PIN ein, um Ihre Identität zu überprüfen.
5. Überprüfen Sie die Zahlungsseite, lesen Sie die Verkaufsbedingungen, nehmen Sie Änderungen an Ihrer Zahlungsmethode vor, und wählen Sie dann  **kaufen** aus.
6. Wählen Sie unter  **Upgrade auf Windows 10 pro die** Option  **install** aus.
7. Speichern Sie bei Bedarf alle Dateien, und wählen Sie dann  **Ja, lassen Sie&#39;s gehen**  , um mit der Installation von Windows zu beginnen.
