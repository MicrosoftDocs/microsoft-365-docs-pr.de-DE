# <a name="upgrade-from-windows-10-home-to-windows-10-pro"></a>Upgrade von Windows 10 Home auf Windows 10 Pro

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Microsoft 365 Business erfordert Windows 10 Pro, um Ihre Daten zu schützen. Wenn Sie über einen Computer mit Windows 10 Home verfügen, führen Sie die folgenden Schritte aus, um auf Windows 10 Pro zu aktualisieren.

1. Wählen Sie auf Ihrem Desktop das  **Startmenü**  aus, suchen Sie  **nach Microsoft Store,** und wählen Sie es aus.
2. Klicken Sie im Microsoft Store auf die Suchleiste, geben **Sie Windows 10 Pro** ein, und wählen Sie dann Upgrade auf Windows **10 Pro aus.**
3. Wählen **Sie unter Upgrade auf Windows 10 Pro** weitere Informationen und dann **"Kaufen" aus.** 
4. Geben Sie  **in das Feld "Windows-Sicherheit"**  Ihre PIN ein, um Ihre Identität zu überprüfen.
5. Überprüfen Sie die Zahlungsseite, lesen Sie die Verkaufsbedingungen, nehmen Sie bei Bezahlmethode Änderungen vor, und wählen Sie dann **"Kaufen" aus.**
6. Wählen **Sie unter "Upgrade auf Windows 10 Pro"** die Option **"Installieren" aus.**
7. Speichern Sie bei Bedarf alle Dateien, und wählen Sie dann  **"Ja" aus,&#39;**  windows zu installieren.
