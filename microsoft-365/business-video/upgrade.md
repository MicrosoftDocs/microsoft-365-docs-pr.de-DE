# <a name="upgrade-from-windows-10-home-to-windows-10-pro"></a>Upgrade von Windows 10 Home auf Windows 10 Pro

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j?autoplay=false]

## <a name="try-it"></a>Probieren Sie es aus!

Microsoft 365 unternehmen erfordert Windows 10 Pro, um Ihre Daten zu schützen. Wenn Sie über einen Computer verfügen, auf dem Windows 10 Home, führen Sie die folgenden Schritte aus, um ein Upgrade auf Windows 10 Pro.

1. Wählen Sie auf Ihrem Desktop das **Menü Start** aus, suchen Sie **nach Microsoft Store** , und wählen Sie es aus.
2. Klicken Sie Microsoft Store in der Suchleiste, geben Sie **Windows 10 Pro** ein, und wählen Sie **dann Upgrade to Windows 10 Pro** aus.
3. Wählen **Sie unter Upgrade to Windows 10 Pro** die Option Weitere **Informationen** aus, und wählen Sie dann **Kaufen aus.**
4. Geben Sie **Windows-Sicherheit** PIN ein, um Ihre Identität zu überprüfen.
5. Überprüfen Sie die Zahlungsseite, lesen Sie die Verkaufsbedingungen, nehmen Sie änderungen an Ihrer Zahlungsmethode vor, wenn Sie möchten, und wählen Sie **dann Kaufen aus.**
6. Wählen **Sie unter Upgrade to Windows 10 Pro** die Option Installieren **aus.**
7. Speichern Sie bei Bedarf alle Dateien, und wählen Sie dann Ja aus, lassen **&#39;los,** um mit der Installation der Windows.
