## Set a password to never expire

Run one of the following commands:

- To set the password of one user to never expire, run the following cmdlet by using the UPN or the user ID of the user:

    ```powershell
    Set-AzureADUser -ObjectId <user ID> -PasswordPolicies DisablePasswordExpiration
    ```

- <span data-ttu-id="8287d-123">Führen Sie das folgende Cmdlet aus, um die Kennwörter aller Benutzer in einer Organisation so festzulegen, dass sie nie ablaufen:</span><span class="sxs-lookup"><span data-stu-id="8287d-123">To set the passwords of all the users in an organization to never expire, run the following cmdlet:</span></span>

    ```powershell
    Get-AzureADUser -All $true | Set-AzureADUser -PasswordPolicies DisablePasswordExpiration
    ```

> [!WARNING]
> <span data-ttu-id="8287d-124">Benutzerkonten, die mit dem Parameter konfiguriert `-PasswordPolicies DisablePasswordExpiration` sind, altern basierend auf dem `pwdLastSet` Attribut noch.</span><span class="sxs-lookup"><span data-stu-id="8287d-124">User accounts configured with the `-PasswordPolicies DisablePasswordExpiration` parameter still age based on the `pwdLastSet` attribute.</span></span> <span data-ttu-id="8287d-125">Wenn Sie den Ablauf basierend auf dem Attribut ändern, muss der `pwdLastSet` Benutzer bei allen Kennwörtern mit einem `-PasswordPolicies None` pwdLastSet, die älter als 90 Tage sind, diese bei der nächsten Anmeldung ändern.</span><span class="sxs-lookup"><span data-stu-id="8287d-125">Based on the `pwdLastSet` attribute, if you change the expiration to `-PasswordPolicies None`, all passwords that have a pwdLastSet older than 90 days require the user to change them the next time they sign in.</span></span> <span data-ttu-id="8287d-126">Diese Änderung kann sich auf eine große Anzahl von Benutzern auswirken.</span><span class="sxs-lookup"><span data-stu-id="8287d-126">This change can affect a large number of users.</span></span>
