# <a name="add-a-new-employee-to-microsoft-365"></a>Hinzufügen eines neuen Mitarbeiters zu Microsoft 365

Dieser Artikel unterstützt Sie bei der Onboard-Installation eines neuen Mitarbeiters bei Microsoft 365 for Business. Wir gehen davon aus, dass Sie ein Administrator sind und [Microsoft 365-Setup bereits abgeschlossen](../setup/setup.md)haben, und jetzt haben Sie eine neue Person, die Ihrem Unternehmen Beitritt.
  
Sie sind am richtigen Ort, wenn Ihr neuer Mitarbeiter Microsoft 365 benötigt und Sie einen [Microsoft 365-Plan](https://products.office.com/business/compare-office-365-for-business-plans) verwenden, mit dem Sie Office-Apps wie Word und Excel auf einem Computer installieren können. 
  
 **Sie sind kein Administrator?** [Erfahren Sie mehr über Microsoft 365, um](https://support.office.com/article/office-basics-video-training-396b8d9e-e118-42d0-8a0d-87d1f2f055fb) Geschäfts-und Privatbenutzer bei der Einrichtung zu unterstützen. 
  
 **Keine Office-Apps in Ihrem Plan?** Führen Sie die folgenden Schritte aus, überspringen Sie jedoch die Abschnitte für die Installation von apps. Verwenden Sie stattdessen die [Online-Versionen von Office](https://support.office.com/article/91a4ec74-67fe-4a84-a268-f6bdf3da1804.aspx) . 
  
Hier finden Sie eine kurze Übersicht:

|:-----|:-----|

|[Schritt 2: Geben Sie dem Mitarbeiter die Benutzer-ID und das Kennwort](#step-2-give-the-employee-their-user-id-and-password) <br/> |Wenn Sie ein Konto erstellen, erhalten Sie eine ID und ein Kennwort, die Sie an Ihren Mitarbeiter übergeben können, damit Sie sich anmelden können.  <br/> |
|[Schritt 3: erklären, wo die Anmeldung durchführen soll](#step-3-explain-where-to-sign-in) <br/> |Der Anmelde Speicherort ist[https://www.office.com](https://www.office.com) <br/> |
|[Schritt 4: unterstützen Sie Ihren Mitarbeiter bei der ersten Schritte](#step-4-help-your-employee-get-started) <br/> |Lassen Sie Ihre Mitarbeiter wissen, wie Sie OneDrive oder beliebige Teamwebsites in Ihrer Organisation verwenden.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Schritt 1: Erstellen eines Microsoft 365-Kontos für den Mitarbeiter


Anweisungen finden Sie unter [Hinzufügen von Benutzern einzeln oder in Massen zur Microsoft 365-Administratorhilfe](add-users.md). Wenn Sie Ihren neuen Mitarbeiter einrichten, können Sie angeben, dass Anmeldedetails an das persönliche Konto des Mitarbeiters gesendet werden sollen. Auf diese Weise erhalten Sie eine e-Mail vom Microsoft Online Service-Team, in der Sie erfahren, wie Sie sich bei Microsoft 365 anmelden.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Schritt 2: Geben Sie dem Mitarbeiter die Benutzer-ID und das Kennwort


Wenn Sie Sie nicht an Ihre persönliche e-Mail-Adresse gesendet haben, Drucken Sie den Anmeldenamen und das Kennwort des Mitarbeiters aus, und geben Sie ihn an. Oder Ihnen die Informationen über das Telefon mitteilen.
  
Da Sie noch nicht auf Ihre e-Mails zugreifen können, senden Sie die Informationen nicht an diese e-Mail-Adresse.
  
## <a name="step-3-explain-where-to-sign-in"></a>Schritt 3: erklären, wo die Anmeldung durchführen soll 


Genau wie Facebook, Amazon oder Gmail meldet sich Ihr Mitarbeiter bei der Verwendung von Microsoft 365 an. Geben Sie Ihnen die folgenden Anmeldeinformationen:
  
- Melden Sie sich [https://www.office.com](https://www.office.com)bei an.
    
- Wählen Sie **Anmelden**aus, und geben Sie dann die Benutzer-ID und das Kennwort ein.
    
## <a name="step-4-help-your-employee-get-started"></a>Schritt 4: unterstützen Sie Ihren Mitarbeiter bei der ersten Schritte


Teilen Sie mit Ihnen die [Mitarbeiter-Schnelleinstellung für Microsoft 365](https://support.office.com/article/employee-quick-setup-b9700090-ce64-4046-ab92-ce8488a7bc0f) , um sich anzumelden, Software zu installieren, e-Mail einzurichten und vieles mehr.
