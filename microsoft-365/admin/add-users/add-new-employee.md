# <a name="add-a-new-employee-to-office-365"></a>Hinzufügen eines neuen Mitarbeiters zu Office 365

Dieser Artikel unterstützt Sie beim Onboard eines neuen Mitarbeiters beim Office 365 für Unternehmen. Wir gehen davon aus, dass Sie ein Administrator sind und bereits [Office 365 eingerichtet](../setup/setup.md)haben, und jetzt haben Sie einen neuen Beitritt zu Ihrem Unternehmen.
  
Wenn Ihr neuer Mitarbeiter Office 365 benötigt, sind Sie am richtigen Ort, und Sie verwenden einen [Office 365 Plan](https://products.office.com/business/compare-office-365-for-business-plans) , mit dem Sie Office-Apps wie Word und Excel auf einem Computer installieren können. 
  
 **Sie sind kein Administrator?** [Hier erfahren](https://support.office.com/article/9b7306d3-8d61-4794-bb6f-6520f65956d9.aspx) Sie, wie Sie Office 365 Geschäfts-und Privatbenutzern bei der Einrichtung von Office 365 helfen können. 
  
 **Keine Office-Apps in Ihrem Plan?** Führen Sie die folgenden Schritte aus, überspringen Sie jedoch die Abschnitte für die Installation von apps. Verwenden Sie stattdessen die [Online-Versionen von Office](https://support.office.com/article/91a4ec74-67fe-4a84-a268-f6bdf3da1804.aspx) . 
  
Hier finden Sie eine kurze Übersicht:

|[Schritt 2: Geben Sie dem Mitarbeiter die Benutzer-ID und das Kennwort](#step-2-give-the-employee-their-user-id-and-password) <br/> |Wenn Sie ein Konto erstellen, erhalten Sie eine ID und ein Kennwort, die Sie an Ihren Mitarbeiter übergeben können, damit Sie sich anmelden können.  <br/> |

## <a name="step-1-create-an-office-365-account-for-the-employee"></a>Schritt 1: Erstellen eines Office 365 Kontos für den Mitarbeiter


Anweisungen hierzu finden Sie unter [Hinzufügen von Benutzern einzeln oder in loser Schüttung zur Office 365-Admin-Hilfe](add-users.md). Wenn Sie Ihren neuen Mitarbeiter einrichten, können Sie angeben, dass Anmeldedetails an das persönliche Konto des Mitarbeiters gesendet werden sollen. Auf diese Weise erhalten Sie eine e-Mail vom Microsoft Online Service-Team, in der Sie erfahren, wie Sie sich bei Office 365 anmelden.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Schritt 2: Geben Sie dem Mitarbeiter die Benutzer-ID und das Kennwort


Wenn Sie Sie nicht an Ihre persönliche e-Mail-Adresse gesendet haben, Drucken Sie den Office 365 Anmeldenamen und das Kennwort des Mitarbeiters aus, und geben Sie ihn an. Oder Ihnen die Informationen über das Telefon mitteilen.
  
Da Sie noch nicht auf Ihre Office 365 e-Mails zugreifen können, senden Sie die Informationen nicht an diese e-Mail-Adresse.
  
## <a name="step-3-explain-where-to-sign-in"></a>Schritt 3: erklären, wo die Anmeldung durchführen soll 


Genau wie Facebook, Amazon oder Gmail meldet sich Ihr Mitarbeiter bei der Verwendung von Office 365 an. Geben Sie Ihnen die folgenden Anmeldeinformationen:
  
- Melden Sie sich [https://www.office.com](https://www.office.com)bei an.
    
- Wählen Sie **Anmelden**aus, und geben Sie dann die Benutzer-ID und das Kennwort ein.
    
## <a name="step-4-help-your-employee-get-started"></a>Schritt 4: unterstützen Sie Ihren Mitarbeiter bei der ersten Schritte


Teilen Sie mit Ihnen die [Mitarbeiter-Schnelleinstellung für Office 365](https://support.office.com/article/69cd80a8-56b8-436f-aa1f-2d2a3cc51060) , um sich anzumelden, Software zu installieren, e-Mail einzurichten und vieles mehr.
