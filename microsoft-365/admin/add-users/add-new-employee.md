# <a name="add-a-new-employee-to-microsoft-365"></a>Hinzufügen eines neuen Mitarbeiters zu Microsoft 365

Dieser Artikel hilft Ihnen beim Onboarding eines neuen Mitarbeiters für Microsoft 365 Business. Wir gehen davon aus, dass Sie ein Administrator sind und bereits Microsoft 365 [eingerichtet](../setup/setup.md)haben, und jetzt haben Sie einen neuen Mitarbeiter in Ihrem Unternehmen.
  
Sie sind am richtigen Ort, wenn Ihr neuer Mitarbeiter Microsoft 365 benötigt, [](https://products.office.com/business/compare-office-365-for-business-plans) und Sie verwenden einen Microsoft 365-Plan, mit dem Sie Office Apps wie Word und Excel auf einem Computer installieren können. 
  
 **Sie sind kein Administrator?** [Erfahren Sie mehr über Microsoft 365](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) hilft Geschäfts- und Privatbenutzern bei der Einrichtung. 
  
 **Keine Office in Ihrem Plan?** Führen Sie die folgenden Schritte aus, überspringen Sie jedoch die Abschnitte für die Installation von Apps. Verwenden Sie [stattdessen die Onlineversionen Office.](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) 
  
Hier finden Sie eine kurze Übersicht: 
  
|**Schritt**|**Warum sollte dies der Grund sein?**|
|:-----|:-----|
|[Schritt 1: Erstellen eines Microsoft 365 für den Mitarbeiter](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Jedes Mal, wenn ein neuer Mitarbeiter ihrem Unternehmen beit tritt, erstellen Sie ein Konto für sie, damit sie mit der Verwendung von Microsoft 365.  <br/> |
|[Schritt 2: Geben Sie dem Mitarbeiter seine Benutzer-ID und das Kennwort](#step-2-give-the-employee-their-user-id-and-password) <br/> |Wenn Sie ein Konto erstellen, erhalten Sie eine ID und ein Kennwort, die Sie an Ihren Mitarbeiter übergeben können, damit er sich anmelden kann.  <br/> |
|[Schritt 3: Erklären, wo sich anmelden soll](#step-3-explain-where-to-sign-in) <br/> |Der Anmeldespeicherort ist [https://www.office.com](https://www.office.com) <br/> |
|[Schritt 4: Helfen Sie Ihren Mitarbeitern bei den ersten Schritte](#step-4-help-your-employee-get-started) <br/> |Teilen Sie Ihren Mitarbeitern mit, wie OneDrive oder teamwebsites in Ihrer Organisation verwendet werden.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Schritt 1: Erstellen eines Microsoft 365 für den Mitarbeiter


Anweisungen finden Sie unter [Hinzufügen von Benutzern und gleichzeitiges](add-users.md)Zuweisen von Lizenzen. Wenn Sie Ihren neuen Mitarbeiter einrichten, können Sie die Anmeldedetails an das persönliche Konto des Mitarbeiters senden. Auf diese Weise erhalten sie eine E-Mail vom Microsoft Online Service Team, in der ihnen erklärt wird, wie sie sich bei Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Schritt 2: Geben Sie dem Mitarbeiter seine Benutzer-ID und das Kennwort


Wenn Sie es nicht an ihre persönliche E-Mail-Adresse gesendet haben, drucken Sie den Anmeldenamen und das Kennwort des Mitarbeiters aus, und geben Sie es an sie weiter. Oder teilen Sie ihnen die Informationen über das Telefon mit.
  
Da sie noch keinen Zugriff auf ihre E-Mails haben, senden Sie die Informationen nicht an diese E-Mail-Adresse.
  
## <a name="step-3-explain-where-to-sign-in"></a>Schritt 3: Erklären, wo sich anmelden soll 


Genau wie Facebook, Amazon oder Gmail meldet sich Ihr Mitarbeiter an, um Microsoft 365. Geben Sie ihnen die folgenden Anmeldeinformationen:
  
- Melden Sie sich unter [https://www.office.com](https://www.office.com) an.
    
- Wählen **Sie Anmelden** aus, und geben Sie die Benutzer-ID und das Kennwort ein.
    
## <a name="step-4-help-your-employee-get-started"></a>Schritt 4: Helfen Sie Ihren Mitarbeitern bei den ersten Schritte


Teilen Sie ihnen die [Mitarbeiter-Schnelleinrichtung](../../business-video/employee-quick-setup.md) für Microsoft 365, um sich zu registrieren, Software zu installieren, E-Mails einrichten und vieles mehr.
