# <a name="add-a-new-employee-to-microsoft-365"></a>Hinzufügen eines neuen Mitarbeiters zu Microsoft 365

Dieser Artikel hilft Ihnen beim Onboarding eines neuen Mitarbeiters in Microsoft 365 Business. Wir gehen davon aus, dass Sie Administrator sind und die Einrichtung von [Microsoft 365](../setup/setup.md)bereits abgeschlossen haben, und jetzt haben Sie einen neuen Mitarbeiter, der Ihrem Unternehmen beitritt.
  
Sie sind am richtigen Ort, wenn Ihr neuer Mitarbeiter Microsoft 365 benötigt und Sie einen [Microsoft 365-Plan](https://products.office.com/business/compare-office-365-for-business-plans) verwenden, mit dem Sie Office-Apps wie Word und Excel auf einem Computer installieren können. 
  
 **Sie sind kein Administrator?** [Erfahren Sie mehr über Microsoft 365,](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) um Geschäfts- und Privatbenutzer bei der Einrichtung zu helfen. 
  
 **Keine Office-Apps in Ihrem Plan?** Führen Sie die folgenden Schritte aus, überspringen Sie jedoch die Abschnitte für die Installation von Apps. Verwenden Sie [stattdessen die Onlineversionen von Office.](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) 
  
Hier ist eine kurze Übersicht: 
  
|**Schritt**|**Warum sollte dies der Grund sein?**|

|[Schritt 1: Erstellen eines Microsoft 365-Kontos für den Mitarbeiter](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Jedes Mal, wenn ein neuer Mitarbeiter Ihrem Unternehmen beitritt, erstellen Sie ein Konto für den Mitarbeiter, damit er mit der Verwendung von Microsoft 365 beginnen kann.  <br/> |
|[Schritt 2: Geben Sie dem Mitarbeiter seine Benutzer-ID und sein Kennwort.](#step-2-give-the-employee-their-user-id-and-password) <br/> |Wenn Sie ein Konto erstellen, erhalten Sie eine ID und ein Kennwort, die Sie an Ihren Mitarbeiter übergeben können, damit er sich anmelden kann.  <br/> |
|[Schritt 3: Erläutern, wo sie sich anmelden müssen](#step-3-explain-where-to-sign-in) <br/> |Der Anmeldespeicherort ist [https://www.office.com](https://www.office.com) <br/> |
|[Schritt 4: Helfen Sie Ihren Mitarbeitern bei den ersten Schritte](#step-4-help-your-employee-get-started) <br/> |Teilen Sie Ihren Mitarbeitern mit, wie Sie OneDrive oder beliebige Teamwebsites in Ihrer Organisation verwenden.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Schritt 1: Erstellen eines Microsoft 365-Kontos für den Mitarbeiter


Anweisungen finden Sie unter [Hinzufügen von Benutzern und gleichzeitiges](add-users.md)Zuweisen von Lizenzen. Wenn Sie Ihren neuen Mitarbeiter einrichten, können Sie festlegen, dass Anmeldedetails an das persönliche Konto des Mitarbeiters gesendet werden. Auf diese Weise erhalten sie eine E-Mail vom Microsoft Online Service-Team, in der ihnen erklärt wird, wie sie sich bei Microsoft 365 anmelden.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Schritt 2: Geben Sie dem Mitarbeiter seine Benutzer-ID und sein Kennwort.


Wenn Sie es nicht an ihre persönliche E-Mail-Adresse gesendet haben, drucken Sie den Anmeldenamen und das Kennwort des Mitarbeiters aus, und geben Sie es an ihn weiter. Oder teilen Sie ihnen die Informationen über das Telefon mit.
  
Da sie noch keinen Zugriff auf ihre E-Mails haben, senden Sie die Informationen nicht an diese E-Mail-Adresse.
  
## <a name="step-3-explain-where-to-sign-in"></a>Schritt 3: Erläutern, wo sie sich anmelden müssen 


Genau wie Facebook, Amazon oder Gmail meldet sich Ihr Mitarbeiter für die Verwendung von Microsoft 365 an. Geben Sie ihnen die folgenden Anmeldeinformationen:
  
- Melden Sie sich bei [https://www.office.com](https://www.office.com) .
    
- Wählen **Sie "Anmelden"** aus, und geben Sie dann die Benutzer-ID und das Kennwort ein.
    
## <a name="step-4-help-your-employee-get-started"></a>Schritt 4: Helfen Sie Ihren Mitarbeitern bei den ersten Schritte


Teilen Sie für sie das schnelle Setup für [Mitarbeiter für Microsoft 365,](https://support.microsoft.com/office/b9700090-ce64-4046-ab92-ce8488a7bc0f) um sich anmelden, Software installieren, E-Mails einrichten und vieles mehr.
