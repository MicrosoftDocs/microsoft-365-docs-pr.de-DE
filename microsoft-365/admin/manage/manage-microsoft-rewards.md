# <a name="manage-microsoft-rewards"></a>Verwalten von Microsoft Rewards

Benutzer, die über persönliche Microsoft-Konten verfügen, können [Microsoft Rewards-Punkte](https://www.microsoft.com/rewards) im Zusammenhang mit ihren Arbeitssuchen sammeln, wenn das *Feature "Link Microsoft Azure Active Directory (Azure AD) mit Rewards"* aktiviert ist.

Wenn der Benutzer dieses Feature nicht abwählt, wird sein persönliches Microsoft-Konto mit dem Azure AD-Konto am Arbeitsplatz verknüpft. Microsoft Rewards empfängt und protokolliert Informationen, dass eine Suche stattgefunden hat, erhält jedoch keine Informationen über den Inhalt der Suche. Der Inhalt von Abfragen von Ihrer Organisation wird im Rahmen dieses Features nicht für Microsoft Rewards freigegeben.

Für Administratoren von Bildungseinrichtungen mit minderjährigen Kinder ist ein microsoft-Elternkonto erforderlich, damit das Kind an Microsoft Rewards teilnehmen kann. Das Elternkonto wird nicht dem Organisationskonto des Schülers zugeordnet. Weitere Informationen zu Microsoft-Konten für Kinder finden Sie unter ["Zustimmung der Eltern" und unter "Microsoft-Kinderkonten".](https://support.microsoft.com/account-billing/c6951746-8ee5-8461-0809-fbd755cd902e)

Dieses Feature ist für Behördenbenutzer nicht verfügbar. Administratoren sollten sicherstellen, dass die Compliancerichtlinien ihrer Organisation die Verwendung von persönlichen Microsoft Rewards-Konten bei Arbeitssuchen zulassen.
