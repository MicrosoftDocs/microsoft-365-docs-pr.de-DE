# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Hier erhalten Sie die neueste Version Microsoft 365 Verwendungsanalyse

Die Vorlagen-App kann mehrmals pro Jahr mit neuen Daten oder neuen Visualisierungen aktualisiert werden. Ihre vorhandene Instanz funktioniert weiterhin, aber wenn Sie die neueste Version erhalten möchten, muss eine neue Instanz erstellt werden, und alle Anpassungen müssen auf die neue Instanz angewendet werden. Weitere [Informationen finden Sie unter Aktivieren Microsoft 365 Verwendungsanalyse](enable-usage-analytics.md).
