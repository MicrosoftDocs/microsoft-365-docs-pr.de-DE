# <a name="download-perpetual-software-and-product-license-keys"></a>Herunterladen unbefristeter Software- und Produktlizenzschlüssel

In diesem Artikel wird erläutert, wie Sie Software- und Produktlizenzschlüssel für unbefristete Software herunterladen, die über das #A0 (CSP Cloud Solution Provider erworben wurde.

## <a name="before-you-begin"></a>Bevor Sie beginnen

Sie müssen ein globaler Administrator sein, um die Schritte in diesem Artikel ausführen zu können. Weitere Informationen finden Sie unter [Informationen zu Administratorrollen](../add-users/about-admin-roles.md).

## <a name="download-software-and-product-license-keys"></a>Herunterladen von Software- und Produktlizenzschlüsseln

1. Wechseln Sie im Microsoft 365 Admin Center zu **Abrechnung** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Ihre Produkte</a>.
2. Wählen Sie **auf der** Registerkarte Produkte im Abschnitt **Software** die Software aus, die Sie herunterladen möchten.
3. Wählen Sie auf der Seite Abonnementdetails im Abschnitt **Downloads & Schlüssel** die Option **Produktversion,** Sprache **und** CPU **& Dateityp** aus, und wählen Sie **dann Herunterladen aus.**
4. Um den Schlüssel herunterzuladen, wählen Sie **Schlüssel in Zwischenablage kopieren aus.**
5. Wählen Sie im rechten Bereich **Kopieren** aus, und schließen Sie dann den Bereich.
6. Fügen Sie den Schlüssel an einem sicheren Speicherort in eine Datei ein, und geben Sie ihn dann wie bei der Softwareinstallation angewiesen ein. Der Schlüssel ist erforderlich, um die heruntergeladene Software zu aktivieren.
