# <a name="add-your-company-branding-to-office-365-sign-in-page"></a>Hinzufügen Ihres Unternehmensbrandings zur Anmeldeseite von Office 365

 [] Sie können nun mit dem Azure Active Directory-Abonnement, das Bestandteil Ihres Office 365-Abonnements ist, die Anmeldeseite anpassen, die Ihre Benutzer sehen. 
  
## <a name="add-company-branding-to-your-sign-in-page-and-access-panel-pages"></a>Hinzufügen von Unternehmensbranding zu Ihrer Anmeldeseite und zu Ihrer Zugriffsbereichsseite

Wenn Sie über ein kostenpflichtiges Abonnement für Office 365, Microsoft Dynamics CRM Online, Enterprise Mobility Suite oder andere Microsoft-Dienste verfügen, haben Sie ein kostenloses Abonnement für Azure Active Directory. Mithilfe von Azure Active Directory können Sie Benutzer- und Gruppenkonten erstellen und verwalten sowie Unternehmensbranding zu Ihren Seiten hinzufügen. Zum Aktivieren dieses Abonnements und Zugreifen auf das Microsoft Azure-Verwaltungsportal müssen Sie einmalig einen Registrierungsprozess durchlaufen. Anschließend können Sie von dem jeweils unterstützten Microsoft-Dienst auf Azure Active Directory zugreifen. Anweisungen dazu, wie Sie Ihr Office 365-Abonnement registrieren, finden Sie unter [Registrieren Ihres kostenlosen Azure Active Directory-Abonnements](https://go.microsoft.com/fwlink/p/?LinkID=527966). Allgemeine Anweisungen zur Verwaltung finden Sie unter [Verwalten des Verzeichnisses für Ihr Office 365-Abonnement in Azure](https://go.microsoft.com/fwlink/p/?LinkId=620076). 
  
Die folgende Abbildung zeigt, welche Teile der Anmeldeseite in Azure geändert werden können.
  
![Bereiche der Office 365 Anmeldeseite, die Sie anpassen können.](../../media/screenshotbranding.png)
  
1. Die große Abbildung und/oder deren Hintergrundfarbe
    
2. Das Bannerlogo
    
3. In diesem Bereich können Sie auch Text hinzufügen.
    
Zusätzlich zur Anmeldeseite können Sie die Zugriffsbereichsseite in Azure anpassen.
  
Wenn Sie Branding hinzufügen möchten, sollten Sie die Anpassungsoptionen in der Azure-Inhaltsgruppe ausprobieren: [Hinzufügen Ihres Unternehmensbranding zur Anmelde- und Zugriffsbereichsseite](https://go.microsoft.com/fwlink/p/?LinkId=620077)
