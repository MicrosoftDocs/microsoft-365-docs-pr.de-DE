# <a name="add-your-company-branding-to-the-sign-in-page"></a>Hinzufügen Ihres Firmenbranding zur Anmeldeseite

 Sie können jetzt das in Ihrem Microsoft 365-Abonnement enthaltene Azure Active Directory (AD)-Abonnement verwenden, um die Anmeldeseite, die Ihre Benutzer sehen, anzupassen. 
  
## <a name="add-company-branding-to-your-sign-in-page-and-access-panel-pages"></a>Hinzufügen von Unternehmensbranding zu Ihrer Anmeldeseite und zu Ihrer Zugriffsbereichsseite

Wenn Sie über ein kostenpflichtiges Abonnement für Microsoft 365 for Business, Microsoft Dynamics CRM Online, Enterprise Mobility Suite oder andere Microsoft-Dienste verfügen, haben Sie ein kostenloses Abonnement für Azure Active Directory. Sie können Azure Active Directory zum Erstellen und Verwalten von Benutzer-und Gruppenkonten sowie zum Hinzufügen von Firmenbranding zu Ihren Seiten verwenden. Um dieses Abonnement zu aktivieren und auf das Microsoft Azure Verwaltungs Portal zuzugreifen, müssen Sie einen einmaligen Registrierungsvorgang abschließen. Anschließend können Sie von Ihrem Microsoft-Dienst, der ihn verwendet, auf Azure Active Directory zugreifen. Anweisungen zum Registrieren Ihres Microsoft 365-Abonnements finden Sie unter [Registrieren Ihres kostenlosen Azure Active Directory-Abonnements](https://go.microsoft.com/fwlink/p/?LinkID=527966)und unter [Verwalten des Verzeichnisses für Ihr Microsoft 365-Abonnement in Azure](https://go.microsoft.com/fwlink/p/?LinkId=620076) für allgemeine Verwaltungsanweisungen. 
  
Die folgende Abbildung zeigt, welche Teile der Anmeldeseite in Azure geändert werden können.
  
![Bereiche der Anmeldeseite, die Sie anpassen können.](../../media/screenshotbranding.png)
  
1. Die große Abbildung und/oder deren Hintergrundfarbe
    
2. Das Bannerlogo
    
3. In diesem Bereich können Sie auch Text hinzufügen.
    
Zusätzlich zur Anmeldeseite können Sie die Zugriffsbereichsseite in Azure anpassen.
  
Wenn Sie Branding hinzufügen möchten, sollten Sie die Anpassungsoptionen in der Azure-Inhaltsgruppe ausprobieren: [Hinzufügen Ihres Unternehmensbranding zur Anmelde- und Zugriffsbereichsseite](https://go.microsoft.com/fwlink/p/?LinkId=620077)
