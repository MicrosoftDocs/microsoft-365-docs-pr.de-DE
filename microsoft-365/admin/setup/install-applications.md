# <a name="install-office-applications"></a>Installieren von Office-Anwendungen

Nachdem Sie nun eine Microsoft 365 eingerichtet haben, können Sie einzelne Office auf Ihrem Mac, PC oder mobilen Geräten installieren.
  
Befolgen Sie diese Links, um Informationen zu den folgenden Informationen zu erhalten:
  
- Installieren Office Anwendungen: [Installieren Office auf Ihrem PC oder Mac](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658)

- Installieren anderer Apps: [Project](https://support.microsoft.com/office/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281), [Visio](https://support.microsoft.com/office/install-visio-f98f21e3-aa02-4827-9167-ddab5b025710)oder [Skype for Business](https://support.microsoft.com/office/install-skype-for-business-8a0d4da8-9d58-44f9-9759-5c8f340cb3fb)

- Einrichten mobiler Geräte: [Microsoft 365 mobile Einrichtung – Hilfe](https://support.microsoft.com/office/7dabb6cb-0046-40b6-81fe-767e0b1f014f)

- Einrichten von E-Mails in Outlook: [Windows](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b) oder [Mac](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b#PickTab=Outlook_for_Mac)
 
- [Aktualisieren Sie Benutzer auf die neuesten Apps](upgrade-users-to-latest-office-client.md)
