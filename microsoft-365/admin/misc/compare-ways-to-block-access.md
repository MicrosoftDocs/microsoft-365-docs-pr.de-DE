# <a name="compare-ways-to-block-access"></a>Vergleichen von Möglichkeiten zum Blockieren des Zugriffs

Wenn ein Mitarbeiter Ihre Organisation zu guten oder schlechten Bedingungen verlässt, müssen Sie den Zugriff auf Microsoft 365. Hier einige Möglichkeiten, wie Sie vorgehen können.
  
||||
|:-----|:-----|:-----|
|**Möglichkeiten zum Blockieren des Zugriffs** <br/> |**Definition** <br/> |**Bewährte Methode** <br/> |
|Anmeldung blockieren  <br/> |Eine Möglichkeit, einen Benutzer am Zugriff auf Microsoft 365 zu blockieren, besteht in der Änderung des Anmeldestatus in **"Anmelden blockiert".** Dadurch wird verhindert, dass sich Microsoft 365 von ihren Computern und mobilen Geräten anmelden, obwohl sie zuvor heruntergeladene oder synchronisierte E-Mails und Dokumente weiterhin anzeigen können. Wenn Sie Blackberry Enterprise Service verwenden, können Sie auch ihren Zugriff dort deaktivieren.  <br/> |Verwenden Sie diese Option, wenn ein Mitarbeiter plant, die Organisation zu verlassen oder langfristig abwesend sein wird.  <br/> |
|Benutzerkennwort zurücksetzen  <br/> |Eine weitere Möglichkeit, einen Benutzer am Zugriff auf Microsoft 365 zu hindern, ist das Zurücksetzen des Kennworts. Damit wird verhindert, dass das Konto genutzt wird, der Benutzer kann jedoch weiterhin heruntergeladene oder synchronisierte E-Mails und Dokumente anzeigen. Sie können sich unter dem Namen des Benutzers anmelden und das Kennwort in ein von Ihnen gewähltes ändern.  <br/> |Verwenden Sie diese Option, wenn ein Mitarbeiter die Organisation unvermittelt und dauerhaft verlässt und Sie sich Sorgen um die Geschäftsdaten machen.  <br/> |
|Alle zugewiesenen Lizenzen entfernen  <br/> |Eine weitere Option ist das Entfernen Microsoft 365 dem Benutzer zugewiesenen Lizenzen. Damit wird verhindert, dass der Benutzer Anwendungen und Dienste wie die Office-Suite, Office-Apps für das Web, Yammer und SharePoint Online verwendet. Der Benutzer kann sich zwar weiterhin anmelden, kann jedoch nicht mehr auf diese Dienste zugreifen.  <br/> |Verwenden Sie, wenn Sie der Meinung sind, dass dieser Benutzer keinen Zugriff mehr auf bestimmte Features in Microsoft 365.  <br/> <br> **Wichtig:** Wenn Sie eine Lizenz entfernen, wird das Postfach des Benutzers nach 30 Tagen gelöscht.
