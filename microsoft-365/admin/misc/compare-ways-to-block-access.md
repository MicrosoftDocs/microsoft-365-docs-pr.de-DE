# <a name="compare-ways-to-block-access"></a>Vergleichen der Möglichkeiten zum Blockieren des Zugriffs

Wenn ein Mitarbeiter Ihre Organisation verlässt, müssen Sie Ihren Zugriff auf Microsoft 365 in gutem Grund oder in schlechtem Rahmen blockieren. Im folgenden finden Sie einige Möglichkeiten, wie Sie dies tun können.
  
||||
|:-----|:-----|:-----|
|**Möglichkeiten zum Blockieren des Zugriffs** <br/> |**Definition** <br/> |**Bewährte Methode** <br/> |
|Anmeldung blockieren  <br/> |Eine Möglichkeit zum Blockieren des Zugriffs auf Microsoft 365 durch einen Benutzer besteht darin, den Anmeldestatus in " **Anmeldung blockiert**" zu ändern. Dadurch wird verhindert, dass Sie sich von ihren Computern und mobilen Geräten bei Microsoft 365 anmelden, obwohl Sie bereits heruntergeladene oder synchronisierte e-Mails und Dokumente anzeigen können. Wenn Sie BlackBerry Enterprise Service verwenden, können Sie auch dort den Zugriff deaktivieren.<br/> |Verwenden Sie diese Option, wenn ein Mitarbeiter plant, die Organisation zu verlassen oder langfristig abwesend sein wird.  <br/> |
|Benutzerkennwort zurücksetzen  <br/> |Eine andere Möglichkeit, um zu verhindern, dass ein Benutzer auf Microsoft 365 zugreift, besteht darin, sein Kennwort zurückzusetzen. Dadurch wird verhindert, dass Sie Ihr Konto verwenden, obwohl Sie noch zuvor heruntergeladene oder synchronisierte e-Mails und Dokumente anzeigen können. Sie können sich dann als diese anmelden und das Kennwort in eins Ihrer Wahl ändern.  <br/> |Verwenden Sie diese Option, wenn ein Mitarbeiter die Organisation unvermittelt und dauerhaft verlässt und Sie sich Sorgen um die Geschäftsdaten machen.  <br/> |
|Alle zugewiesenen Lizenzen entfernen  <br/> |Eine weitere Option besteht darin, alle Microsoft 365-Lizenzen zu entfernen, die dem Benutzer zugewiesen sind. Damit wird verhindert, dass der Benutzer Anwendungen und Dienste wie die Office-Suite, Office-Apps für das Web, Yammer und SharePoint Online verwendet. Der Benutzer kann sich zwar weiterhin anmelden, kann jedoch nicht mehr auf diese Dienste zugreifen.  <br/> |Verwenden Sie diese Funktion, wenn Sie der Meinung sind, dass dieser Benutzer nicht mehr auf bestimmte Features in Microsoft 365 zugreifen muss.  <br/> <br> **Wichtig:** Wenn Sie eine Lizenz entfernen, wird das Postfach des Benutzers nach 30 Tagen gelöscht.
