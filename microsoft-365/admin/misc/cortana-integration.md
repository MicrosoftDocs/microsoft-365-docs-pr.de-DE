# <a name="cortana-in-microsoft-365"></a>Cortana in Microsoft 365

Cortana, Ihr persönlicher Produktivitäts Assistent, bietet AI-basierte Erfahrungen, um Zeit zu sparen und die Aufmerksamkeit auf die wichtigsten Aspekte zu lenken. Cortana hilft Ihren Benutzern, Ihre persönliche Produktivität für den gesamten Arbeits-und Lebensalltag zu steigern. Wenn Sie mit gültigen Arbeits-oder Schul Konten angemeldet sind, können Benutzer in den Microsoft 365-Produktivitätstools Cloud-basierte Cortana-Unterstützungsdienste erhalten. Cortana in Microsoft 365 erfüllt Datenschutz-, Sicherheits-und Konformitäts Zusagen von Microsoft 365 auf Unternehmensebene.

Als persönlicher Produktivitäts Assistent wurde Cortana entwickelt, um Funktionen bereitzustellen, die sicher und sicher mit Microsoft 365-Daten wie e-Mails, Dateien, Chats usw. verarbeitet werden, um Zeit zu sparen, die Effizienz zu erhöhen und die Produktivität Ihrer Benutzer zu verbessern.

Wir konzentrieren uns Cortana auf die Unternehmensproduktivität.

- In Übereinstimmung mit anderen Microsoft 365-Diensten erfüllt Cortana in Microsoft 365 die gleichen Zusagen auf Unternehmensebene wie Datenschutz, Sicherheit und Compliance, die in den [Online Services Terms (Ost)](https://www.microsoft.com/en-us/licensing/product-licensing/products)widergespiegelt werden.

- Neue Cloud-basierte Cortana-Unterstützungsdienste, wie beispielsweise die e-Mail-Nachricht und das Abspielen von "meine e-Mails", werden mit Cortana in Microsoft 365 aktiviert und erfüllen diese Zusagen vollständig. Diese Features sind derzeit im Englisch (USA)-Markt für Benutzer mit Postfächern im US-Geo und für Mitarbeiter von multinationalen Unternehmen mit US-Postfächern verfügbar.

- Benutzer können eine Verbindung zu Cortana in Microsoft 365-Diensten herstellen, die hier über Clientanwendungen wie Outlook Mobile beschrieben werden, vorbehaltlich gesonderter Lizenzierungsbedingungen.

- Vorhandene Consumer-Benutzeroberflächen, einschließlich Cortana in Windows 10 (Version 1909 und früher) und der Cortana-App auf IOS und Android, unterliegen dem [Microsoft-Dienstvertrag](https://www.microsoft.com/en-us/licensing/product-licensing/products) und der [Microsoft-Datenschutzerklärung](https://privacy.microsoft.com/en-us/privacystatement). Diese Begriffe Regeln auch Cortana in Microsoft 365-Diensten, die dem Benutzer zur Verfügung gestellt werden, wenn er mit seinen Consumer-Anmeldeinformationen angemeldet ist.

## <a name="what-data-is-processed-by-cortana-in-microsoft-365"></a>Welche Daten werden von Cortana in Microsoft 365 verarbeitet?

Cortana in Microsoft 365 verarbeitet Abfragen des Benutzers, Office-Daten, die zur Erfüllung der Anforderung des Benutzers benötigt werden, und andere von Microsoft-Systemen generierte Telemetrie, um den Dienst auszuführen. Die Daten, die von Cortana in Microsoft 365 gesammelt werden, enthalten die Textdarstellung der gesprochenen Abfragen des Benutzers (Transkriptionen von der Spracherkennung). Diese Textdaten sind Kundendaten und werden gemäß den [Online Dienstleistungsbedingungen](https://www.microsoft.com/en-us/licensing/product-licensing/products)verwaltet. Es wird nur zum entwickeln und verbessern von maschinellen Lernmodellen verwendet, die mit den [Online Dienstbedingungen](https://www.microsoft.com/en-us/licensing/product-licensing/products)konsistent sind.

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-in-microsoft-365"></a>Was ist das Steuerungsmodell für Kundendaten in Cortana in Microsoft 365?

Im Einklang mit anderen Microsoft 365-und Microsoft 365-Diensten ist Cortana in Microsoft 365 gesichert und unterliegt den [Online-Dienstbedingungen](https://www.microsoft.com/en-us/licensing/product-licensing/products). Dies umfasst eine Reihe von Zusagen zum Schutz von Kundendaten gegen versehentlichen Verlust, Änderung, unberechtigte Offenlegung oder Zugriff oder unrechtmäßige Vernichtung. Kundendaten unterliegen ebenfalls strengen Zugriffsbeschränkungen. Microsoft verwendet Kundendaten nur für die Bereitstellung der vereinbarten Dienste und für Zwecke, die mit diesen Diensten kompatibel sind. Ausführliche Informationen finden Sie in der folgenden Tabelle.

## <a name="cortana-in-microsoft-365-features"></a>Cortana in Microsoft 365-Features

### <a name="play-my-emails"></a>Meine e-Mails abspielen

"Meine e-Mail-Nachrichten abspielen" (wie mit Outlook Mobile verbunden) ist eine sprachgesteuerte, Freisprechfunktion für Benutzer, die Ihren Posteingang über die Lautsprecher Ihres Telefons, Ihres Kopfhörers oder eines angeschlossenen Audiogeräts anhören möchten. Benutzer können Cortana bitten, ihre letzten e-Mail-Nachrichten vorzulesen und bitten Cortana, Aktionen wie kennzeichnen, archivieren, löschen und Überspringen von Nachrichten durchführen zu lassen. Dieses Feature ist besonders hilfreich, um Ihre e-Mails beim Pendeln oder unterwegs aufzuholen. Wenn der Benutzer in "meine e-Mails abspielen" mit Cortana kommuniziert, wird die sprach-audioanforderung direkt an Cortana in Microsoft 365 gesendet. Eine Text-zu-Sprache-Anzeige der e-Mail-Adresse des Benutzers wird innerhalb der Microsoft 365-Cloud verarbeitet. Während dieses Vorgangs werden keine Microsoft 365-Daten auf dem mobilen Gerät des Benutzers verarbeitet, und es werden keine neuen e-Mail-Daten gespeichert. Eine Abschrift der gesprochenen Befehle (d. h. "als gelesen markieren", "weiter", "kennzeichnen" usw.) kann in Übereinstimmung mit den Datenschutzbedingungen in den [Online-Dienstbedingungen](https://www.microsoft.com/en-us/licensing/product-licensing/products)aufbewahrt werden.

Cortana Ruft ab, wenn eine e-Mail geschützt ist, und kurze Pause vor dem Lesen der Nachricht, damit die Benutzer genügend Zeit haben, die Wiedergabe anzuhalten oder zur nächsten Nachricht zu springen. Ähnlich wie bei einem privaten Telefonanruf sollten Benutzer Vorsicht walten lassen, wenn Sie die Wiedergabe an Orten initiieren, an denen vertrauliche Informationen möglicherweise zu hören sind. In diesen Fällen wird empfohlen, dass die Mitarbeiter Ihrer Organisation bei der Verwendung von "meine e-Mails in Outlook Mobile abspielen" in entsprechenden Umgebungen Kopfhörer tragen.

### <a name="opt-out-of-play-my-emails"></a>Deaktivieren von "meine e-Mails wiedergeben"

Einzelpersonen können mit den folgenden Schritten das Abspielen von "meine e-Mails" deaktivieren.

1. Öffnen Sie Outlook Mobile.

2. Wechseln Sie zu **Einstellungen**.
  
3. Wählen Sie **meine e-Mails abspielen**aus.

4. Bewegen Sie die Umschaltfläche für die Konten, die Sie deaktivieren möchten, auf aus.

### <a name="briefing-email"></a>Briefing e-Mail

Cortana sendet eine personalisierte Briefing-e-Mail mit Aufgaben und Zusagen, die Sie mit einer bequemen Möglichkeit getroffen haben, um Sie als **erledigt** zu kennzeichnen, oder planen Sie die Fokus Zeit, um Sie zu erledigen. Es enthält auch eine Zusammenfassung Ihrer Besprechungen und relevanten Dokumente für den Tag. Cortana extrahiert Informationen aus Microsoft 365-Nachrichten eines Benutzers und speichert Sie in Ihrem Microsoft 365-Postfach, bis Sie in Briefing-e-Mails konsolidiert wurde. Auf personenbezogene Daten kann zu keinem Zeitpunkt außerhalb Ihres Microsoft 365-Postfachs zugegriffen werden.

### <a name="how-to-opt-out-of-briefing-email"></a>Deaktivieren von Briefing-e-Mails

Administratoren können das Briefing für Ihre Organisation mithilfe einer [UMSCHALT](https://docs.microsoft.com/briefing/be-admin) Fläche im Microsoft 365 Admin Center konfigurieren. Einzelne Personen können sich von Cortana-Briefing-e-Mails abmelden, indem Sie in der Fußzeile der Nachricht auf Abmelden klicken.

Wir werden weiterhin weitere Cortana in Microsoft 365-Funktionen einführen, um die Produktivität Ihrer Organisation zu steigern.

Erfahren Sie mehr über [Microsoft Compliance-Angebote](https://docs.microsoft.com/microsoft-365/compliance/offering-home).
