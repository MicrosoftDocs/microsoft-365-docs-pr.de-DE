# <a name="icann-verification-of-contact-information"></a>Überprüfung der ICANN von Kontaktinformationen

Wenn Sie eine Domäne erwerben oder Änderungen an den Kontaktinformationen für eine Domäne vornehmen, muss die Registrierungsstelle Ihre Kontaktinformationen überprüfen, um die ICANN-Bestimmungen einzuhalten.

Zur Überprüfung wird eine e-Mail an die Adresse des Registranten gesendet.

 **Um Ihre Kontaktinformationen zu überprüfen, wählen Sie den Link in der Bestätigungs-e-Mail aus.** Wenn die Domänenbesitzer die e-Mail-Adresse nicht überprüfen, wird die Domäne angehalten. Wechseln Sie dann zur [Seite Domänen](https://admin.microsoft.com/adminportal/home?ref=Domains) im Admin Center, und überprüfen Sie Ihren Domänenstatus.
