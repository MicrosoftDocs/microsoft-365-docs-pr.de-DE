# <a name="icann-verification-of-contact-information"></a>ICANN-Überprüfung von Kontaktinformationen

Wenn Sie eine Domäne erwerben oder Änderungen an den Kontaktinformationen für eine Domäne vornehmen, muss die Registrierungsstelle Ihre Kontaktinformationen überprüfen, um die ICANN-Bestimmungen einzuhalten.

Zur Überprüfung wird eine E-Mail an die Adresse des Registranten gesendet.

 **Um Ihre Kontaktinformationen zu überprüfen, wählen Sie den Link in der Überprüfungs-E-Mail aus.** Wenn die Domänenbesitzer die E-Mail-Adresse nicht überprüfen, wird die Domäne angehalten. Wechseln Sie dann zur Seite [Domänen im](https://admin.microsoft.com/adminportal/home?ref=Domains) Admin Center, und überprüfen Sie Ihren Domänenstatus.
