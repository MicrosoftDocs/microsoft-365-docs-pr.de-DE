# <a name="quick-help-deleted-user"></a>Schnelle Hilfe: Gelöschter Benutzer

[] Es kann vorkommen, dass Sie einen [Benutzer](../add-users/add-users.md) in Ihrer Organisation löschen möchten. In der Regel löschen Sie einen Benutzer, wenn die Person die Organisation verlässt. 
  
## <a name="delete-a-user"></a>Löschen eines Benutzers
  
::: moniker range="o365-worldwide"

1. Wechseln Sie im Admin Center zu der Seite **Benutzer** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Aktive Benutzer</a>.

::: moniker-end

::: moniker range="o365-germany"

 1. Wechseln Sie im Admin Center zu der Seite **Benutzer** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">Aktive Benutzer</a>.

::: moniker-end

::: moniker range="o365-21vianet"

 1. Wechseln Sie im Admin Center zu der Seite **Benutzer** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850628" target="_blank">Aktive Benutzer</a>.

::: moniker-end

2. Wählen Sie einen Benutzer aus, und wählen Sie dann über der Liste der Benutzer die drei Punkte (weitere Aktionen) aus, und wählen Sie dann **Benutzer löschen aus.**
  
3. Wählen Sie im Löschbestätigungsbereich Benutzer **löschen aus.** <br/>[Lesen Sie den vollständigen Artikel.](../add-users/delete-a-user.md)

  
## <a name="restore-a-user"></a>Wiederherstellen eines Benutzers

Sie können Benutzer wiederherstellen, indem Sie auf der Startseite nach gelöschten Benutzern  suchen und **in** den Ergebnissen gelöschte Benutzer auswählen oder über die linke Navigation zur Seite Gelöschte Benutzer navigieren und den Benutzer auswählen, den Sie wiederherstellen möchten.  [Den vollständigen Artikel lesen](../add-users/delete-a-user.md)
  
## <a name="what-are-deleted-users"></a>Was sind gelöschte Benutzer?

Ein gelöschter [Benutzer](../add-users/add-users.md) ist ein Geschäfts- oder Schulkonto, das Sie aus Ihrer Organisation löschen. 
  
## <a name="does-deleting-a-user-free-up-their-microsoft-365-license"></a>Gibt das Löschen eines Benutzers seine Microsoft 365 frei?

Ja, durch das Löschen des Benutzers wird die diesem Microsoft 365 zugewiesene Lizenz frei. Diese Lizenz kann dann einem anderen Benutzer zugewiesen werden. [Den vollständigen Artikel lesen](../../commerce/licenses/buy-licenses.md)
  
## <a name="i-had-a-deleted-user-listed-in-deleted-users-and-then-it-disappeared-what-happened"></a>In meiner Liste der gelöschten Benutzer wurde ein gelöschter Benutzer aufgeführt, der nun nicht mehr angezeigt wird. Was ist geschehen?

Der gelöschte Benutzer wird in Ihrer Organisation 30 Tage lange aufbewahrt. Nach Ablauf dieser 30 Tage wird der Benutzer endgültig aus Ihrer Organisation entfernt.
  
## <a name="how-long-do-i-have-if-i-want-to-restore-a-user"></a>Wie lange kann ich einen Benutzer wiederherstellen?

Einen gelöschten Benutzer können Sie innerhalb von 30 Tagen nach dem Löschen wiederherstellen.
  
## <a name="do-i-lose-all-the-users-data-when-i-delete-them"></a>Gehen die Daten eines Benutzers verloren, wenn ich ihn lösche?

Wenn Sie einen Benutzer löschen, gehen dessen Daten nicht verloren. Sie werden 30 Tage lang gespeichert. Wenn Sie den Benutzer innerhalb von 30 Tagen wiederherstellen, werden auch alle zugehörigen Daten wiederhergestellt.
  
## <a name="how-do-i-restore-a-user"></a>Wie kann ich einen Benutzer wiederherstellen?

Sie können Benutzer wiederherstellen, indem Sie auf der Startseite nach gelöschten Benutzern  suchen und **in** den Ergebnissen gelöschte Benutzer auswählen oder über die linke Navigation zur Seite Gelöschte Benutzer navigieren und den Benutzer auswählen, den Sie wiederherstellen möchten.  [Den vollständigen Artikel lesen](../add-users/delete-a-user.md)
