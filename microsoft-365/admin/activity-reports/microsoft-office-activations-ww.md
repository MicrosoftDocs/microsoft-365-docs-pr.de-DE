# <a name="microsoft-365-reports-in-the-admin-center---microsoft-office-activations"></a>Microsoft 365 Berichte im Admin Center – Microsoft Office Aktivierungen

Das Microsoft 365 **-Dashboard** zeigt Ihnen die Übersicht über die Aktivitäten in den Produkten in Ihrer Organisation. Sie können Drilldowns zu Einzelberichten auf Produktebene ausführen und auf diese Weise genauere Einblicke in die Aktivitäten innerhalb der einzelnen Produkte erhalten. Sehen Sie sich die [Übersicht über Berichte](activity-reports.md) an.
  
Im Bericht zu den Office-Aktivierungen wird angezeigt, welche Benutzer ihr Office-Abonnement auf mindestens einem Gerät aktiviert haben. Es enthält eine Aufschlüsselung der Microsoft 365 Apps for Enterprise, Project und Visio Pro für Office 365 Abonnementaktivierungen sowie die Aufschlüsselung der Aktivierungen auf desktop- und geräteübergreifend. Dieser Bericht kann hilfreich sein, wenn Sie Benutzer identifizieren möchten, die möglicherweise zusätzliche Hilfe und Unterstützung beim Aktivieren ihres Office-Abonnements benötigen.
  
> [!NOTE]
> Sie müssen ein globaler Administrator, globaler Leser oder Berichtleser in Microsoft 365 oder ein Exchange-, SharePoint-, Teams-Dienst-, Teams-Kommunikations- oder Skype for Business-Administrator sein, um Berichte anzeigen zu können.  
  
## <a name="how-to-get-to-the-office-activations-report"></a>Aufrufen des Berichts zu den Office-Aktivierungen

1. Wechseln Sie im Admin Center zur Seite **Berichte** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">Verwendung</a>. 
2. Klicken Sie auf der Startseite des Dashboards auf **die** Schaltfläche Weitere Anzeigen auf der Office Aktivierungskarte.
  
## <a name="interpret-the-office-activations-report"></a>Auslegen des Berichts zu den Office-Aktivierungen
  
Sie können die Aktivierungen im Office 365 anzeigen, indem Sie die Registerkarte **Aktivierungen** auswählen.<br/>![Microsoft 365 Berichte – Microsoft Office 365 Aktivierung.](../../media/e1df82a2-3336-4b38-b66c-b286c44b82ee.png)

Wählen **Sie Spalten auswählen** aus, um Spalten aus dem Bericht hinzuzufügen oder zu entfernen.  <br/> ![Office 365 Aktivierungsbericht – Spalten auswählen](../../media/d11a0efa-a067-4440-a4f3-71b618a90301.png)

Sie können die Berichtsdaten auch im CSV-Format in eine Excel-Datei exportieren, indem Sie den Link **Exportieren** auswählen. Dadurch werden Daten aller Benutzer exportiert, und Sie können einfache Sortier- und Filtervorgänge zur weiteren Analyse ausführen. Bei weniger als 2.000 Benutzern können Sie innerhalb der Tabelle im Bericht selbst sortieren und filtern. Bei mehr als 2.000 Benutzern müssen Sie die Daten zum Filtern und Sortieren exportieren. 

|Element|Beschreibung|
|:-----|:-----|
|**Metrik**|**Definition**|
|Benutzername  <br/> |Die E-Mail-Adresse des Benutzers.  <br/> |
|Anzeigename  <br/> |Der vollständige Name des Benutzers.  <br/> |
|Produktlizenzen  <br/> |Die Produkte, die diesem Benutzer zugewiesen sind.  <br/> |
|Datum der letzten Aktivierung (UTC)  <br/> |Das Datum, an dem der Office auf einem Desktop oder Gerät aktiviert wurde.  <br/> |
|Aktivierung auf Windows Computern  <br/> |Die Anzahl Windows Desktops, die ein Benutzer aktiviert Office.  <br/> |
|Aktivierung auf Mac-Computern <br/> |Die Anzahl der mac-Desktops, die ein Benutzer aktiviert hat, Office aktiviert wurde.|
|Aktivierung auf Windows 10 Und Tablets  <br/> |Die Anzahl der Windows 10 mobilen Geräte, auf die ein Benutzer Office.  <br/> |
|Aktivierung auf iOS-Telefonen und Tablets  <br/> |Die Anzahl der iOS-Geräte, die ein Benutzer aktiviert Office.|
|Aktivierung auf Android-Smartphones und Tablets  <br/> |Die Anzahl der von einem Benutzer aktivierten Office.  <br/> |
|Aktivierung gemeinsam genutzter Computer |Dies ist der Fall, wenn der Benutzer Office aktivierung des freigegebenen Computers verwendet hat.|
