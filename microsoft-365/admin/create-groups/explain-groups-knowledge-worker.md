# <a name="explaining-microsoft-365-groups-to-your-users"></a>Erläutern Microsoft 365 Gruppen für Ihre Benutzer

Microsoft 365 Mit Gruppen können Sie eine Sammlung von Ressourcen einrichten, die freigegeben werden können, darunter ein freigegebenes Postfach und ein freigegebener Kalender, eine SharePoint-Website mit einem OneNote-Notizbuch und ein Microsoft Planner unter anderem. Microsoft Teams können auch einbezogen werden, wenn Sie eine Gruppe erstellen, oder sie kann später hinzugefügt werden. Ressourcen für Berechtigungsgruppen werden über die Gruppe verwaltet.

Gruppen können durch Erstellen einer beliebigen der freigegebenen Ressourcen erstellt werden. Das Erstellen einer Gruppe in Outlook ergibt dasselbe Ergebnis wie das Erstellen einer mit einer Gruppe verbundenen SharePoint Teamwebsite oder eines Plans in Planner. Wenn Ihre Benutzer neu in Microsoft 365 sind, wird ihnen dies möglicherweise nicht bewusst. Dies kann zu Verwirrung für Ihre Benutzer und zur Möglichkeit der Erstellung doppelter Ressourcen führen. (Bei Beispielen kann jemand eine SharePoint-Website für die Dokumentzusammenarbeit erstellen und später eine separate Instanz von Planner erstellen, ohne dass Planner bereits als Teil der Gruppe verfügbar war.)

Da Gruppen auf verschiedene Weise erstellt werden können, wird empfohlen, Ihre Benutzer für die Verwendung der Methode zu trainieren, die für Ihre Organisation am besten geeignet ist:

- Wenn Ihre Organisation den Größten Teil ihrer Kommunikation über E-Mail vor sich hat, müssen Sie Ihre Benutzer anweisen, Gruppen in Outlook.
- Wenn Ihre Organisation SharePoint oder eine Migration von SharePoint lokal verwendet, müssen Sie Ihre Benutzer anweisen, SharePoint Teamwebsites für die Zusammenarbeit zu erstellen.
- Wenn Ihre Organisation eine Teams bereitgestellt hat, müssen Sie Die Benutzer anweisen, ein Team zu erstellen, wenn sie einen Raum für die Zusammenarbeit benötigen.

[![image desc ](../../media/03.png)](../../media/03.png#lightbox)

Wenn Sie Ihre Benutzer so schulen, dass sie immer die Methode zum Erstellen von Gruppen verwenden, die am besten auf ihre Arbeitsweise ausgerichtet ist, wenn sie einen Raum für die Zusammenarbeit mit anderen benötigen, können Sie Verwirrung und Doppelte von Ressourcen vermeiden. Je erfahrener benutzer werden, desto besser verstehen sie die Sammlung von Diensten, die mit einer Gruppe zusammenkommen, und dass unterschiedliche Erstellungsmethoden zu demselben Ergebnis führen.

Sie können die Vorlage [Microsoft 365 Gruppen](https://www.microsoft.com/download/details.aspx?id=102396) für PowerPoint als Ausgangspunkt für Schulungspräsentationen für Ihre Benutzer verwenden.
