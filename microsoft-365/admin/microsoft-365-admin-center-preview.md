# <a name="about-the-microsoft-365-admin-center"></a>Informationen zum Microsoft 365 Admin Center

::: moniker range="o365-worldwide"

Das Microsoft 365 Admin Center wurde für IT-Teams als vereinfachte Methode zum Verwalten Ihrer Microsoft 365-Dienste entwickelt. Das Admin Center bietet eine maßgeschneiderte Erfahrung basierend auf den individuellen Anforderungen Ihrer Rolle oder Organisation, verbessert die Effizienz für alltägliche Aufgaben und bietet umsetzbare Einblicke, mit deren Hilfe Sie datengesteuerte Entscheidungen treffen können, um Ihren Benutzern eine bessere Benutzererfahrung zu bieten.

Das Microsoft 365 Admin Center ist der häufige Einstiegspunkt für alle Microsoft 365-Administratoren und kann unter zugegriffen [https://admin.microsoft.com](https://go.microsoft.com/fwlink/?linkid=2024339) werden. Spezielle Arbeitsbereiche, z. B. Sicherheit oder Geräteverwaltung, ermöglichen eine genauere Steuerung. Weitere Informationen zur Zusammenarbeit der Admin Center finden Sie unter [What about the specific types of IT roles and other workspaces like Security, Device Management, or Exchange?](#what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange) weiter unten in diesem Artikel.

::: moniker-end

### <a name="turn-on-targeted-release"></a>Aktivieren der gezielten Veröffentlichung

1. Melden Sie sich [unter admin.microsoft.com](https://admin.microsoft.com)an, wechseln Sie zum Navigationsbereich, und wählen Sie **Einstellungen** \> **Organisationsprofil aus.**

2. Wechseln Sie zur **Freigabeeinstellungskarte,** und wählen Sie dann **Bearbeiten aus.** 
    
3. Wählen Sie **entweder Zielversion für alle benutzer oder** gezielte Version für ausgewählte Benutzer **aus.** Wenn Sie gezielte Version für ausgewählte Benutzer auswählen, stellen Sie sicher, dass Sie Ihr Administratorkonto (und alle anderen Administratoren in Ihrer Organisation, die teilnehmen möchten) zur Liste der ausgewählten Benutzer hinzufügen.
    
### <a name="admin-center-feedback"></a>Admin Center-Feedback

Während Sie im Admin Center sind, können Sie  Microsoft Feedback zu Ihrer Erfahrung geben, indem Sie Feedback direkt neben der Schaltfläche Hilfe **benötigen?** unten auf jeder Seite auswählen. Teilen Sie uns mit, was Ihnen gefällt und was wir besser tun könnten. Darüber hinaus erhalten Sie möglicherweise Popupumfragen von Zeit zu Zeit, in der Sie nach Ihren Gesamtmeinungen oder einer bestimmten Erfahrung fragen, die neu veröffentlicht wurde. Sie können auch feedback am Ende dieses Artikels geben, indem Sie **War diese Informationen hilfreich?**

### <a name="switch-to-the-new-admin-center-and-back-again"></a>Wechseln zum neuen Admin Center und wieder zurück

Während sich das neue Admin Center in der Vorschau befindet, können Sie zwischen dem neuen Admin Center und dem alten Admin Center hin- und herwechseln, indem Sie die Umschalte oben auf der Admin Center-Startseite verwenden. Das neue Admin Center ist voll funktionsfähig und verfügt über alle Funktionen des alten Admin Centers.

Um das neue Admin Center auszuprobieren, aktivieren Sie die Umschalte auf <b>Vorschau testen.</b> Der Umschalter zeigt dann Vorschau <b>auf an.</b> Wenn Sie sich das nächste Mal beim Admin Center anmelden, merken wir uns Ihre Auswahl und melden Sie sich beim neuen Admin Center an.

Um zurück zur Verwendung des alten Admin Centers zu wechseln, schalten Sie die Umschalte von <b>Vorschau</b> an die Aus-Position ein. Wenn Sie sich das nächste Mal beim Admin Center anmelden, melden wir uns beim alten Admin Center an.

Wir können nicht versprechen, jedem Feedback nach folgen, aber wir werden es versuchen! 
  
Sie können auch Feedback von außerhalb des Admin Centers in unserem UserVoice-Forum geben. Sie können diese Seite verwenden, um Featurevorschläge zu machen, über die andere Forumbenutzer abgestimmt werden können: [UserVoice-Forum für das neue Admin Center](https://go.microsoft.com/fwlink/?linkid=2024994).

## <a name="what-about-the-specific-types-of-it-roles-and-other-workspaces-like-security-device-management-or-exchange"></a>Was ist mit den spezifischen Typen von IT-Rollen und anderen Arbeitsbereichen wie Sicherheit, Geräteverwaltung oder Exchange?

Das Microsoft 365 Admin Center ist der häufige Einstiegspunkt für alle Teams und Rollen, die Microsoft 365 verwalten. Die Benutzererfahrung, Informationen und Steuerelemente sind für jeden Administrator und jede Rolle angepasst und anpassbar. Darüber hinaus ermöglichen spezielle Arbeitsbereiche eine tiefe, differenzierte Steuerung. Zu diesen spezialisierten Arbeitsbereichen gehören SharePoint, Teams &amp; Skype, Exchange, Sicherheit, Compliance, Geräteverwaltung und Azure Active Directory. Die spezialisierten Arbeitsbereiche finden Sie im Navigationsbereich im Microsoft 365 Admin Center unter [https://admin.microsoft.com](https://go.microsoft.com/fwlink/?linkid=2024339) .
  
Das Aktualisieren aller Admin Center, um eine einheitliche Erfahrung zu haben, dauert eine Weile, aber Sie können mehr über die Admin Center erfahren, die wir bereits mit der Umarbeitung begonnen haben:
