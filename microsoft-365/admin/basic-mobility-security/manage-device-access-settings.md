# <a name="manage-device-access-settings-in-basic-mobility-and-security"></a>Verwalten von Gerätezugriffseinstellungen in Basic Mobility and Security

Wenn Sie Basic Mobility and Security verwenden, gibt es möglicherweise Geräte, die Sie mit Basic Mobility and Security nicht verwalten können. Wenn ja, sollten Sie Exchange ActiveSync Zugriff auf Microsoft 365-E-Mails für mobile Geräte blockieren, die von Basic Mobility and Security nicht unterstützt werden. Dies trägt dazu bei, Ihre Unternehmensinformationen auf mehr Geräten zu schützen.

Verwenden Sie die folgenden Schritte:

1. Melden Sie sich mit Ihrem globalen Administratorkonto bei Microsoft 365 an.

2. Geben Sie in Ihrem Browser:  [https://protection.office.com](https://protection.office.com/) ein.    

    >[!IMPORTANT]
    >Wenn Sie Basic Mobility and Security für Microsoft 365 Business Standard zum ersten Mal verwenden, aktivieren Sie es hier: [Aktivieren der Grundlegenden Sicherheit und Mobilität](https://admin.microsoft.com/EAdmin/Device/IntuneInventory.aspx). Nachdem Sie es aktiviert haben, verwalten Sie Ihre Geräte mit [Office 365 Security & Compliance](https://protection.office.com/).

3. Wechseln Sie zu "Verhinderung von Datenverlust> **Geräterichtlinien** für die   >  **** Geräteverwaltung, und wählen **Sie "Organisationsweite Gerätezugriffseinstellungen verwalten" aus.**

4. Select **Block**.

    :::image type="content" source="../../media/basic-mobility-security/bms-5-block-access.png" alt-text="Kontrollkästchen für den Zugriff auf grundlegende Mobilität und Sicherheit":::

5. Wählen Sie **"Speichern"** aus.

Informationen zu den von Basic Mobility and Security unterstützten Geräten finden Sie unter ["Funktionen der grundlegenden Mobilität und Sicherheit".](capabilities.md)
