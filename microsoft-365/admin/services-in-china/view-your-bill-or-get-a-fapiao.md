# <a name="view-your-bill-or-get-a-fapiao-in-office-365-operated-by-21vianet"></a>Anzeigen Ihrer Rechnung oder Erhalten eines Fapiao in Office 365 betrieben von 21Vianet

Sie können Ihre Abrechnung oder Rechnung für Informationen zu Ihren Office 365 betrieben von 21Vianet-Abonnementgebühren anzeigen. Sie können auch einen Fapiao (Ihren endgültigen Kaufnachweis) erhalten. Sie können Ihre Fapiao-Anforderung an unser [Fapiao-Verwaltungssystem](https://go.microsoft.com/fwlink/p/?linkid=837465) etwa drei Tage nach der Bezahlten übermitteln. 
  
## <a name="find-your-billing-statement-or-request-a-fapiao"></a>Suchen Sie nach Ihrer Abrechnung oder fordern Sie einen Fapiao an.

1. Wechseln Sie [im Admin Center](https://go.microsoft.com/fwlink/p/?linkid=850627)zur Seite  \> **Abrechnungsrechnungen & Zahlungen.**

2. Suchen Sie die Rechnung, die Sie möchten, wählen Sie zum Anzeigen aus, oder wählen Sie aus, um die PDF herunterzuladen.

  - Wenn Sie einen Fapiao anfordern, wählen **Sie Fapiaos get aus.**

    > [!NOTE]
    > Sie benötigen ein registriertes Konto, um auf das [fapiao-Verwaltungssystem zu zugreifen.](https://go.microsoft.com/fwlink/p/?linkid=837465) Wenn Sie noch kein Konto eingerichtet haben, lesen Sie [Apply for a Fapiao for Office 365 operated by 21Vianet](apply-for-a-fapiao.md).
