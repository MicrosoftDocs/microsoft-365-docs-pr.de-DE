# <a name="about-registration-numbers-and-under-review-notifications"></a>Informationen zu Registrierungsnummern und Benachrichtigungen, die überprüft werden

Dieser Artikel gilt nur für kommerzielle Kunden, die Produkte oder Dienste direkt von Microsoft kaufen oder aktivieren. Dieser Artikel gilt nicht für Volumenlizenzierung oder Cloudlösungsanbieter (Cloud Solution Provider, CSP), die direkt mit einem Partner zusammenarbeiten.

## <a name="what-is-a-registration-number"></a>Was ist eine Registrierungsnummer?  

Wir verwenden die Registrierungsnummer, um die Details Ihres Kontos zu überprüfen. Dadurch können wir ermitteln, ob Microsoft Ihnen Produkte und Dienste bereitstellen kann. Weitere Informationen dazu, welche Werte in dieses Feld eingegeben werden sollen, finden Sie unten im Abschnitt ["Registrierungsnummern nach Land".](#registration-numbers-by-country)

In Ländern, in denen die Registrierungsnummer obligatorisch ist, gibt die Bezeichnung über dem Textfeld an, welche Art von Nummer erforderlich ist.

Im folgenden Screenshot zeigt die Bezeichnung beispielsweise an, dass eine CNPJ-Registrierungsnummer (Brasilien) erforderlich ist.

:::image type="content" source="../media/macregnum-cnpj-screenshot-400.png" alt-text="Screenshot des Registrierungsnummernfelds für die C N P J-Nummer.":::

Für Länder, in denen die Registrierungsnummer optional ist, können Sie eine gesetzliche Registrierungsnummer des Unternehmens angeben. Geben Sie in diesem Feld keine persönliche ID ein.

Der folgende Screenshot zeigt ein Beispiel dafür, wann die Registrierungsnummer optional ist.

:::image type="content" source="../media/macregnum-reg-screenshot_400.png" alt-text="Screenshot des optionalen Registrierungsnummernfelds.":::

Wenn Sie keine gültige Registrierungsnummer haben, finden Sie weitere Informationen unter [Registrierungsnummern nach Land.](#registration-numbers-by-country)

## <a name="what-should-i-do-if-i-get-an-under-review-notification"></a>Was sollte ich tun, wenn ich eine Benachrichtigung über die Unterprüfung erhalte?  

Wenn Sie einen Kauf abschließen, erhalten Sie möglicherweise eine Benachrichtigung, dass Ihr Konto überprüft wird. Während des Überprüfungsprozesses können Sie den Status überprüfen, indem Sie zur Seite   >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">"Abrechnungskonten"</a> navigieren und das Konto auswählen, das Sie zum Abschließen Des Kaufs verwendet haben.

Der Überprüfungsprozess dauert normalerweise etwa einen Tag, kann aber länger dauern.

Der folgende Screenshot zeigt die Benachrichtigung zur Überprüfung, die beim Auschecken angezeigt wird.

:::image type="content" source="../media/macregnum_statuscheckout_screenshot_400.png" alt-text="Screenshot des Kontoüberprüfungshinweiss auf der Seite &quot;Auschecken&quot;.":::

Eine E-Mail-Benachrichtigung wird auch an alle globalen und Abrechnungsadministratoren in Ihrem Konto gesendet. In einigen Fällen wird die Benachrichtigung an Benutzer gesendet, die über die Rolle "Abrechnungskontobesitzer" oder "Abrechnungskontomitwirkende" für das Konto verfügen. Die Benachrichtigung besagt, dass derzeit eine Überprüfung ausgeführt wird. Nach Abschluss des Überprüfungsprozesses wird eine Bestätigungs-E-Mail-Benachrichtigung gesendet.

Der folgende Screenshot zeigt die Benachrichtigung, die auf der Seite mit den Details des Abrechnungskontos angezeigt wird.

:::image type="content" source="../media/macregnum-statusbillacct-screenshot_700.png" alt-text="Screenshot der Benachrichtigung zur Überprüfung auf der Seite &quot;Abrechnungskonten&quot;.":::
