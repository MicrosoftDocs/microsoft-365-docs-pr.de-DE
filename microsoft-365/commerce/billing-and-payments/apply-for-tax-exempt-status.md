# <a name="apply-for-tax-exempt-status-for-office-365-for-business"></a>Beantragen einer Steuerbefreiung für Office 365 Business

Wenn Sie sich für einen steuerfreien Status in Ihrem Land oder Ihrer Region qualifizieren, wenden Sie sich an den [Support](../../admin/contact-support-for-business-products.md) , um eine Serviceanfrage zum Einrichten eines Steuerbefreiungsstatus für Ihr Abonnement zu erstellen. Wenn Sie Ihre Anforderung übermitteln, müssen Sie die folgende Dokumentation bereitstellen: 
  
|**Standort**|**Belege**|
|:-----|:-----|
|USA und Kanada  <br/> |Sales Use Tax Certificate  <br/> |
|Puerto Rico  <br/> |Certificado de compras Exentas  <br/> |
|Irland  <br/> |13B/56A Tax Exemption Certificate  <br/> |
|Internationale Organisationen, die von der Mehrwertsteuer freigestellt werden können  <br/> |Zertifizierung/Bestätigungsschreiben der zuständigen örtlichen Finanzbehörden  <br/> |
