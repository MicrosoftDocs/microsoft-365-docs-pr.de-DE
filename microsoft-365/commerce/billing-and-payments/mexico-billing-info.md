# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Abrechnungsinformationen für Microsoft 365 Business in Mexiko

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. Wo kann ich eine Rechnung für meinen Microsoft 365 Business-Kauf erhalten?

Ihre Rechnung wird 24 Stunden nach Bestätigung Ihres Kaufs an die E-Mail-Adresse der registrierten Abrechnungsbenachrichtigung gesendet. Um Ihre Rechnung im Microsoft 365 Admin Center herunterzuladen, wechseln Sie zur Seite  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">Abrechnungsrechnungen & Zahlungen.</a>

> [!IMPORTANT]
> Wenn die Rechnung Ihre RFC enthalten soll, müssen Sie sie vor der Kaufbestätigung hinzufügen.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. Wie kann ich meine RFC (Registro Federal de Contribuyentes) zur Rechnung hinzufügen, die ich für den Kauf von Microsoft 365 Business erhalten habe?

### <a name="during-the-purchase-process-checkout"></a>Während des Kaufvorgangs (Checkout)

Während des Kaufvorgangs beim Auschecken, wenn Sie zu Schritt **4 kommen. Zahlung und Abrechnung**, wählen Sie das Feld aus, um eine RFC ein eingeben. Dieser Schritt ist erforderlich, damit sie in Ihrer Einkaufsrechnung widerspiegelt wird.

### <a name="for-your-future-purchases"></a>Für Ihre zukünftigen Einkäufe

Sie können Ihre RFC hinzufügen oder ändern, sodass die Informationen in zukünftigen Microsoft 365 For Business-Käufen enthalten sind. Um Ihre RFC hinzuzufügen oder zu ändern, wechseln Sie zur Seite  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">Abrechnungskonten.</a>

> [!NOTE]
> Wenn Sie sich in einem monatlichen oder jährlich wiederkehrenden Abrechnungsabonnement befindet, wird das Hinzu- oder Ändern der RFC auf der Rechnung Ihrer nächsten Gebühr widergeschlagen.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. Kann ich meine RFC einer bereits generierten Rechnung hinzufügen oder ändern?

Es ist wichtig, dass Sie Ihre RFC hinzufügen oder ändern, bevor Sie Ihren Kauf tätigen. Nachdem wir Ihre Microsoft 365 for Business-Rechnung generiert haben, können Sie die RFC-Informationen nicht hinzufügen oder ändern.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. Warum bekommt ich keinen CFDI (Comprobante Fiscal Digital por Internet)?

Die Microsoft Corporation, die in den Vereinigten Staaten von Amerika ansässig ist, ist nur in der Lage, Rechnungen nach den Gesetzen der Gerichtsbarkeit aus zu stellen und ist nicht verpflichtet, eine elektronische CFDI-Rechnung zur Verfügung zu stellen.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. Ist die Rechnung, die ich erhalte, für meine Buchhaltungsvorgänge in Mexiko gültig?

Eine von fremden Entitäten (in diesem Fall Microsoft Corporation) generierte Rechnung, die die entsprechende RFC enthält, ist für Steuerzwecke in Mexiko gültig. (Es wird empfohlen, ihren Buchhalter für weitere Informationen zu informieren).
