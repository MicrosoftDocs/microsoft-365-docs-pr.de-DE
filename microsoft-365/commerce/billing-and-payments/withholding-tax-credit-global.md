# <a name="request-a-credit-for-withholding-tax-on-your-account-global-customers"></a>Anfordern einer Gutschrift für die Quellensteuer für Ihr Konto (globale Kunden)

> [!NOTE]
>
> Wenn Ihre Organisation ihren Sitz in Indien hat, finden Sie weitere Informationen unter Anfordern einer Gutschrift für Die Quellensteuer auf Ihr [Konto (Indien-Kunden).](withholding-tax-credit-india.md)

Einige Kunden erhalten Web Direct-Rechnungen (Azure und Microsoft 365), die von einer Microsoft-Entität in einem fremden Land in Rechnung gestellt werden. Wenn Ihre Organisation grenzüberschreitende Zahlungen an diese Entität vor nimmt, kann die Steuerbehörde in Ihrem Land verlangen, dass Sie einen Teil der grenzüberschreitenden Zahlung als Quellensteuer (Withholding Tax, WHT) zurückbehalten. Wenn Sie Steuern einbehalten haben, wie von Ihrer Steuerbehörde beim Übermitleiden von Zahlungen an Microsoft erforderlich, wird in diesem Artikel das Verfahren zum Beanspruchen einer Gutschrift für die einbehaltene Steuer erläutert.

## <a name="for-invoice-pay-customers-who-pay-by-check-or-wire"></a>Für Abrechnungskunden, die per Scheck oder Überweisung bezahlen

Wenn Sie steuern bei der Auszahlung einbehalten und die einbehaltene Steuer bei der zuständigen Steuerbehörde hinterlegt haben, müssen Sie eine WHT-Anforderung einreichen, um den ausstehenden Saldo auf Ihrem Konto zu löschen.

Ihre WHT-Anforderung muss die folgenden Elemente enthalten:

- Eine abgeschlossene Kopie des [Quellensteuergutschriftsformulars](https://download.microsoft.com/download/a/a/f/aaf8306b-79d4-455b-975f-41ce9e67b9cb/wht%20credit%20form%20-%20global.docx) (ausgefüllt vom Kunden)
- Eine signierte oder gescannte Kopie des Quellensteuerzertifikats oder -belegs

Übermitteln Sie die WHT-Anforderung, indem Sie ein Ticket mit dem Microsoft-Support öffnen.

## <a name="for-customers-who-pay-by-credit-card"></a>Für Kunden, die per Kreditkarte bezahlen

Wenn Es sich bei Ihrer Zahlungsmethode um eine Kreditkarte handelt und Sie eine vollständige Zahlung an Microsoft vorgenommen haben und whT auch an die zuständige Steuerbehörde gezahlt haben, müssen Sie eine WHT-Anforderung einreichen, um die Rückerstattung des Steuerbetrags zu fordern.

Ihre WHT-Anforderung muss die folgenden Elemente enthalten:

- Eine abgeschlossene Kopie des [Quellensteuergutschriftsformulars](https://download.microsoft.com/download/a/a/f/aaf8306b-79d4-455b-975f-41ce9e67b9cb/wht%20credit%20form%20-%20global.docx) (ausgefüllt vom Kunden)
- Eine signierte oder gescannte Kopie des Quellensteuerzertifikats oder -belegs

Übermitteln Sie die WHT-Anforderung, indem Sie ein Ticket mit dem Microsoft-Support öffnen.

> [!IMPORTANT]
>
> - Kunden können nur nach Zahlung der Rechnung eine Anforderung für eine Anpassung oder Rückerstattung des WHT-Betrags einreichen.
> - Der Rechnungsbetrag auf dem Formular zur Quellensteuergutschrift muss mit dem Rechnungsbetrag übereinstimmen, der im Quellensteuerzertifikat oder -beleg angegeben ist. Wenn der Rechnungsbetrag zwischen den beiden Formularen unterschiedlich ist, müssen Sie den Grund für die Differenz im Formular "Quellensteuergutschrift" angeben. Diese Informationen werden vom Überprüfungsteam überprüft, das gegebenenfalls klärende Fragen stellt.
> - Quellensteuerzertifikat- oder Belegdateien müssen in einem der folgenden Dateiformate vorliegen: .PDF oder Bild (. JPEG, .PNG oder .GIF). Darüber hinaus dürfen Dateinamen keine Leerzeichen oder Sonderzeichen enthalten. Die Dateigröße darf 1 MB nicht überschreiten.

Nachdem Sie die Anforderung übermittelt haben, wird sie in den Genehmigungsprozess ein, wo sie entweder zur Fertigstellung genehmigt oder zur Korrektur an Sie zurückgeschickt wird.

Wenn ein Problem mit Ihrer Anforderung vor liegt, muss das Überprüfungsteam möglicherweise Korrekturen am Quellenbetrag oder an der Ersetzung des Zertifikats oder der Quittung vornehmen. Sie müssen die Anforderung erneut übermitteln, bevor sie genehmigt werden kann. Das Überprüfungsteam genehmigt die Anforderung oder fordert weitere Änderungen an.

## <a name="approved-requests"></a>Genehmigte Anforderungen

**Für Kunden, die per Scheck oder Überweisung bezahlen:** Genehmigte WHT-Anforderungen werden mit dem unbezahlten Teil des Rechnungsbetrags abgerechnet, der im Formular zur Quellensteuergutschrift widergespiegelt wird.

Nachdem Ihr Anspruch genehmigt wurde, wird er im nächsten Abrechnungszyklus angezeigt. Der gezahlte WHT-Betrag ist im Zahlungsabschnitt Ihrer nächsten Rechnung enthalten. Der Betrag wird auch unter dem bezahlten Betrag im Kundenportal angezeigt.

**Für Kunden, die per Kreditkarte bezahlen:** Nachdem Ihr Anspruch genehmigt wurde, wird Ihre Überzahlung auf Ihre Kreditkarte zurückgezahlt.

> [!IMPORTANT]
>
> - Wenn Änderungen erforderlich sind, kann der Genehmigungsprozess aufgrund der korrekturen länger dauern, die vorgenommen und dann erneut übermittelt werden müssen.
> - Wenn Sie Fragen zum WHT-Anforderungsprozess haben, öffnen Sie bitte ein Ticket beim Microsoft-Support.
