# <a name="enter-your-office-365-for-business-product-key-purchased-from-microsoft-open"></a>Eingabe Ihres Office 365 Business-Product Keys, den Sie von Microsoft Open erworben haben

Wenn Sie Office 365 für Unternehmen von einem [Microsoft Open License-Programm](https://go.microsoft.com/fwlink/p/?LinkID=613298)erworben haben, ist dieser Artikel für Sie. Volumenlizenzkunden, die einen Product Key gekauft haben, können diesem Thema folgen, um Lizenzen zu aktivieren, zu verlängern oder einem Abonnement hinzuzufügen.
  
 **Benötigen Sie sofort Hilfe?** [Rufen Sie den Microsoft-Support an](../admin/contact-support-for-business-products.md) . 
  
 **Benötigen Sie etwas anderes?**
 
 **Hilfe zu Office 365 privaten oder persönlichen**Informationen finden Sie unter [Verwenden von Produktschlüsseln mit Office](https://support.office.com/article/12a5763a-d45c-4685-8c95-a44500213759.aspx).
  
 **Hilfe zu Office 365 von einem Einzelhandelsgeschäft erworbenen Business-Premium**finden Sie unter [eingeben des von einem Einzelhandelsgeschäft erworbenen Product Keys](enter-your-product-key.md). 
  
## <a name="activate-renew-or-add-licenses-to-a-subscription-purchased-from-microsoft-open"></a>Aktivieren, Verlängern oder Hinzufügen von Lizenzen zu einem Abonnement, das über Microsoft Open erworben wurde

Wenn Sie Office 365 von einem Microsoft Open License-Programm erworben haben, wird der Schlüssel automatisch von dem System eingegeben, wenn Sie Lizenzen zu einem Abonnement aktivieren, verlängern oder hinzufügen.
  
1. Nachdem Sie einen Product Key von einem Office 365-Partner erworben haben, überprüfen Sie Ihren Posteingang auf eine E-Mail von Microsoft, die einen Aktivierungslink enthält.
    
    > [!TIP]
    >  Diese E-Mail wird an die Person gesendet, die im Volumenlizenzvertrag als Online-Administrator aufgeführt ist. Wenn Sie nicht sicher sind, wer als Online-Administrator festgelegt wurde, fragen Sie Ihren Office 365-Partner. 
  
2. Klicken Sie auf den Link in der E-Mail, um sich mit Ihrem Geschäfts-, Schul- oder Unikonto anzumelden, akzeptieren Sie den Open License-Lizenzvertrag, und erneuern Sie das Abonnement. Eine ausführliche exemplarische Vorgehensweise für diesen Prozess finden Sie im PDF-Leitfaden [Aktivieren von Onlinediensten: Office 365-Abonnement](https://go.microsoft.com/fwlink/p/?LinkId=618100). Hier finden Sie Anleitungen zum Aktivieren ein neues Abonnements, zum Erneuern eines vorhandenes Abonnements oder Hinzufügen weiterer Benutzerlizenzen zu einem vorhandenen Abonnement.
    
> [!NOTE]
> Volumenlizenzkunden können auch Abonnements im [Volumen Lizenzierungs-Service Center](https://go.microsoft.com/fwlink/p/?LinkID=282016)aktivieren. Wie das geht, erfahren Sie, wenn Sie den PDF-Leitfaden [Aktivieren von Onlinediensten im Volume Licensing Service Center](https://go.microsoft.com/fwlink/p/?LinkId=618096) herunterladen.
