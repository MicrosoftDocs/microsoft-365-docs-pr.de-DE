# <a name="payment-information-by-deposit-or-electronic-transfer-only-for-mexico"></a>Zahlungsinformationen per Einzahlung oder elektronischer Überweisung (nur für Mexiko)

[Suchen Sie ein anderes Land oder eine andere Region.](../billing-and-payments/pay-for-your-subscription.md).

So bezahlen Sie per Einzahlung oder elektronischer Überweisung:

1. Wählen Sie beim Auschecken die Zahlungsmethode "Rechnung (Zahlung per Scheck oder Überweisung)" aus. Innerhalb der nächsten 24 bis 72 Stunden erhalten Sie die Rechnung des Produkts in der als Kontakt registrierten E-Mail. Die Rechnung enthält die Anweisungen zum Abschließen der Zahlung.
2. Bezahlen Sie an das folgende Bankkonto (damit Ihre Zahlung ordnungsgemäß verarbeitet wird, geben Sie bitte den Namen des Käufers und die entsprechende Rechnungsnummer an):  

**Bank:** Citibank/Banamex  
**Branch:** Act. Roberto Medellin 800, P.B. Sur, Col. Santa Fe, Mexico City C.P. 01210, Mexiko  
**SWIFT-Code:** BNMXMM  
**CLABE** 002180002337160225  
**Kontonummer:** 0233716022  
**Kontoname:** Microsoft Payments Inc.  
**Währung:** Pesos in Mexiko

## <a name="important-information"></a>Wichtige Informationen

1. Die Zahlung per Einzahlung oder elektronischer Überweisung ist nur für Beträge verfügbar, die größer als MXN $3.500 sind.
2. Die Zahlung muss innerhalb von 30 Tagen vorgenommen werden. Andernfalls wird die Bestellung automatisch abgebrochen.
