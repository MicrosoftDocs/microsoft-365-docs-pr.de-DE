# <a name="add-licenses-to-or-extend-a-subscription-paid-for-using-a-product-key"></a>Hinzufügen von Lizenzen zu oder verlängern eines Abonnements, das für die Verwendung eines Product Keys bezahlt wurde

Prepaid-Lizenzen werden Ihnen als 25-stelliger alphanumerischer Code ausgestellt. Nachdem Sie die benötigten Lizenzen erworben haben, können Sie Sie zu Ihrem Abonnement hinzufügen. Sie können auch einen Product Key verwenden, um das Ablaufdatum Ihres Abonnements zu verlängern.

> [!NOTE]
> Wenn Sie keinen neuen Product Key kaufen möchten, können Sie jederzeit eine Kreditkarte oder ein Bankkonto zu Ihrem Abonnement hinzufügen, um weitere Lizenzen zu bezahlen oder ihr Ablaufdatum für das Abonnement zu verlängern. Weitere Informationen finden Sie unter [Erneuern Ihres Abonnements](../subscriptions/renew-your-subscription.md).
  
## <a name="add-licenses-to-your-subscription"></a>Hinzufügen von Lizenzen zu Ihrem Abonnement

::: moniker range="o365-worldwide"

1. Wechseln Sie im Admin Center zur Seite **Fakturierung** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">ihrer Produkte</a> .

2. Suchen Sie nach dem Abonnement, dem Sie Lizenzen hinzufügen möchten. Wählen Sie im Abschnitt **Lizenzen** die Option **Weitere Lizenzen hinzufügen**aus.

3. Wählen Sie im **Bereich weitere Lizenzen hinzufügen**die Option **neuen und nicht verwendeten Product Key verwenden**aus, und wählen Sie dann **weiter**aus.

4. Geben Sie den Product Key ein, und wählen Sie dann **weiter**aus.

    > [!NOTE]
    > Wenn Sie mehr als einen Product Key hinzufügen möchten, können Sie **Weiteren Product Key hinzufügen** auswählen und diesen dann eingeben.
  
5. Überprüfen Sie Ihre Bestelldetails, und wählen Sie dann **einlösen**aus.

::: moniker-end

::: moniker range="o365-germany"

1. Navigieren Sie im Admin Center zur Seite **Abrechnung** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">Produkte und Dienste</a>.

2. Suchen Sie nach dem Abonnement, dem Sie Lizenzen hinzufügen möchten. Wählen Sie die Dropdownliste **Weitere Aktionen** aus, und wählen Sie dann **Lizenz mit Token erneuern oder hinzufügen**aus.

3. Wählen Sie auf der Seite **Benutzerlizenzen erneuern oder hinzufügen** die Option **neuen und nicht verwendeten Product Key verwenden**aus, und wählen Sie dann **weiter**aus.

4. Geben Sie den Product Key ein, und wählen Sie dann **weiter**aus.

    > [!NOTE]
    > Wenn Sie mehr als einen Product Key hinzufügen möchten, können Sie **Weiteren Product Key hinzufügen** auswählen und diesen dann eingeben.

5. Wählen Sie aus, um Benutzer zum Abonnement hinzuzufügen, und wählen Sie dann **weiter**aus.

6. Überprüfen Sie Ihre Bestelldetails, und wählen Sie dann **einlösen**aus.

::: moniker-end

::: moniker range="o365-21vianet"

1. Wechseln Sie im Admin Center zur Seite **Abrechnungs** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">Produkte & Dienste</a> .  

2. Suchen Sie nach dem Abonnement, dem Sie Lizenzen hinzufügen möchten. Wählen Sie die Dropdownliste **Weitere Aktionen** aus, und wählen Sie dann **Lizenz mit Token erneuern oder hinzufügen**aus.

3. Wählen Sie auf der Seite **Benutzerlizenzen erneuern oder hinzufügen** die Option **neuen und nicht verwendeten Product Key verwenden**aus, und wählen Sie dann **weiter**aus.

4. Geben Sie den Product Key ein, und wählen Sie dann **weiter**aus.

    > [!NOTE]
    > Wenn Sie mehr als einen Product Key hinzufügen möchten, können Sie **Weiteren Product Key hinzufügen** auswählen und diesen dann eingeben.

5. Wählen Sie aus, um Benutzer zum Abonnement hinzuzufügen, und wählen Sie dann **weiter**aus.

6. Überprüfen Sie Ihre Bestelldetails, und wählen Sie dann **einlösen**aus.

::: moniker-end
  
## <a name="extend-the-expiration-date-of-your-subscription"></a>Verlängern des Ablaufdatums Ihres Abonnements

::: moniker range="o365-worldwide"

1. Wechseln Sie im Admin Center zur Seite **Fakturierung** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">ihrer Produkte</a> .

2. Suchen Sie das Abonnement, das Sie erweitern möchten. Wählen Sie im Abschnitt **Abrechnung** die Option **End-Datum verlängern**aus.

3. Wählen Sie im Bereich **Abonnement verlängern** die Option **neuen und nicht verwendeten Product Key verwenden**aus, und wählen Sie dann **weiter**aus.

4. Geben Sie den Product Key ein, und wählen Sie dann **weiter**aus.

    > [!NOTE]
    > Wenn Sie mehr als einen Product Key hinzufügen möchten, können Sie **Weiteren Product Key hinzufügen** auswählen und diesen dann eingeben.
  
5. Überprüfen Sie Ihre Bestelldetails, und wählen Sie dann **einlösen**aus.
