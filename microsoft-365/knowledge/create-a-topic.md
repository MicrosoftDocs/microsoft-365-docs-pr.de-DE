# <a name="create-a-new-topic-preview"></a>Erstellen eines neuen Themas (Vorschau)

> [!Note] 
> Der Inhalt dieses Artikels ist für Project Cortex Private Preview vorgesehen. [Erfahren Sie mehr über Project Cortex](https://aka.ms/projectcortex).

In "Themenerfahrungen" können Sie ein neues Thema erstellen, wenn eines nicht durch Indizierung gefunden wurde oder wenn die KI-Technologie nicht genügend Nachweise gefunden hat, um es als Thema zu finden.

## <a name="requirements"></a>Anforderungen

Um ein neues Thema zu erstellen, müssen Sie:
- Sie verfügen über eine Topic Experiences-Lizenz.
- Sie verfügen über Berechtigungen für [**Personen, die Themen erstellen oder bearbeiten können.**](https://docs.microsoft.com/microsoft-365/knowledge/topic-experiences-user-permissions) Wissensadministratoren können Benutzern diese Berechtigung in den Themeneinstellungen des Knowledge Network erteilen. 

> [!Note] 
> Benutzer, die über die Berechtigung zum Verwalten von Themen im Themencenter (Knowledge Manager) verfügen, verfügen bereits über Berechtigungen zum Erstellen und Bearbeiten von Themen.

## <a name="to-create-a-new-topic"></a>So erstellen Sie ein neues Thema:

1. Wählen Sie auf der Seite "Themencenter" **"Neu"** und dann **"Themenseite" aus.** Wenn die Option "Neu" im Themencenter nicht angezeigt wird, verfügen Sie möglicherweise nicht über die erforderlichen Berechtigungen. 

    ![Neues Thema](../media/knowledge-management/k-new-topic.png)

2. Auf der Seite "Neues Thema" können Sie die Informationen zur neuen Themenvorlage ausfüllen:

    - Geben Sie **im Abschnitt "Name dieses Themas"** den Namen des neuen Themas ein.
    
    - Geben Sie **im Abschnitt "Alternative** Namen" Namen oder Akronyme ein, die auch zum Verweisen auf das Thema verwendet werden.
    
    - Geben Sie **im Abschnitt "Kurzbeschreibung"** eine beschreibung des Themas mit einem oder zwei Sätzen ein. Dieser Text wird für die zugeordnete Themenkarte verwendet.
    
    - Geben Sie **im** Abschnitt "Personen" die Namen der Experten für das Thema ein. Personen, die Sie manuell zum Thema hinzufügen, werden auf der Themenseite als **angeheftet angezeigt.**
    
    - Wählen Sie **im Abschnitt "Dateien** und **Seiten"** "Hinzufügen" aus, und auf der nächsten Seite können Sie zugeordnete #A0 oder SharePoint #A1 auswählen.
    
    - Wählen Sie **im Abschnitt "Websites"** die Option **"Hinzufügen" aus.** Wählen Sie  **im angezeigten** Bereich "Websites" die Websites aus, die dem Thema zugeordnet sind.

    ![Seite "Neues Thema"](../media/knowledge-management/k-new-topic-page.png)
    
3. Wenn Sie der Seite weitere Komponenten hinzufügen müssen, z. B. Text, Bilder, Webparts, Links usw., wählen Sie das Zeichenbereichssymbol in der Mitte der Seite aus, um sie zu suchen und hinzuzufügen.

    ![Hinzufügen von Elementen zur Seite](../media/knowledge-management/static-icon.png)

4. Wenn Sie fertig sind, wählen Sie **"Veröffentlichen"** aus, um die Themenseite zu veröffentlichen. Veröffentlichte Themenseiten werden auf der Registerkarte **"Seiten"** angezeigt.

Nachdem Sie den Artikel veröffentlicht haben, werden der Name des Themas, der alternative Name, die Beschreibung und angeheftete Personen allen lizenzierten Benutzern angezeigt, die den Artikel anzeigen. Dateien, Seiten und Websites werden nur dann auf der Themenseite angezeigt, wenn der Viewer über Office 365-Berechtigungen für das Element verfügt. 

Die neue Themenseite besteht aus Webparts, die Überkenntnisse des *Wissensnetzwerks haben.* Dies bedeutet, dass die Informationen in diesen Webparts mit Vorschlägen aktualisiert werden, um die Seite für Benutzer nützlicher zu machen, wenn ai weitere Informationen zu diesem Thema sammelt.
