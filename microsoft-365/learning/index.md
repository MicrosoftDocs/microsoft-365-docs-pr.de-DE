# <a name="introduction-to-microsoft-viva-learning-preview"></a>Einführung in Microsoft Viva Learning (Vorschau)

> [!NOTE]
> Die Informationen in diesem Artikel beziehen sich auf ein Vorschauprodukt, das möglicherweise erheblich geändert wird, bevor es kommerziell veröffentlicht wird. 

Microsoft Viva Learning (Preview) ermöglicht Teams und Einzelpersonen in Ihrer Organisation, das Lernen zu einem natürlichen Bestandteil ihres Tages zu machen. Die App erstellt einen zentralen Hub in Microsoft Teams, in dem Mitarbeiter Inhaltsbibliotheken in Ihrer Organisation freigeben, zuweisen und daraus lernen können.

Die Ressourcen auf dieser Seite sind so konzipiert, dass Sie mit dem Erlernen und Verwenden von Viva Learning (Preview) in Ihrer Organisation beginnen.
