# <a name="share-files-and-videos"></a>Freigeben von Dateien und Videos

Wenn Sie steuern müssen, wer Ihre Dateien anzeigen und bearbeiten kann, müssen Sie die Dateien an einem sicheren Speicherort speichern, an dem Sie sicherstellen können, dass Berechtigungen entsprechend angewendet werden. Sie können Microsoft Teams verwenden, um Ihre Dateien zu speichern und die Dateien entweder innerhalb oder außerhalb Ihrer Firma, Praxis oder Kampagne zu teilen, indem Sie Microsoft Teams verwenden oder SharePoint senden. Das Senden eines Links anstelle einer E-Mail-Anlage bedeutet, dass Sie wissen, wer die Dateien sieht und ändert, und sie können ohne Ihre Erlaubnis nicht angezeigt oder geändert werden.

![Diagramm eines fensters Microsoft Teams, das die Registerkarte Dateien und den Link "Get" im Menü zeigt](../media/m365-democracy-teams-sharefiles.png)

Mit Ihren Dateien in Microsoft Teams und SharePoint können Sie auch gemeinsam an den Dateien arbeiten und sich gegenseitig die Änderungen überprüfen. Verwenden Microsoft Teams, um Dateien innerhalb eines Unternehmens, einer Praxis oder einer Kampagne zu teilen. Wenn Sie externe Freigaben für Personen außerhalb Ihrer Organisation benötigen, können Sie sie als Gäste zu einem Team hinzufügen oder ihnen einen sicheren SharePoint senden.

Sie können Microsoft Stream auch verwenden, um Videos intern zu speichern und zu teilen. Diese Videos können nicht öffentlich angezeigt werden, sodass sie ideal für interne Kampagnennachrichten geeignet sind.

## <a name="best-practices"></a>Bewährte Methoden

Verwenden Sie diese Methoden, um Dateien und Videos sicher zu teilen:

1. Store dateien in Microsoft Teams oder SharePoint, und stellen Sie sicher, dass nur die Personen, die Zugriff auf diese Dateien benötigen, über diese Dateien verfügen.
2. Wenn Sie freigeben möchten, fügen Sie keine Dateien an eine E-Mail an. Wählen Sie stattdessen **Link aus Microsoft Teams** oder SharePoint und senden Sie den Link per E-Mail.
3. Um eine Datei extern zu teilen, fügen Sie den Benutzer als Gast zu Ihrem Team hinzu, oder verwenden Sie SharePoint, um einen sicheren Link zum Freigeben dieser Datei zu erhalten.
4. Verwenden Sie Microsoft Stream, um Videos zu hosten, die Ihre Kampagne anzeigen soll.
5. Verwenden Microsoft Teams oder SharePoint zum Speichern von Videodateien, an deren Zusammenarbeit oder Freigabe Ihr Team benötigt wird.
