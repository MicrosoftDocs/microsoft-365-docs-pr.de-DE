# <a name="set-up-meetings"></a>Einrichten von Besprechungen

Besprechungen in Microsoft Teams umfassen Audio, Video und Freigabe. Und da sie online sind, verfügen Sie immer über einen Besprechungsraum (ohne einen Raum oder Einen Projektor zu benötigen!), auch wenn Ihre Mitarbeiter geografisch verteilt sind oder remote arbeiten. Microsoft Teams-Besprechungen sind eine hervorragende Möglichkeit, mit Ihren Mitarbeitern innerhalb und außerhalb Ihrer Organisation zusammen zu kommen. Sie müssen kein Mitglied Ihrer Organisation sein oder über ein Konto verfügen, um an einer Besprechung teilnehmen zu können. Sie können Onlinebesprechungen mit Microsoft Teams planen und ausführen. Während einer Besprechung können Sie Ihren Bildschirm freigeben, Dateien freigeben, Aufgaben zuweisen und vieles mehr. Zu den politischen Kampagnen können Mitarbeiter, Freiwilligen und Gäste außerhalb Ihrer Organisation gehören. Kleine Unternehmen oder Praktiken können sich mit ihren Mitarbeitern treffen oder sich über Microsoft Teams mit Kunden oder Partnern treffen.

[![Abbildung von zwei Benutzern in einer Besprechung](../media/HostOnlineMeeting-thumb-358x201.png)](https://go.microsoft.com/fwlink/?linkid=2078712)

Laden Sie eine Infografik in [PDF](https://go.microsoft.com/fwlink/?linkid=2078712) oder [PowerPoint herunter,](https://go.microsoft.com/fwlink/?linkid=2079515) um einen schnellen Überblick über die Teilnahme an oder das Hosten einer Online-Besprechung mit Microsoft Teams zu erhalten.

## <a name="best-practices"></a>Bewährte Methoden

Befolgen Sie die folgenden bewährten Methoden für Ihre Onlinebesprechungen:

- Planen Sie Ihre Onlinebesprechungen direkt in Microsoft Teams. Sie können ein Team und einen Kanal auswählen, und Teams lädt die Teilnehmer in diesem Team oder Kanal automatisch ein.
- Benötigen Sie eine ungeahnte Besprechung? Wenn Sie in einem 1:1-Chat sind, wählen Sie "Jetzt treffen" aus, um einen Video- oder Audioanruf mit der Person zu starten, mit der Sie chatten.
