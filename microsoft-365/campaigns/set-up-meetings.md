# <a name="set-up-meetings"></a>Einrichten von Besprechungen

Besprechungen in Microsoft Teams umfassen Audio, Video und Freigabe. Und da sie online sind, haben Sie immer einen Besprechungsraum (ohne Raum oder Projektor!), auch wenn Ihre Mitarbeiter geografisch verteilt sind oder remote arbeiten. Microsoft Teams Besprechungen sind eine großartige Möglichkeit, mit Ihren Mitarbeitern innerhalb und außerhalb Ihrer Organisation zusammen zu kommen. Sie müssen kein Mitglied Ihrer Organisation sein oder über ein Konto verfügen, um an einer Besprechung teilnehmen zu können. Sie können Onlinebesprechungen mithilfe von Microsoft Teams. Während einer Besprechung können Sie Ihren Bildschirm freigeben, Dateien freigeben, Aufgaben zuweisen und vieles mehr. Politische Kampagnen können Mitarbeiter, Freiwillige und Gäste außerhalb Ihrer Organisation in der Besprechung enthalten. Kleine Unternehmen oder Praxen können sich mit ihren Mitarbeitern treffen oder sich mit Kunden oder Partnern in anderen Microsoft Teams.

[![Eine Abbildung von zwei Benutzern in einer Besprechung](../media/HostOnlineMeeting-thumb-358x201.png)](https://go.microsoft.com/fwlink/?linkid=2078712)

Laden Sie eine Infografik in [PDF](https://go.microsoft.com/fwlink/?linkid=2078712) oder [PowerPoint,](https://go.microsoft.com/fwlink/?linkid=2079515) um einen schnellen Überblick über die Teilnahme oder das Hosten einer Online-Besprechung mit Microsoft Teams.

## <a name="best-practices"></a>Bewährte Methoden

Befolgen Sie die folgenden bewährten Methoden für Ihre Onlinebesprechungen:

- Planen Sie Ihre Onlinebesprechungen direkt in Microsoft Teams. Sie können ein Team und einen Kanal auswählen, und Teams werden die Teilnehmer in diesem Team oder Kanal automatisch eingeladen.
- Benötigen Sie eine impromptu-Besprechung? Wenn Sie in einem 1:1-Chat sind, wählen Sie Jetzt treffen aus, um einen Video- oder Audioanruf mit der Person zu starten, mit der Sie chatten.
