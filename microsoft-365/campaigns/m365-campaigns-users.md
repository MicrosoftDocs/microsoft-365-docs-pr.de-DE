# <a name="how-these-security-recommendations-affect-your-users"></a>Auswirkungen dieser Sicherheitsempfehlungen auf Ihre Benutzer

Die Sicherheitsempfehlungen für Microsoft 365 in dieser Lösung erschweren Hackern den Zugriff auf Ihre Umgebung. Der Tradeoff ist, dass Ihre Benutzer wissen müssen, wie sie in dieser sichereren Umgebung arbeiten. Wir wissen, dass etwas mehr Geduld erforderlich ist, aber es lohnt sich, Ihre Organisation zu schützen.

![Abbildung, die die wichtigsten Punkte von unten für iPhones, Android-Geräte, Macs, Windows 10, Freigabe und Wichtige Mitarbeiter zusammen fasst](../media/M365-democracy-Users_900px.png)

## <a name="use-secure-email-practices"></a>Verwenden sicherer E-Mail-Methoden

Alle Benutzer sollten diese E-Mail-Methoden kennen und verwenden, um ihre E-Mails sicher zu halten:

- Richten Sie E-Mails für die Verwendung der mehrstufigen Authentifizierung mit der Authenticator-App ein.
- Überprüfen Sie legitime E-Mails, und suchen Sie nach Sicherheitstipps von Advanced Phishing in Defender for Office 365 Protection.
- Öffnen Sie nur sichere Links und Anlagen, wie durch sichere Links und sichere Anlagen überprüft.

Erfahren Sie mehr [über mehrstufige Authentifizierung](m365-campaigns-multifactor-authenication.md) und [Phishing und andere Angriffe.](m365-campaigns-phishing-and-attacks.md)

Laden Sie [eine Infografik](m365-campaigns-protect-campaign-infographic.md) mit Tipps für Sie und die Mitglieder Ihres Teams herunter.

## <a name="set-up-iphones-and-android-devices"></a>Einrichten von iPhones und Android-Geräten

Alle Benutzer, die Sie Zu Ihrer Umgebung hinzufügen, benötigen einige Minuten, um [iPhones](../business/set-up-mobile-devices.md?toc=%2Fmicrosoft-365%2Fcampaigns%2Ftoc.json) und Android-Geräte sicher zu installieren:

- Richten Sie Geräte ein, um die mehrstufige Authentifizierung mit der Authenticator-App zu verwenden.
- Verwenden Sie mobile Microsoft-Apps, Outlook Mobile, Word, OneDrive und andere Microsoft-Apps aus dem App Store. Die systemeigenen E-Mail-Apps, die auf iPhones und Android-Geräten enthalten sind, werden nicht unterstützt. 
- Fordern Sie eine PIN an, damit Benutzer ihr Gerät entsperren können.

Nachdem Sie diese eingerichtet haben, werden Ihre Benutzer aufgefordert, die Authenticator-App zu verwenden, wenn sie auf Ihre Organisationsdaten auf diesen Geräten zugreifen, einschließlich E-Mails.

## <a name="keep-byod-macs-and-windows-10-pcs-fresh"></a>Halten Sie BYOD-Macs und Windows 10 PCs auf dem neusten

Außerdem ist es wichtig, dass Benutzer ihr primäres Arbeitsgerät auf dem neuesten Stand halten:

- Installieren Sie die neuesten Versionen Office Desktop-Apps, und halten Sie diese bei Aufforderung mit Updates auf dem neuesten Stand.
- Bleiben Sie auf dem Neuesten mit Betriebssystemupdates, z. B. Windows Updates.

## <a name="collaborate-using-microsoft-teams-onedrive-sharepoint-online-and-other-tools"></a>Zusammenarbeit mit Microsoft Teams, OneDrive, SharePoint Online und anderen Tools

Ihre Benutzer sind möglicherweise versucht, Ihre Organisationsdateien an anderen Orten als Microsoft 365. Microsoft 365 ermöglicht es so einfach wie möglich, sicher zusammenzuarbeiten und zu teilen. Sie können [Dateien und Videos](share-files-and-videos.md) direkt aus Microsoft Teams, OneDrive, Streamen und sogar aus einer Datei freigeben. Die Freigabe innerhalb dieser Tools trägt dazu bei, dass Ihre Daten nicht undicht werden. Sie können vertraulichen Daten zusätzlichen Schutz hinzufügen, um die Freigabe außerhalb Ihrer Organisation zu verhindern.

Wir zeigen Ihnen, wie Sie [diese Geräte verwalten und sichern.](../business/set-up-windows-devices.md?toc=/microsoft-365/campaigns/toc.json) Dadurch wird sichergestellt, dass Mitarbeiter, die das höchste Wertziel für Hacker sind, den höchsten Schutz erhalten.
