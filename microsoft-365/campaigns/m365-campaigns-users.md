# <a name="how-these-security-recommendations-affect-your-users"></a>Wie sich diese Sicherheitsempfehlungen auf Ihre Benutzer auswirken

Die Sicherheitsempfehlungen für Microsoft 365 in dieser Lösung erschweren Hackern den Zugriff auf Ihre Umgebung erheblich. Der Kompromiss besteht darin, dass Ihre Benutzer wissen müssen, wie sie in dieser sichereren Umgebung arbeiten können. Wir wissen, dass ein wenig zusätzliche Geduld erforderlich ist, aber es lohnt sich, Ihre Organisation zu schützen.

![Abbildung, die wichtige Punkte von unten für iPhones, Android-Geräte, Macs, Windows 10, Freigaben und Wichtige Mitarbeiter zusammenfasst](../media/M365-democracy-Users_900px.png)

## <a name="use-secure-email-practices"></a>Verwenden sicherer E-Mail-Methoden

Alle Benutzer sollten diese E-Mail-Methoden kennen und verwenden, um ihre E-Mails zu schützen:

- Richten Sie E-Mails für die Verwendung der mehrstufigen Authentifizierung mit der Authentifikator-App ein.
- Überprüfen Sie seriöse E-Mails, und suchen Sie nach Sicherheitstipps von Advanced Phishing in Defender for Office 365 Protection.
- Öffnen Sie nur sichere Links und Anlagen, wie durch sichere Links und sichere Anlagen überprüft.

Erfahren Sie mehr über [mehrstufige Authentifizierung](m365-campaigns-multifactor-authenication.md) und [Phishing und andere Angriffe.](m365-campaigns-phishing-and-attacks.md)

Laden Sie eine [Infografik](m365-campaigns-protect-campaign-infographic.md) mit Tipps für Sie und die Mitglieder Ihres Teams herunter.

## <a name="set-up-iphones-and-android-devices"></a>Einrichten von iPhones und Android-Geräten

Alle Benutzer, die Sie Zu Ihrer Umgebung hinzufügen, müssen einige Minuten benötigen, um [iPhones und Android-Geräte](../business/set-up-mobile-devices.md?toc=%2Fmicrosoft-365%2Fcampaigns%2Ftoc.json) einzurichten, damit sie sicher funktionieren:

- Richten Sie Geräte für die Verwendung der mehrstufigen Authentifizierung mit der Authentifikator-App ein.
- Verwenden Sie mobile Microsoft-Apps, einschließlich Outlook Mobile, Word, OneDrive und andere Microsoft-Apps aus dem App Store. Die nativen Mail-Apps, die auf iPhones und Android-Geräten enthalten sind, werden nicht unterstützt. 
- Fordern Sie eine PIN an, damit Benutzer ihr Gerät entsperren können.

Nach der Einrichtung werden Ihre Benutzer aufgefordert, die Authentifikator-App zu verwenden, wenn sie auf Ihre Organisationsdaten auf diesen Geräten zugreifen, einschließlich E-Mails.

## <a name="keep-byod-macs-and-windows-10-pcs-fresh"></a>BYOD Macs und Windows 10 PCs auf dem neu zu halten

Außerdem ist es wichtig, dass Benutzer ihr primäres Arbeitsgerät auf dem neuesten Stand halten:

- Installieren Sie die neuesten Versionen von Office Desktop-Apps, und halten Sie diese mit Updates auf dem neuesten Stand, wenn Sie dazu aufgefordert werden.
- Bleiben Sie auf dem Laufenden über Betriebssystemupdates, z. B. Windows Updates.

Bei [nicht verwalteten Windows 10- und Mac-Geräten](m365-campaigns-protect-pcs-macs.md)müssen Benutzer sicherstellen, dass grundlegende Sicherheitsfeatures aktiviert sind.

## <a name="collaborate-using-microsoft-teams-onedrive-sharepoint-online-and-other-tools"></a>Zusammenarbeit mit Microsoft Teams, OneDrive, SharePoint Online und anderen Tools

Ihre Benutzer sind möglicherweise versucht, Ihre Organisationsdateien an anderen Stellen als Microsoft 365 zu teilen und zu speichern. Microsoft 365 erleichtert die Zusammenarbeit und den sicheren Austausch. Sie können Dateien und Videos direkt aus Microsoft Teams, OneDrive, Stream und sogar aus einer Datei [heraus freigeben.](share-files-and-videos.md) Die Freigabe innerhalb dieser Tools trägt dazu bei, dass Ihre Daten nicht offengelegt werden. Sie können sensiblen Daten zusätzlichen Schutz hinzufügen, um die Freigabe außerhalb Ihrer Organisation zu verhindern.

## <a name="set-up-managed-windows-10-devices"></a>Einrichten von verwalteten Windows 10 Geräten

Wir empfehlen, dass Ihre wichtigsten Mitarbeiter neu erworbene Windows 10 Geräte verwenden, die Sie verwalten. Wir zeigen Ihnen, wie [Sie diese Geräte verwalten und schützen.](../business/set-up-windows-devices.md?toc=/microsoft-365/campaigns/toc.json) Dadurch wird sichergestellt, dass Mitarbeiter, die das höchste Ziel für Hacker sind, den größten Schutz erhalten.
