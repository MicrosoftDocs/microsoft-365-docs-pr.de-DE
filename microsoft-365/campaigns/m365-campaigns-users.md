# <a name="how-these-security-recommendations-affect-your-users"></a>Auswirkungen dieser Sicherheitsempfehlungen auf Ihre Benutzer

Die Sicherheitsempfehlungen für Microsoft 365 in dieser Lösung erschweren Hackern den Zugriff auf Ihre Umgebung deutlich. Der Tradeoff ist, dass die Benutzer wissen müssen, wie sie in dieser sichereren Umgebung arbeiten. We understand a little extra patience is required, but it's worth it to keep your organization protected.

![Abbildung, die die wichtigsten Punkte von unten für iPhones, Android-Geräte, Macs, Windows 10, Freigabe und wichtige Mitarbeiter zusammenstellen](../media/M365-democracy-Users_900px.png)

## <a name="use-secure-email-practices"></a>Verwenden sicherer E-Mail-Methoden

Alle Benutzer sollten diese E-Mail-Methoden kennen und verwenden, um die Sicherheit ihrer E-Mails zu gewährleisten:

- Richten Sie E-Mails so ein, dass die mehrstufige Authentifizierung mit der Authentisierungs-App verwendet wird.
- Überprüfen Sie legitime E-Mails, und suchen Sie nach Sicherheitstipps von Advanced Phishing in Defender for Office 365 Protection.
- Öffnen Sie nur sichere Links und Anlagen, wie durch sichere Links und sichere Anlagen überprüft.

Erfahren Sie mehr über [mehrstufige Authentifizierung](m365-campaigns-multifactor-authenication.md) und [Phishing und andere Angriffe.](m365-campaigns-phishing-and-attacks.md)

Laden Sie [eine Infografik](m365-campaigns-protect-campaign-infographic.md) mit Tipps für Sie und die Mitglieder Ihres Teams herunter.

## <a name="set-up-iphones-and-android-devices"></a>Einrichten von iPhones und Android-Geräten

Alle Benutzer, die Sie Ihrer Umgebung hinzufügen, müssen einige Minuten in Die einrichtung [von iPhones](../business/set-up-mobile-devices.md?toc=%2Fmicrosoft-365%2Fcampaigns%2Ftoc.json) und Android-Geräten, um sicher zu arbeiten:

- Richten Sie Geräte für die Verwendung der mehrstufigen Authentifizierung mit der Authentisierungs-App ein.
- Verwenden Sie mobile Microsoft-Apps, einschließlich Outlook Mobile, Word, OneDrive und andere #A0 aus dem App Store. Die systemeigenen Mail-Apps, die auf iPhones und Auf -Android-Geräten enthalten sind, werden nicht unterstützt. 
- Fordern Sie eine PIN an, damit Benutzer ihr Gerät entsperren können.

Nachdem Sie diese eingerichtet haben, werden Ihre Benutzer aufgefordert, die Authentator-App zu verwenden, wenn sie auf ihre Unternehmensdaten auf diesen Geräten zugreifen, einschließlich E-Mails.

## <a name="keep-byod-macs-and-windows-10-pcs-fresh"></a>Halten Sie BYOD Macs und Windows 10-PCs auf dem Neusten

Außerdem ist es wichtig, dass Benutzer ihr primäres Arbeitsgerät auf dem neuesten Stand halten:

- Installieren Sie die neuesten Versionen von Office-Desktop-Apps, und halten Sie diese bei Aufforderung mit Updates auf dem neuesten Stand.
- Bleiben Sie über Betriebssystemupdates auf dem Neuesten, z. B. Windows-Updates.

Bei [nicht verwalteten Windows 10-](m365-campaigns-protect-pcs-macs.md)und Mac-Geräten müssen Benutzer sicherstellen, dass grundlegende Sicherheitsfeatures aktiviert sind.

## <a name="collaborate-using-microsoft-teams-onedrive-sharepoint-online-and-other-tools"></a>Zusammenarbeit mit Microsoft Teams, OneDrive, SharePoint Online und anderen Tools

Ihre Benutzer sind möglicherweise versucht, Ihre Organisationsdateien an anderen Orten als Microsoft 365 zu teilen und zu speichern. Microsoft 365 erleichtert die Zusammenarbeit und sichere Freigabe. Sie können [Dateien und Videos direkt](share-files-and-videos.md) aus Microsoft Teams, OneDrive, Stream und sogar aus einer Datei freigeben. Durch die Freigabe innerhalb dieser Tools können Sie datenlecks schützen. Sie können vertraulichen Daten zusätzlichen Schutz hinzufügen, um die Freigabe außerhalb Ihrer Organisation zu verhindern.

## <a name="set-up-managed-windows-10-devices"></a>Einrichten verwalteter Windows 10-Geräte

Es wird empfohlen, dass Ihre wichtigsten Mitarbeiter erworbene Windows 10-Geräte verwenden, die Sie verwalten. Wir zeigen Ihnen, wie Sie [diese Geräte verwalten und schützen.](../business/set-up-windows-devices.md?toc=/microsoft-365/campaigns/toc.json) Dadurch wird sichergestellt, dass Mitarbeiter, die das höchste Ziel für Hacker sind, den größten Schutz erhalten.
