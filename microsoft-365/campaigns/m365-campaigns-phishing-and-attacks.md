# <a name="protect-yourself-against-phishing-and-other-attacks"></a>Schützen Sie sich vor Phishing und anderen Angriffen

Microsoft 365 umfasst viele Schutzmaßnahmen zum Schutz vor Angriffen, aber Es gibt Dinge, die Sie selbst tun können, um das Risiko von Onlineangriffen zu begrenzen und zu minimieren.

- **Spam- oder Junk-E-Mails** Es gibt viele Gründe, warum Sie Junk-E-Mails erhalten, und nicht alle Junk-E-Mails sind identisch. Sie können jedoch reduzieren, was sie durchkommt, und somit die Risiken von Angriffen verringern, indem Sie Junk-E-Mails filtern.
- **Phishing** Ein Phishing-Betrug ist eine E-Mail, die legitim erscheint, aber ein Versuch ist, Ihre persönlichen Informationen zu erhalten oder Ihr Geld zu stehlen.
- **Spoofing** Betrüger können auch eine Technik namens Spoofing verwenden, um sie so zu erscheinen, als hätten Sie eine E-Mail von sich selbst erhalten. 
- **Schadsoftware** ist Schadsoftware, die auf Ihrem Computer installiert werden kann, in der Regel installiert, nachdem Sie auf einen Link geklickt oder ein Dokument aus einer E-Mail geöffnet haben. Es gibt verschiedene Arten von Schadsoftware (z. B. Ransomware, wenn Ihr Computer übernommen wird), aber Sie möchten keine davon haben. 

## <a name="best-practices"></a>Bewährte Methoden

### <a name="reduce-spam-mail"></a>Reduzieren von Spam-E-Mails

Folgen Sie [diesen 10 Tipps, wie Sie Spam reduzieren können.](https://support.microsoft.com/en-us/office/10-tips-on-how-to-help-reduce-spam-55f756e8-688b-41c3-a086-8f68ccc592f6)

### <a name="report-it"></a>Melden Sie es!

Melden Sie phishing- oder andere E-Mails, die Sie erhalten. Wählen Sie die Nachricht aus, und wählen Sie **auf dem** Menüband Meldung melden aus.
Hier finden Sie weitere Informationen zum Melden [von Junk- und Phishing-E-Mails.](https://support.office.com/article/Use-the-Report-Message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2)

### <a name="avoid-phishing"></a>Vermeiden von Phishing

- Antworten Sie niemals auf eine E-Mail, in der Sie zum Senden von persönlichen Oder Kontoinformationen gefragt werden.
- Wenn Sie eine E-Mail erhalten, die verdächtig aussieht oder Sie nach dieser Art von Informationen fragt, klicken Sie nie auf Links, die Sie angeblich zu einer Unternehmenswebsite nehmen.
- Öffnen Sie niemals eine Datei, die an eine verdächtig aussehende E-Mail angefügt ist.
- Wenn die E-Mail von einem Unternehmen zu stammen scheint, wenden Sie sich per Telefon oder Webbrowser an den Kundendienst des Unternehmens, um zu überprüfen, ob die E-Mail legitim ist.
- Durchsuchen Sie das Web nach der Betreffzeile E-Mail gefolgt von dem Wort Hoax, um zu sehen, ob jemand anderen diesen Betrug gemeldet hat.

Informationen zu fünf gängigen Arten von Betrug finden Sie unter [Deal with abuse, phishing, or spoofing](https://support.office.com/article/Deal-with-abuse-phishing-or-spoofing-in-Outlook-com-0d882ea5-eedc-4bed-aebc-079ffa1105a3).

### <a name="make-sure-your-emails-look-legitimate-to-others"></a>Stellen Sie sicher, dass Ihre E-Mails für andere legitim aussehen

Helfen Sie Ihren Kunden, Ihrer Kommunikation zu vertrauen, indem Sie eine digitale Signatur hinzufügen, um zu beweisen, dass sie von Ihnen kommt. Weitere [Informationen finden Sie unter Sichern von Nachrichten mithilfe einer digitalen Signatur.](https://support.office.com/article/secure-messages-by-using-a-digital-signature-549ca2f1-a68f-4366-85fa-b3f4b5856fc6)
