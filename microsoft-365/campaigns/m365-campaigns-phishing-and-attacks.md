# <a name="protect-yourself-against-phishing-and-other-attacks"></a>Sich vor Phishing und anderen Angriffen schützen

Microsoft 365 bietet viele Schutzmaßnahmen zum Schutz vor Angriffen, aber es gibt Dinge, die Sie selbst tun können, um das Risiko von Onlineangriffen zu begrenzen und zu minimieren.

- **Spam oder Junk-E-Mail** Es gibt viele Gründe, warum Sie Junk-E-Mails erhalten, und nicht alle Junk-E-Mails sind identisch. Sie können jedoch die Benachrichtigungen reduzieren und so die Risiken von Angriffen verringern, indem Sie Junk-E-Mails herausfiltern.
- **Phishing** Ein betrügerischer Phishingversuch ist eine E-Mail, die legitim erscheint, aber ein Versuch ist, Ihre persönlichen Informationen zu erhalten oder Ihr Geld zu stehlen.
- **Spoofing** Scammer können auch eine Technik namens Spoofing verwenden, um sie so erscheinen zu lassen, als hätten Sie eine E-Mail von sich selbst erhalten. 
- **Schadsoftware** ist Schadsoftware, die auf Ihrem Computer installiert werden kann, die in der Regel installiert wird, nachdem Sie auf einen Link geklickt oder ein Dokument aus einer E-Mail geöffnet haben. Es gibt verschiedene Arten von Schadsoftware (z. B. Ransomware, wenn Ihr Computer übernommen wird), aber Sie möchten keine davon haben. 

## <a name="best-practices"></a>Bewährte Methoden

### <a name="reduce-spam-mail"></a>Reduzieren von Spam-E-Mails

Befolgen Sie [die folgenden 10 Tipps zur Reduzierung von Spam.](https://support.microsoft.com/en-us/office/10-tips-on-how-to-help-reduce-spam-55f756e8-688b-41c3-a086-8f68ccc592f6)

### <a name="report-it"></a>Melden Sie es!

Melden Sie phishing- oder andere betrügerische E-Mails, die Sie erhalten. Wählen Sie die Nachricht aus, und wählen Sie auf **dem** Menüband "Nachricht melden" aus.
Hier finden Sie weitere Informationen zum Melden [von Junk- und Phishing-E-Mails.](https://support.office.com/article/Use-the-Report-Message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2)

### <a name="avoid-phishing"></a>Vermeiden von Phishing

- Antworten Sie niemals auf eine E-Mail, in der Sie zum Senden persönlicher oder Kontoinformationen gefragt werden.
- Wenn Sie eine E-Mail erhalten, die verdächtig aussieht oder Sie nach diesen Informationen fragt, klicken Sie nie auf Links, die Sie zu einer Unternehmenswebsite verschicken sollen.
- Öffnen Sie niemals eine Datei, die an eine verdächtig aussehende E-Mail angefügt ist.
- Wenn die E-Mail von einem Unternehmen zu stammen scheint, wenden Sie sich per Telefon oder Webbrowser an den Kundendienst des Unternehmens, um zu sehen, ob die E-Mail legitim ist.
- Durchsuchen Sie das Web nach der E-Mail-Betreffzeile, gefolgt von dem Wort "Hoax", um zu sehen, ob jemand anderen diesen Betrug gemeldet hat.

Lesen Sie mehr über fünf gängige Arten von Betrugsversuchen im [Umgang mit Missbrauch, Phishing oder Spoofing.](https://support.office.com/article/Deal-with-abuse-phishing-or-spoofing-in-Outlook-com-0d882ea5-eedc-4bed-aebc-079ffa1105a3)

### <a name="make-sure-your-emails-look-legitimate-to-others"></a>Stellen Sie sicher, dass Ihre E-Mails für andere legitim aussehen

Helfen Sie Ihren Kunden, Ihrer Kommunikation zu vertrauen, indem Sie eine digitale Signatur hinzufügen, um nachzuweisen, dass sie von Ihnen kommt. Siehe ["Sichere Nachrichten mithilfe einer digitalen Signatur".](https://support.office.com/article/secure-messages-by-using-a-digital-signature-549ca2f1-a68f-4366-85fa-b3f4b5856fc6)
