# <a name="get-started-with-powershell-for-microsoft-365"></a>Erste Schritte mit PowerShell für Microsoft 365

*Dieser Artikel gilt sowohl für Microsoft 365 Enterprise als auch für Office 365 Enterprise.*

Sie können Befehle und Skripts in PowerShell für Microsoft 365 verwenden, um Microsoft 365 zu verwalten und Ihre tägliche Arbeit zu optimieren. Verwenden Sie die folgenden Informationen, um zu erfahren, warum PowerShell für Microsoft 365 für die Verwaltung von Microsoft 365 wichtig ist, wie Sie eine Verbindung mit Ihrem Microsoft 365-Abonnement herstellen und Berichte erstellen und wo Sie weitere Informationen aus der Microsoft 365-Community erhalten.
  
Wählen Sie aus den folgenden Themen aus:
  
- [**Warum Sie PowerShell für Microsoft 365 verwenden müssen**](why-you-need-to-use-microsoft-365-powershell.md)
    
    Beginnen Sie hier, wenn Sie powerShell for Microsoft 365. Erfahren Sie, warum Sie PowerShell für Microsoft 365.
    
- [**Verbinden mit Microsoft 365 über PowerShell**](connect-to-microsoft-365-powershell.md)
    
    Beginnen Sie hier, um eine Verbindung mit Microsoft 365-Abonnement herzustellen, indem Sie PowerShell für Microsoft 365 verwenden und verwaltungstechnische Aufgaben über die Befehlszeile ausführen.
    
- [**Herstellen einer Verbindung mit allen Microsoft 365-Diensten in einem einzigen PowerShell-Fenster**](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
    
    Sie können Microsoft 365 in separaten Fenstern für Skype for Business Online, SharePoint Online, Microsoft Exchange Online und Microsoft 365 und Lizenzen verwalten. Sie können sie auch alle über ein einzelnes Fenster verwalten. In diesem Artikel wird dies erläutert.
    
- [**Verwenden von PowerShell zum Erstellen von Berichten in Microsoft 365**](use-windows-powershell-to-create-reports-in-microsoft-365.md)
    
    Beginnen Sie hier, wenn Sie die PowerShell für Microsoft 365 installiert haben und mehr über die Verwendung von Automatisierungsbefehlen zum schnellen Erstellen von Berichten erfahren möchten.
    
- [**Cmdlet-Referenzen für Microsoft 365-Dienste**](cmdlet-references-for-microsoft-365-services.md)
    
    Erfahren Sie mehr über die Cmdlets für die PowerShell für Microsoft 365 Module.
    
- [**Microsoft 365 Community-Ressourcen für PowerShell**](microsoft-365-powershell-community-resources.md)
    
    Beginnen Sie hier, um eine Verbindung mit der PowerShell-Community herzustellen und weitere Informationen zur Verwendung von PowerShell für Microsoft 365.
