# <a name="modern-desktop-deployment-center"></a>Bereitstellungscenter für modernen Desktop

<img src="media/getting-started-media/getting-started-media-1.png" alt="Desktop Deployment Wheel" height="450" width="802" align="middle" style="background-color: #fff;" />

Führen Sie die folgenden Schritte aus, um Ihre umfangreiche Bereitstellung von Windows 10 und Office 365 ProPlus zu planen und durchzuführen. Die einzelnen Schritte sind Teil des gesamten Planungs- und Bereitstellungsprozesses; sie werden in der Regel parallel in einer stufenweisen Bereitstellung ausgeführt. Laden Sie das kostenlose [Lab-Kit für moderne Desktop-Bereitstellung und -Verwaltung](https://aka.ms/howtoshiftlabs) für eine praktische Schulung zu den Tools herunter, die im Bereitstellungsprozess hervorgehoben werden.
