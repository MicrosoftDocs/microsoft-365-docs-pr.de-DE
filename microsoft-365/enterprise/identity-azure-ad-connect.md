<span data-ttu-id="6dc33-p102">Die erste Entscheidung in Ihrer Identitätshybridlösung ist Ihre Authentifizierungsanforderung. - <span data-ttu-id="6dc33-p103">Bei der **verwalteten Authentifizierung** verarbeitet Azure AD den Authentifizierungsprozess für die Benutzeranmeldung. There are two methods for managed authentication:</span></span> 
    - <span data-ttu-id="6dc33-116">**Kennworthashsynchronisierung (Password Hash Sync, PHS)** [Empfohlen und für einige Premiumfunktionen erforderlich].</span><span class="sxs-lookup"><span data-stu-id="6dc33-116">**Password Hash Sync (PHS)** [Recommended and required for some premium features].</span></span> <span data-ttu-id="6dc33-117">Dies ist die einfachste Methode, um die Authentifizierung für lokale Verzeichnisobjekte in Azure AD zu aktivieren.</span><span class="sxs-lookup"><span data-stu-id="6dc33-117">This is the simplest way to enable authentication for on-premises directory objects in Azure AD.</span></span> <span data-ttu-id="6dc33-118">Azure AD Connect extrahiert das Kennworthash aus AD DS, führt eine zusätzliche Sicherheitsverarbeitung für das Kennwort durch und speichert es in Azure AD.</span><span class="sxs-lookup"><span data-stu-id="6dc33-118">Azure AD Connect extracts the hashed password from AD DS, does extra security processing on the password, and saves it in Azure AD.</span></span> <span data-ttu-id="6dc33-119">Weitere Informationen finden Sie unter [Implementieren der Kennworthashsynchronisierung mit der Azure AD Connect-Synchronisierung](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnectsync-implement-password-synchronization).</span><span class="sxs-lookup"><span data-stu-id="6dc33-119">For more information, see [Implement password hash synchronization with Azure AD Connect sync](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnectsync-implement-password-synchronization).</span></span>
