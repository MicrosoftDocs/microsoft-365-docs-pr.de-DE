# <a name="use-powershell-to-create-reports-for-microsoft-365"></a>Verwenden von PowerShell zum Erstellen von Berichten für Microsoft 365

*Dieser Artikel gilt sowohl für Microsoft 365 Enterprise als auch für Office 365 Enterprise.*

Viele verschiedene Berichte sind im Microsoft 365 Admin Center verfügbar. Aber diese Berichte liefern nur so viele Informationen, und manchmal brauchen Sie mehr. Dann benötigen Sie PowerShell für Microsoft 365.
  
In diesen Artikeln wird beschrieben, wie Sie PowerShell für Microsoft 365 verwenden, um Informationen von Ihrem Microsoft 365-Mandanten abzubekommen:
  
- Erste Schritte mit der Berichterstellung mit PowerShell für Microsoft 365:
    
  - [Warum Sie PowerShell für Microsoft 365 verwenden müssen](./why-you-need-to-use-microsoft-365-powershell.md)
    
    
- Berichte für Benutzerkonten und Lizenzen:
    
  - [Anzeigen Microsoft 365 Lizenzen und Dienste mit PowerShell](view-licenses-and-services-with-microsoft-365-powershell.md)
    
  - [Microsoft 365 lizenzierte und nicht lizenzierte Benutzer mit PowerShell anzeigen](view-licensed-and-unlicensed-users-with-microsoft-365-powershell.md)
    
  - [Anzeigen Microsoft 365 Kontolizenz- und Servicedetails mit PowerShell](view-account-license-and-service-details-with-microsoft-365-powershell.md)
    
  - [Anzeigen Microsoft 365 Benutzerkonten mit PowerShell](view-user-accounts-with-microsoft-365-powershell.md)
    
- Berichte für SharePoint Online:
    
  - [Erste Schritte mit der SharePoint Online-Verwaltungsshell](/powershell/sharepoint/sharepoint-online/connect-sharepoint-online)
    
  - [Get-SPOSiteGroup - Ruft alle Gruppen in einer angegebenen Websitesammlung ab](/powershell/module/sharepoint-online/get-spositegroup)
    
- Berichte für Exchange Online:
    
  - [Verwenden Exchange Online PowerShell zum Anzeigen von Postfach](/exchange/recipients-in-exchange-online/manage-user-mailboxes/use-powershell-to-display-mailbox-information)
