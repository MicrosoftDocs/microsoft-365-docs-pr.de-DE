# <a name="plan-your-upgrade-from-office-2007-or-office-2010-servers-and-clients"></a>Planen des Upgrades von Office 2007- oder Office 2010-Servern und -Clients

*Dieser Artikel gilt sowohl für Microsoft 365 for Enterprise als auch für Office 365 for Enterprise.*

Wenn Ihre Organisation alte Versionen von Office-Produkten und -Servern verwendet, ist es jetzt ein guter Zeitpunkt, mit der Planung Ihres Upgrades zu beginnen. Office 2007-Produkte und -Dienste haben das [Ende des Supports](upgrade-from-office-2007-servers-and-products.md) erreicht. Für Office 2010-Produkte und -Dienste:

- Office 2010 und Exchange 2010 haben ihr Ende der Unterstützung am *13. Oktober 2020 erreicht.* 
- Der Support für SharePoint 2010 und Project Server 2010 endet am *13. April 2021*. 

Weitere Informationen finden Sie unter [Upgrade von Office 2010-Servern und -Clients](upgrade-from-office-2010-servers-and-products.md).

Verwenden Sie die Ressourcen in diesem Artikel, um mit dem Upgrade zu beginnen.

## <a name="what-is-microsoft-365"></a>Was ist Microsoft 365?

[Microsoft 365](https://www.microsoft.com/microsoft-365) ist eine Kombination aus innovativer Office-Apps, intelligenten Clouddiensten und erstklassiger Sicherheit, die Ihnen dabei hilft, mehr zu erreichen.

Microsoft 365 enthält die Lizenzen und Funktionen, um sicherzustellen, dass Ihre Organisation auf dem neuesten Windows-Betriebssystem arbeitet. Außerdem wird sichergestellt, dass Ihre Windows-, iOS- und Android-Geräte mit Richtlinien registriert und gesichert werden, die Authentifizierung und Datenschutz erfordern. Darüber hinaus werden Windows 10 und Ihre Microsoft 365 Apps for Enterprise (zuvor Office 365 ProPlus genannt) Clientsoftware kontinuierlich aktualisiert, um die neuesten Features und Sicherheitsupdates zu enthalten.
  
Microsoft 365 ist die Möglichkeit, Ihr Unternehmen mit ständig verbesserten Geräten und Produktivitätserfahrungen, die durch die Microsoft Cloud aktiviert und gesichert sind, digital zu transformieren.

## <a name="what-happens-if-i-dont-upgrade"></a>Was geschieht, wenn ich kein Upgrade durchführe?

Sie müssen nicht unbedingt zum jetzigen Zeitpunkt ein Upgrade durchführen. Ihre lokalen Server und Anwendungen können auch weiterhin ausgeführt werden. Wenn Sie jedoch keine Sicherheitsupdates oder Supportoptionen mehr erhalten, kann Ihre Organisation anfällig für Sicherheitsverletzungen sein. Es wird dringend empfohlen, das Upgrade bald zu planen. Sie können ein Upgrade auf Microsoft 365 oder auf neuere Versionen Ihrer lokalen Server und Anwendungen durchführen.

## <a name="what-upgrade-options-are-available"></a>Welche Upgradeoptionen sind verfügbar?      

Organisationen sollten mehrere Upgradeoptionen in Betracht ziehen:

- **Upgrade Ihrer lokalen Server und Anwendungen**. Wenn Sie lokale Office-Produkte und Serveranwendungen verwenden, lesen Sie die folgenden Planungsinhalte:<br/> 

  |Office 2007-Produkte und -Dienste|Office 2010-Produkte und -Dienste|
  |---|---|
  |[Office 2007](/DeployOffice/office-2007-end-support-roadmap) (Desktop)|[Office 2010](/DeployOffice/office-2010-end-support-roadmap) (Desktop)|
  |[Exchange 2007](exchange-2007-end-of-support.md)|[Exchange 2010](exchange-2010-end-of-support.md)|
  |[SharePoint 2007](sharepoint-2007-end-of-support.md)|[SharePoint 2010](upgrade-from-sharepoint-2010.md)|
  |[Office Communications Server](/skypeforbusiness/plan-your-deployment/upgrade)|[Lync Server 2010](/skypeforbusiness/plan-your-deployment/upgrade)|
  |[Project Server 2007](project-server-2007-end-of-support.md)|[Project Server 2010](project-server-2010-end-of-support.md)|
  |[PerformancePoint Server 2007](pps-2007-end-of-support.md)||
 
- **Implementieren Sie eine Hybridlösung mit Microsoft 365 oder Office 365.** Eine Hybridlösung verwendet Ihre lokalen Server und Anwendungen sowie deren Cloudäquivalente. Wenn Sie phasenweise in die Cloud umstiegen oder einige Server und Anwendungen lokal halten müssen, ist eine Hybridlösung möglicherweise für Ihre Organisation geeignet. Weitere Informationen finden Sie unter [Microsoft Cloud Architecture Models](../solutions/cloud-architecture-models.md). 
    
- **Wechseln sie mit Microsoft 365 oder Office 365 in die Cloud.** Für viele Kunden ist der Wechsel in die Cloud eine effiziente und kostengünstige Lösung. Ein vollständiger Wechsel in die Cloud vereinfacht das Setup und die laufende Verwaltung. Diese Option bietet nahtlos alle neuesten Features und Sicherheitsupdates. Weitere Informationen finden Sie im Abschnitt [Was ist Microsoft 365?](#what-is-microsoft-365) in diesem Artikel.
    
## <a name="can-i-get-help-for-my-organization"></a>Kann ich Hilfe für meine Organisation erhalten?

Wenn Sie Hilfe bei der Planung Ihres Upgrades benötigen, sollten Sie eine oder mehrere der folgenden Optionen in Betracht ziehen:

- Arbeiten Sie mit einem Partner oder Spezialisten für Volumenlizenzen zusammen. [Suchen Sie Ihren Microsoft 365-Partner oder -Händler](https://support.office.com/article/b6c18a9b-2aed-4c84-9d75-af709160258c.aspx). 

- Wenn Ihre Organisation eine qualifizierte Anzahl von Microsoft 365-Lizenzen kauft, kann Ihnen unser FastTrack-Team beim Setup helfen. Weitere Informationen finden Sie unter [FastTrack für Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365).

- Wenn Sie Teil einer kleinen Organisation sind oder das Office-Upgrade Ihrer Organisation lieber selbst durchführen möchten, lesen Sie [Upgrade Ihrer Microsoft 365 Business-Benutzer](/office365/admin/setup/upgrade-users-to-latest-office-client)auf den neuesten Office-Client. 
  
## <a name="im-a-home-user-what-do-i-do"></a>Ich bin ein privater Benutzer. Wie gehe ich jetzt vor?

Wenn Sie Office 2007 oder Office 2010 zu Hause verwenden, sollten Sie die folgenden Upgradeoptionen in Betracht ziehen:

- **Verwenden Sie Office kostenlos in einem Browser.** Erstellen, Anzeigen und Bearbeiten von Office-Dateien in Ihrem Browser. Erhalten Sie Zugriff auf diese Dateien von einem beliebigen Gerät mit Internetzugang. 

  [Office im Web](https://products.office.com/office-online/documents-spreadsheets-presentations-office-online) umfasst [Word für das Web,](https://go.microsoft.com/fwlink/p/?linkid=746664)Excel für [das Web,](https://go.microsoft.com/fwlink/p/?linkid=746665)PowerPoint für [das Web,](https://go.microsoft.com/fwlink/p/?linkid=746666) [OneNote für das Web,](https://go.microsoft.com/fwlink/p/?linkid=746674) [Sway,](https://go.microsoft.com/fwlink/p/?linkid=746675) [E-Mail,](https://go.microsoft.com/fwlink/p/?linkid=746676) [Kalender](https://go.microsoft.com/fwlink/p/?linkid=746678)und [OneDrive.](https://go.microsoft.com/fwlink/p/?linkid=746679) Um zu beginnen, besuchen Sie [Office.com](https://office.com) und melden Sie sich mit Ihrem [Microsoft-Konto an.](https://account.microsoft.com/account) Wenn Sie über kein Microsoft-Konto verfügen, können Sie eins unter [Office.com.](https://office.com)

- **Testen Sie Microsoft 365 Family.** Starten Sie eine Testversion der [Microsoft 365-Familie,](https://www.microsoft.com/microsoft-365/p/microsoft-365-family/cfq7ttc0k5dm?rtc=2&activetab=pivot:overviewtab) um zu sehen, wie sie für Sie funktioniert. Mit der Microsoft 365-Familie profitieren Sie von Cloudspeicher mit OneDrive.

  Die Unterstützung für Windows 7 [wurde am 14. Januar 2020 beendet.](https://www.microsoft.com/microsoft-365/windows/end-of-windows-7-support) Die Versionen von Word, Excel, PowerPoint, Outlook, Publisher und Access, die mit Office 365 Home oder Office 365 Personal bereitgestellt werden und auf Windows 7-Geräten ausgeführt werden, erhalten Sicherheitsupdates, aber keine Funktionsupdates. Wenn Sie weiterhin Featureupdates für diese Anwendungen erhalten möchten, [aktualisieren Sie Ihre Windows 7-Geräte auf Windows 10](https://support.microsoft.com/help/12435/windows-10-upgrade-faq).
    
- **Kaufen Sie Office Home &amp; Student.** Wenn Sie diese Option auswählen, erwerben Sie einmal und installieren Dann Office auf Ihrem Windows-PC oder Mac. Bei diesem Kauf handelt es sich nicht um ein Abonnement. Es ist eine einmal unbefristete Lizenz für einen Computer. Zeigen Sie die [Anforderungen](https://office.com/systemrequirements) an, und wählen Sie dann eine Version aus.

  - Wird auf Ihrem Windows-PC Windows 10 ausgeführt, erwägen Sie den Erwerb von [Office Home & Student 2019](https://www.microsoft.com/p/office-home-student-2019/cfq7ttc0k7c8).

  - Wenn Auf Ihrem Windows-PC Windows 7, 8 oder 8.1 ausgeführt wird und Sie jetzt kein Upgrade auf Windows 10 durchführen, sollten Sie Office Home & Student 2016 oder eine andere Edition von Microsoft Office. Sie können es von einem autorisierten Händler erhalten.
     
    Die Unterstützung für Windows 7 [wurde am 14. Januar 2020 beendet.](https://www.microsoft.com/microsoft-365/windows/end-of-windows-7-support) Microsoft stellt keine Sicherheitsupdates mehr dafür zur Verfügung. Aktualisieren Sie Ihre Windows 7-Geräte auf Windows 10, um weiter Sicherheits- und Featureupdates sowie durchgängigen Support zu erhalten.

Wenn Sie jetzt kein Upgrade durchführen möchten, werden Ihre Office-Apps weiterhin gemäß den [Zeitachsen ausgeführt.](https://support.microsoft.com/lifecycle/search/13615) Um jedoch Sicherheitsupdates oder neue und verbesserte Features zu erhalten, müssen Sie ein Upgrade durchführen.
