# <a name="migrate-your-organization-data-to-microsoft-365-enterprise"></a>Migrieren Sie Ihre Organisationsdaten zu Microsoft 365 Enterprise

## <a name="ready-to-migrate-your-data-to-microsoft-365"></a>Sind Sie bereit, Ihre Daten zu Microsoft 365?

Unsere [grundlegenden Einrichtungsanweisungen](https://support.office.com/article/Set-up-Office-365-for-business-6a3a29a0-e616-4713-99d1-15eda62d04fa) helfen Ihnen, alle Ihre Microsoft 365 und Anwendungen in kürzester Zeit zu nutzen. Dadurch werden die Hybridszenarien und die fortgeschritteneren Methoden für die Migration zu Microsoft 365. 
  
Wenn Sie Hilfe beim einrichten Microsoft 365 möchten, [ist FastTrack](https://fasttrack.microsoft.com/office) die einfachste Möglichkeit zum Bereitstellen von Microsoft 365, Sie können sich auch anmelden und die Setupanleitungen für Microsoft 365 [verwenden.](setup-guides-for-microsoft-365.md)

## <a name="migrate-email-to-microsoft-365"></a>Migrieren von E-Mails zu Microsoft 365
- Migrieren Sie mit Exchange Hybrid mithilfe des [Exchange-Bereitstellungs-Assistenten](https://technet.microsoft.com/exdeploy2013) (Administrator)
- Erfahren Sie mehr über die verschiedenen [Methoden zum Migrieren von E-Mails zu Microsoft 365.](https://support.office.com/article/Ways-to-migrate-multiple-email-accounts-to-Office-365-0a4913fe-60fb-498f-9155-a86516418842)
- Finden Sie alternative Verfahren, [mit denen die Benutzer in Ihrer Organisation ihre E-Mail, ihre Kontakte und Kalender selbst migrieren können](https://support.office.com/article/Migrate-email-and-contacts-to-Office-365-for-business-a3e3bddb-582e-4133-8670-e61b9f58627e).
