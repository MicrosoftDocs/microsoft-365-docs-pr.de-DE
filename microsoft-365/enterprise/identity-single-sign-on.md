# <a name="step-10-simplify-user-sign-in"></a>Schritt 10: Vereinfachen der Benutzeranmeldung

*Dieser Schritt ist optional für Hybridumgebungen und gilt für die Versionen E3 und E5 von Microsoft 365 Enterprise.*

![](./media/deploy-foundation-infrastructure/identity_icon-small.png)

In diesem Schritt richten Sie Azure Active Directory Seamless Single Sign-On (Azure AD Seamless SSO) ein, damit Ihre Benutzer sich bei Diensten anmelden können, die Azure AD-Benutzerkonten verwenden, ohne ihre Kennwörter eingeben zu müssen, und in vielen Fällen ihre Benutzernamen. Damit erhalten die Benutzer einfacheren Zugriff auf cloudbasierte Anwendungen, z. B. Office 365, ohne dass zusätzliche lokale Komponenten wie Identitätsverbundserver erforderlich sind.

Sie konfigurieren Azure AD Seamless SSO mit Azure AD Connect.

Lesen Sie die [Anweisungen zum Konfigurieren von Azure AD Seamless SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start).
