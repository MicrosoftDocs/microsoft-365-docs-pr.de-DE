<span data-ttu-id="08336-146">Sie können beispielsweise die folgenden PowerShell-Befehle ausführen, um alle contoso.local-Suffixe auf contoso.com:</span><span class="sxs-lookup"><span data-stu-id="08336-146">For example, you could run the following PowerShell commands to update all contoso.local suffixes to contoso.com:</span></span>

