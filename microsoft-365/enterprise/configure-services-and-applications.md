# <a name="configure-microsoft-365-enterprise-services-and-applications"></a>Konfigurieren Microsoft 365 Enterprise Dienste und Anwendungen

Unsere [grundlegenden Einrichtungsanweisungen](../admin/setup/setup.md) helfen Ihnen, alle Ihre Microsoft 365 und Anwendungen in kürzester Zeit zu nutzen. Manchmal wird es vorgezogen, Dinge zu konfigurieren, bevor jeder mit der Verwendung beginnt. Beispielsweise, wenn Sie E-Mail-Routing, Dateispeicherung oder Freigaberichtlinien konfigurieren möchten. 
  
Wenn Sie Hilfe beim Einrichten Microsoft 365 möchten, verwenden Sie **[FastTrack](https://www.microsoft.com/fasttrack/microsoft-365)** oder die Setupanleitungen für Microsoft 365 [und Office 365 Dienste.](setup-guides-for-microsoft-365.md)
