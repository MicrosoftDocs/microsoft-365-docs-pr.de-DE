# <a name="setup-guides-for-microsoft-365-and-office-365-services"></a>Einrichtungsanleitung für Microsoft 365- und Office 365-Dienste

Microsoft 365- und Office 365-Setupanleitungen bieten Ihnen maßgeschneiderte Anleitungen und Ressourcen für die Planung und Bereitstellung Ihres Mandanten, Ihrer Apps und Dienste. Diese Anleitungen werden mit den gleichen bewährten Methoden erstellt, die [Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) FastTrack-Onboardingspezialisten in einzelnen Interaktionen gemeinsam verwenden, und sie stehen allen Administratoren im Microsoft 365 Admin Center zur Verfügung. Sie enthalten Informationen zur Produkteinrichtung, zum Aktivieren von Sicherheitsfeatures, zum Bereitstellen von Tools für die Zusammenarbeit und zum Bereitstellen von Skripts zur Beschleunigung erweiterter Bereitstellungen.

## <a name="how-to-access-setup-guides-in-the-microsoft-365-admin-center"></a>Zugreifen auf Setupanleitungen im Microsoft 365 Admin Center

Auf die Einrichtungsanleitungen kann über die [Seite Setupanleitung](https://aka.ms/setupguidance) im Microsoft 365 Admin Center zugegriffen werden. Sie können den Status Ihres Fortschritts nachverfolgen, und Sie haben die Möglichkeit, jederzeit zurückzukehren, um eine Anleitung zu vervollständigen. So erreichen Sie die **Seite "Setupanleitung":**

1. Wechseln Sie [im Microsoft 365 Admin Center](https://admin.microsoft.com/)zur **Startseite.**

2. Suchen Sie die **& Schulungsanleitungen.** 

   ![Schulungs& im Microsoft 365 Admin Center](../media/setup-guides-for-microsoft-365/adminportal-trainingandguides.png)

3. Wählen Sie **Benutzerdefinierte Einrichtungsanleitung aus.**

   ![Screenshot der Seite "Setupanleitung" im Microsoft 365 Admin Center](../media/setup-guides-for-microsoft-365/adminportal-setupguidance.png)

>[!NOTE]
>Mandantenadministratorberechtigungen sind für den Zugriff auf das Microsoft 365 Admin Center erforderlich.

## <a name="how-do-setup-guides-work-in-the-microsoft-365-admin-center"></a>Wie funktionieren Setuphandbücher im Microsoft 365 Admin Center?

In jedem Handbuch finden Sie schrittweise Anweisungen, Ressourcen, Artikel und bei Bedarf Skripts, mit denen Sie Konfigurationsänderungen vornehmen können. Diese Anleitungen bieten Ihnen Auswahlmöglichkeiten, die die spezifischen Anforderungen kleiner und großer Organisationen widerspiegeln. Darüber hinaus enthält die Anleitung Unterstützung für neue und erfahrene Administratoren.

![Beispiel für eine Einrichtungsanleitung](../media/setup-guides-for-microsoft-365/m365-setupguide-example.png)

Sie können die Anleitungen verwenden, um mehr über bestimmte Microsoft 365- und Office 365-Features während der Planungsphase, während der Bereitstellung und des Rollouts zu erfahren oder sie nach Abschluss einer Bereitstellung erneut zu lesen, um eine Einstellung zu ändern.

## <a name="guides-for-initial-setup"></a>Anleitungen für die Ersteinrichtung

### <a name="prepare-your-environment"></a>Vorbereiten der Umgebung

Das [Handbuch Vorbereiten Ihrer Umgebung](https://aka.ms/prepareyourenvironment) hilft Ihnen bei der Vorbereitung der Umgebung Ihrer Organisation auf Microsoft 365- und Office 365-Dienste. Unabhängig von Ihren Zielen müssen Sie Aufgaben ausführen, um eine erfolgreiche Bereitstellung sicherzustellen. Um Fehler bei der Vorbereitung Ihrer Umgebung zu vermeiden, erhalten Sie schrittweise Anweisungen zum Verbinden Ihrer Domäne, zum Hinzufügen von Benutzern, zum Zuweisen von Lizenzen, zum Einrichten von E-Mails mit Exchange Online und zum Installieren oder Bereitstellen von Office-Apps. 

### <a name="email-setup-advisor"></a>E-Mail-Setupratgeber

Der [E-Mail-Setupratgeber](https://aka.ms/office365setup) bietet Ihnen die schrittweise Anleitung zum Konfigurieren von Exchange Online für Ihre Organisation. Dazu gehören das Einrichten neuer E-Mail-Konten, das Migrieren von E-Mails und das Konfigurieren des E-Mail-Schutzes. Für eine erfolgreiche E-Mail-Einrichtung verwenden Sie diesen Ratgeber, und Sie erhalten die empfohlene Migrationsmethode basierend auf dem aktuellen E-Mail-System Ihrer Organisation, der Anzahl der migrierten Postfächer und der Art und Weise, wie Sie Benutzer und deren Zugriff verwalten möchten.

### <a name="migrate-gmail-contacts-and-calendar-items"></a>Migrieren von Gmail-Kontakten und Kalenderelementen

Wenn Sie das Postfach eines Gmail-Benutzers zu Microsoft 365 migrieren, werden E-Mail-Nachrichten migriert, Kontakte und Kalenderelemente jedoch nicht. Der [Gmail-Kontakt- und Kalenderratgeber](https://aka.ms/gmailcontactscalendar) enthält Schritte zum Importieren von Google-Kontakten und Google-Kalenderelementen nach Microsoft 365 mithilfe von Import- und Exportmethoden mit Outlook.com, dem Outlook-Client oder PowerShell.

### <a name="microsoft-365-deployment-advisor"></a>Microsoft 365-Bereitstellungsratgeber

Der [Microsoft 365-Bereitstellungsratgeber](https://aka.ms/microsoft365setupguide) bietet Ihnen Anleitungen zum Einrichten von Produktivitätstools, Sicherheitsrichtlinien und Geräteverwaltungsfunktionen. Mit einem Microsoft 365 Business Premium- oder Microsoft 365 For Enterprise-Abonnement können Sie diesen Ratgeber verwenden, um die Geräte Ihrer Organisation zu einrichten und zu konfigurieren. 

Sie erhalten Anleitungen und Zugriff auf Ressourcen, um Ihre Clouddienste zu aktivieren, Geräte auf die neueste unterstützte Version von Windows 10 zu aktualisieren und Geräte mit Azure Active Directory (Azure AD) an einem zentralen Ort zu verbinden.


### <a name="remote-work-setup-guide"></a>Anleitung zum Einrichten von Remotearbeit

Das [Handbuch zum Einrichten](https://aka.ms/remoteworksetup) von Remotearbeit bietet Organisationen die Tipps und Ressourcen, die sie benötigen, um sicherzustellen, dass Ihre Benutzer remote erfolgreich arbeiten können, Ihre Daten sicher sind und die Anmeldeinformationen der Benutzer geschützt werden. 

Sie erhalten Anleitungen zur Optimierung des Gerätedatenverkehrs von Remotemitarbeitern zu Microsoft 365-Ressourcen in der Cloud und dem Netzwerk Ihrer Organisation, wodurch die Belastung Ihrer REMOTEzugriffs-VPN-Infrastruktur reduziert wird. 

### <a name="windows-virtual-desktop-setup-guide"></a>Windows Virtual Desktop-Setuphandbuch

Windows Virtual Desktop ist ein umfassender Desktop- und App-Virtualisierungsdienst, der in der Cloud ausgeführt wird. Es ist die einzige virtuelle Desktopinfrastruktur (Virtual Desktop Infrastructure, VDI), die vereinfachte Verwaltung, Windows 10 mit mehreren Sitzungen, Optimierungen für Microsoft 365-Apps und Unterstützung für Remotedesktopdienste(RDS)-Umgebungen bietet. Stellen Sie Ihre Windows-Desktops und -Apps in Minuten bereit und skalieren Sie sie in Azure, und erhalten Sie integrierte Sicherheits- und Compliancefeatures. 

Das [Windows Virtual Desktop-Setuphandbuch](https://aka.ms/wvdsetupguide) stellt Administratoren Planungsressourcen und die Voraussetzungen für die Bereitstellung, Anleitungen zum Einrichten und zusätzliche Ressourcen zur Verfügung. 

### <a name="microsoft-edge-setup-guide"></a>Microsoft Edge-Setuphandbuch

Microsoft Edge wurde von Grund auf neu erstellt, um Ihnen erstklassige Kompatibilität und Leistung, die Sicherheit und den Datenschutz, die Sie verdienen, sowie neue Features zu bieten, die Ihnen das Beste aus dem Web bieten.

Das [Microsoft Edge-Setuphandbuch](https://aka.ms/edgeadvisor) hilft Ihnen beim Konfigurieren der Enterprise Site Discovery, um zu sehen, auf welche Websites in Ihrer Organisation möglicherweise der IE-Modus verwendet werden muss, wichtige Sicherheitsfeatures zu überprüfen und zu konfigurieren, Datenschutzrichtlinien und zusätzliche Richtlinien zu konfigurieren, um die Anforderungen Ihrer Organisation zu erfüllen, und den Webzugriff auf Ihren Geräten zu verwalten. Sie können Microsoft Edge auf einzelne Geräte herunterladen, oder wir zeigen Ihnen, wie Sie für mehrere Benutzer in Ihrer Organisation mit Configuration Manager oder Microsoft Intune bereitstellen. 

### <a name="microsoft-search-setup-guide"></a>Microsoft Search-Setuphandbuch

Microsoft Search hilft Ihrer Organisation, zu finden, wofür sie arbeitet. Unabhängig davon, ob sie nach Personen, Dateien, Organigrammen, Websites oder Antworten auf häufige Fragen sucht, kann Ihre Organisation Microsoft Search während des gesamten Arbeitstags verwenden, um Antworten zu erhalten.

Das [Microsoft Search-Setuphandbuch](https://aka.ms/MicrosoftSearchSetup) hilft Ihnen bei der Konfiguration von Microsoft Search, ob Sie es für eine Gruppe von Benutzern pilotprojekten oder für alle Benutzer in Ihrer Organisation ausführen möchten. Sie weisen Suchadministratoren und Sucheditoren zu und passen dann die Sucherfahrung für Ihre Benutzer mit Antworten und zusätzlichen Optionen an, z. B. hinzufügen der Bing-Erweiterung zu Chrome oder Festlegen von Bing als Standardsuchmaschine.

### <a name="intune-configuration-manager-co-management-setup-guide"></a>Einrichtungshandbuch für die Einrichtung der gemeinsamen Verwaltung von Intune Configuration Manager

Verwenden Sie den [Intune Configuration Manager Co-Management](https://aka.ms/comanagementsetup) Setup Guide für vorhandene Configuration Manager-Clientgeräte und neue internetbasierte Geräte, die Ihre Organisation gemeinsam mit Microsoft Intune und Configuration Manager verwalten möchte. Mit diesem Bereitstellungshandbuch für die co-Management-Bereitstellung können Sie Windows 10-Geräte verwalten und den Geräten Ihrer Organisation neue Funktionen hinzufügen und gleichzeitig die Vorteile beider Lösungen nutzen.

## <a name="guides-for-authentication-and-access"></a>Anleitungen für Authentifizierung und Zugriff

### <a name="azure-ad-setup-guide"></a>Azure AD-Einrichtungshandbuch

Das [Azure AD-Setuphandbuch](https://aka.ms/aadpguidance) enthält Informationen, um sicherzustellen, dass Ihre Organisation über eine starke Sicherheitsgruppe verfügt. Sie enthält außerdem wichtige Informationen zum Aktivieren von Self-Service-Kennwortzurücksetzungen, zum bedingten Zugriff und zum integrierten Anmelden von Drittanbietern, einschließlich optionaler erweiterter Identitätsschutz und Automatisierung der Benutzerbereitstellung.

### <a name="sync-users-from-your-orgs-directory"></a>Synchronisieren von Benutzern aus dem Verzeichnis Ihrer Organisation

Dadurch werden Ihre lokalen und cloudbasierten Identitäten für einen einfacheren Zugriff und eine vereinfachte Verwaltung miteinander vereint. Entsperren Sie neue Funktionen, z. B. einmaliges Anmelden, Self-Service-Optionen, automatische Kontobereitstellung, Steuerelemente für bedingten Zugriff und Compliancerichtlinien. Dadurch wird sichergestellt, dass Ihre Benutzer von überall aus Zugriff auf die benötigten Ressourcen haben.

### <a name="plan-your-passwordless-deployment"></a>Planen der kennwortlosen Bereitstellung

- Der Microsoft Authenticator-App
- Sicherheitsschlüssel 

Verwenden Sie [den Assistenten zum Planen der kennwortlosen](https://aka.ms/passwordlesssetup) Bereitstellung, um die besten kennwortlosen Authentifizierungsmethoden zu ermitteln und Anleitungen zur Bereitstellung zu erhalten. 

### <a name="plan-your-self-service-password-reset-sspr-deployment"></a>Planen der Self-Service Password Reset (SSPR)-Bereitstellung

Geben Sie Benutzern die Möglichkeit, ihr Kennwort unabhängig zu ändern oder zurückzusetzen, wenn ihr Konto gesperrt ist oder sie ihr Kennwort vergessen, ohne sich an einen Helpdesktechniker wenden zu müssen. 

Verwenden Sie den Bereitstellungs-Assistenten für die Self-Service-Kennwortzurücksetzung, um relevante Artikel und Anweisungen zum Konfigurieren der entsprechenden [Azure-Portaloptionen](https://aka.ms/SSPRSetupGuide) zu erhalten, die Ihnen bei der Bereitstellung von SSPR in Ihrer Umgebung helfen.

### <a name="active-directory-federation-services-ad-fs-deployment-advisor"></a>Bereitstellungsratgeber für Active Directory Federation Services (AD FS)

Der AD FS-Bereitstellungsratgeber bietet Ihnen schrittweise Anleitungen zum Bereitstellen einer lokalen AD FS-Infrastruktur, die Benutzer für Microsoft 365- und Office 365-Dienste authentifiziert. [](https://aka.ms/adfsguidance) Mit diesem Handbuch kann Ihre Organisation ad FS-Komponenten und -Anforderungen überprüfen, für die Bereitstellung erforderliche SSL-Zertifikate erwerben und installieren und einen erforderlichen Webanwendungsproxyserver installieren.

Zum Verwalten der Daten Ihrer Organisation auf gemeinsam genutzten und persönlichen Geräten können Sie die Mobile Application Management (MAM)-Features von Intune verwenden. 

Mit dem [Microsoft Intune-Setuphandbuch](https://aka.ms/intunesetupguide)richten Sie Geräte- und App-Compliancerichtlinien ein, weisen App-Schutzrichtlinien zu und überwachen den Geräte- und App-Schutzstatus.

Das Handbuch enthält Informationen und Ressourcen zum Einrichten und Überprüfen der Lizenzierung Ihres Benutzers, zum Zuweisen von Berechtigungen im Microsoft 365 Admin Center und zum Konfigurieren der Anti-Malware- und Spamrichtlinien Ihrer Organisation im Security & Compliance Center. 

### <a name="microsoft-defender-for-office-365-advisor"></a>Microsoft Defender for Office 365 Advisor

In diesem Handbuch erhalten Sie die Ressourcen und Informationen, mit deren Hilfe Sie den Defender for Office 365-Plan vorbereiten und identifizieren können, der den Anforderungen Ihrer Organisation entspricht.

### <a name="integrate-a-third-party-cloud-app-with-azure-ad"></a>Integrieren einer Cloud-App eines Drittanbieters in Azure AD

Improve the user experience and provide an additional layer of security by integrating your third-party app with ‎Azure Active Directory (Azure AD). Mit dieser End-to-End-Erfahrung können Sie die meisten Konfigurationen direkt über diesen Assistenten ausführen. Gegebenenfalls leiten wir Sie zur entsprechenden Konfigurationsseite um.

Verwenden Sie [den Assistenten zum Integrieren von Drittanbieter-Cloud-Apps,](https://admin.microsoft.com/Adminportal/Home?#/azureadappintegration) um eine Cloud-App eines Drittanbieters in Azure AD zu integrieren.
