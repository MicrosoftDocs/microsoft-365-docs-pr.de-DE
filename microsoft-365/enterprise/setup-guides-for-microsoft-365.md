# <a name="setup-guides-for-microsoft-365-and-office-365-services"></a>Einrichtungsanleitung für Microsoft 365- und Office 365-Dienste

Microsoft 365 und Office 365-Setupanleitungen bieten Ihnen maßgeschneiderte Anleitungen und Ressourcen für die Planung und Bereitstellung Ihres Mandanten, Ihrer Apps und Dienste. Diese Anleitungen werden mit denselben bewährten Methoden erstellt, die Microsoft 365 FastTrack-Onboarding-Spezialisten in einzelnen Interaktionen gemeinsam verwenden, und sie stehen allen Administratoren im Microsoft 365 Admin Center zur Verfügung. [](https://www.microsoft.com/fasttrack/microsoft-365) Sie enthalten Informationen zur Produkteinrichtung, zum Aktivieren von Sicherheitsfeatures, zum Bereitstellen von Tools für die Zusammenarbeit und zum Bereitstellen von Skripts zur Beschleunigung erweiterter Bereitstellungen.

## <a name="how-to-access-setup-guides-in-the-microsoft-365-admin-center"></a>Zugreifen auf Setupanleitungen im Microsoft 365 Admin Center

Der Zugriff auf die Setupanleitungen finden Sie auf der [Seite Setupanleitung](https://aka.ms/setupguidance) im Microsoft 365 Admin Center. Sie können den Status Ihres Fortschritts nachverfolgen, und Sie haben die Möglichkeit, jederzeit zurückzukehren, um eine Anleitung zu vervollständigen. So erreichen Sie die **Seite "Setupanleitung":**

1. Wechseln Sie [Microsoft 365 Admin Center](https://admin.microsoft.com/)zur **Startseite.**

2. Suchen Sie die **& Schulungsanleitungen.** 

   ![Schulungs& im Microsoft 365 Admin Center](../media/setup-guides-for-microsoft-365/adminportal-trainingandguides.png)

3. Wählen Sie **Benutzerdefinierte Einrichtungsanleitung aus.**

   ![Screenshot der Seite "Setupanleitung" im Microsoft 365 Admin Center](../media/setup-guides-for-microsoft-365/adminportal-setupguidance.png)

>[!NOTE]
>Mandantenadministratorberechtigungen sind für den Zugriff auf das Microsoft 365 Admin Center erforderlich.

## <a name="how-do-setup-guides-work-in-the-microsoft-365-admin-center"></a>Wie funktionieren Setuphandbücher im Microsoft 365 Admin Center?

In jedem Handbuch finden Sie schrittweise Anweisungen, Ressourcen, Artikel und bei Bedarf Skripts, mit denen Sie Konfigurationsänderungen vornehmen können. Diese Anleitungen bieten Ihnen Auswahlmöglichkeiten, die die spezifischen Anforderungen kleiner und großer Organisationen widerspiegeln. Darüber hinaus enthält die Anleitung Unterstützung für neue und erfahrene Administratoren.

![Beispiel für eine Einrichtungsanleitung](../media/setup-guides-for-microsoft-365/m365-setupguide-example.png)

Sie können die Anleitungen verwenden, um während der Planungsphase, während der Bereitstellung und des Rollouts mehr über bestimmte Microsoft 365- und Office 365-Features zu erfahren oder sie nach Abschluss einer Bereitstellung erneut zu lesen, um eine Einstellung zu ändern.

## <a name="guides-for-initial-setup"></a>Anleitungen für die Ersteinrichtung

### <a name="prepare-your-environment"></a>Vorbereiten der Umgebung

Das [Handbuch Vorbereiten Ihrer Umgebung](https://aka.ms/prepareyourenvironment) hilft Ihnen bei der Vorbereitung der Umgebung Ihrer Organisation auf Microsoft 365 und Office 365 Dienste. Unabhängig von Ihren Zielen müssen Sie Aufgaben ausführen, um eine erfolgreiche Bereitstellung sicherzustellen. Um Fehler bei der Vorbereitung Ihrer Umgebung zu vermeiden, erhalten Sie schrittweise Anweisungen zum Herstellen einer Verbindung mit Ihrer Domäne, zum Hinzufügen von Benutzern, zum Zuweisen von Lizenzen, zum Einrichten von E-Mails mit Exchange Online und zum Installieren oder Bereitstellen von Office Apps. 

### <a name="email-setup-advisor"></a>E-Mail-Setupratgeber

Der [E-Mail-Setupratgeber](https://aka.ms/office365setup) bietet Ihnen die schritt-für-Schritt-Anleitung zum Konfigurieren Exchange Online Ihrer Organisation. Dazu gehören das Einrichten neuer E-Mail-Konten, das Migrieren von E-Mails und das Konfigurieren des E-Mail-Schutzes. Für eine erfolgreiche E-Mail-Einrichtung verwenden Sie diesen Ratgeber, und Sie erhalten die empfohlene Migrationsmethode basierend auf dem aktuellen E-Mail-System Ihrer Organisation, der Anzahl der migrierten Postfächer und der Art und Weise, wie Sie Benutzer und deren Zugriff verwalten möchten.

### <a name="migrate-gmail-contacts-and-calendar-items"></a>Migrieren von Gmail-Kontakten und Kalenderelementen

Wenn Sie das Postfach eines Gmail-Benutzers zu Microsoft 365, werden E-Mail-Nachrichten migriert, Kontakte und Kalenderelemente jedoch nicht. Der Gmail-Kontakt- und [Kalenderratgeber](https://aka.ms/gmailcontactscalendar) enthält Schritte zum Importieren von Google-Kontakten und Google-Kalenderelementen Microsoft 365 verwendung von Import- und Exportmethoden mit Outlook.com, dem Outlook-Client oder PowerShell.

### <a name="microsoft-365-deployment-advisor"></a>Microsoft 365-Bereitstellungsratgeber

Der [Microsoft 365 bietet](https://aka.ms/microsoft365setupguide) Ihnen Anleitungen zum Einrichten von Produktivitätstools, Sicherheitsrichtlinien und Geräteverwaltungsfunktionen. Mit einem Microsoft 365 Business Premium oder Microsoft 365 enterprise-Abonnement können Sie diesen Ratgeber verwenden, um die Geräte Ihrer Organisation zu einrichten und zu konfigurieren. 

Sie erhalten Anleitungen und Zugriff auf Ressourcen, um Ihre Clouddienste zu aktivieren, Geräte auf die neueste unterstützte Version von Windows 10 zu aktualisieren und Geräte an Azure Active Directory (Azure AD) an einem zentralen Ort zu verbinden.


### <a name="remote-work-setup-guide"></a>Anleitung zum Einrichten von Remotearbeit

Das [Handbuch zum Einrichten](https://aka.ms/remoteworksetup) von Remotearbeit bietet Organisationen die Tipps und Ressourcen, die sie benötigen, um sicherzustellen, dass Ihre Benutzer remote erfolgreich arbeiten können, Ihre Daten sicher sind und die Anmeldeinformationen der Benutzer geschützt werden. 

Sie erhalten Anleitungen zur Optimierung des Gerätedatenverkehrs von Remotemitarbeitern für Microsoft 365-Ressourcen in der Cloud und im Netzwerk Ihrer Organisation, wodurch die Belastung Ihrer Remotezugriffs-VPN-Infrastruktur reduziert wird. 

### <a name="windows-virtual-desktop-setup-guide"></a>Windows Anleitung zum Einrichten von virtuellen Desktops

Windows Virtual Desktop ist ein umfassender Desktop- und App-Virtualisierungsdienst, der in der Cloud ausgeführt wird. Es ist die einzige virtuelle Desktopinfrastruktur (Virtual Desktop Infrastructure, VDI), die eine vereinfachte Verwaltung, mehrstufige Windows 10, Optimierungen für Microsoft 365 Apps und Unterstützung für Remotedesktopdiensteumgebungen (RDS) bietet. Stellen Sie Ihre Windows und Apps in Minuten auf Azure bereit und skalieren Sie sie, und erhalten Sie integrierte Sicherheits- und Compliancefeatures. 

Das [Windows Virtual Desktop stellt](https://aka.ms/wvdsetupguide) Administratoren Planungsressourcen und die Voraussetzungen für die Bereitstellung, Anleitungen zum Einrichten und zusätzliche Ressourcen zur Verfügung. 

### <a name="microsoft-edge-setup-guide"></a>Microsoft Edge-Setupanleitung

Microsoft Edge wurde von Grund auf neu erstellt, um Ihnen erstklassige Kompatibilität und Leistung, die Sicherheit und den Datenschutz, die Sie verdienen, sowie neue Features zu bieten, die Ihnen das Beste aus dem Web bieten.

Mithilfe des Microsoft Edge-Setuphandbuchs können Sie Enterprise Site Discovery konfigurieren, um zu sehen, auf welche Websites in Ihrer Organisation möglicherweise der IE-Modus verwendet werden muss, wichtige Sicherheitsfeatures überprüfen und konfigurieren, Datenschutzrichtlinien und zusätzliche Richtlinien konfigurieren, um die Anforderungen Ihrer Organisation zu erfüllen, und den Webzugriff auf Ihren Geräten verwalten. [](https://aka.ms/edgeadvisor) Sie können Microsoft Edge auf einzelne Geräte herunterladen, oder wir zeigen Ihnen, wie Sie sie für mehrere Benutzer in Ihrer Organisation mit Configuration Manager oder Microsoft Intune. 

### <a name="microsoft-search-setup-guide"></a>Microsoft Search-Setuphandbuch

Microsoft Search hilft Ihrer Organisation, zu finden, wofür sie arbeitet. Unabhängig davon, ob sie nach Personen, Dateien, Organigrammen, Websites oder Antworten auf häufige Fragen sucht, kann Ihre Organisation Microsoft Search während des gesamten Arbeitstags verwenden, um Antworten zu erhalten.

Das [Microsoft Search-Setuphandbuch](https://aka.ms/MicrosoftSearchSetup) hilft Ihnen bei der Konfiguration von Microsoft Search, ob Sie es für eine Gruppe von Benutzern pilotprojekten oder für alle Benutzer in Ihrer Organisation ausführen möchten. Sie weisen Suchadministratoren und Such-Editoren zu und passen dann die Sucherfahrung für Ihre Benutzer mit Antworten und zusätzlichen Optionen an, z. B. das Hinzufügen der Bing-Erweiterung zu Chrome oder das Festlegen von Bing als Standardsuchmaschine.

### <a name="intune-configuration-manager-co-management-setup-guide"></a>Einrichtungshandbuch für die Einrichtung der gemeinsamen Verwaltung von Intune Configuration Manager

Verwenden Sie das [Intune Configuration Manager Co-Management](https://aka.ms/comanagementsetup) Setup Guide für vorhandene Configuration Manager-Clientgeräte und neue internetbasierte Geräte, die Ihre Organisation gemeinsam mit Microsoft Intune und Configuration Manager verwalten möchte. Mit diesem Bereitstellungshandbuch zur gemeinsamen Verwaltung können Sie Windows 10 Geräte verwalten und den Geräten Ihrer Organisation neue Funktionen hinzufügen und gleichzeitig die Vorteile beider Lösungen nutzen.

## <a name="guides-for-authentication-and-access"></a>Anleitungen für Authentifizierung und Zugriff

### <a name="azure-ad-setup-guide"></a>Azure AD-Einrichtungshandbuch

Das [Azure AD-Setuphandbuch](https://aka.ms/aadpguidance) enthält Informationen, um sicherzustellen, dass Ihre Organisation über eine starke Sicherheitsgruppe verfügt. In diesem Handbuch richten Sie erste Features ein, z. B. azure role-based access control (Azure RBAC) für Administratoren, Azure AD Verbinden für Ihr lokales Verzeichnis und Azure AD Verbinden Health, damit Sie die Integrität Ihrer Hybrididentität während automatisierter Synchronisierungen überwachen können. 

Sie enthält außerdem wichtige Informationen zum Aktivieren von Self-Service-Kennwortzurücksetzungen, zum bedingten Zugriff und zum integrierten Anmelden von Drittanbietern, einschließlich optionaler erweiterter Identitätsschutz und Automatisierung der Benutzerbereitstellung.

### <a name="sync-users-from-your-orgs-directory"></a>Synchronisieren von Benutzern aus dem Verzeichnis Ihrer Organisation

Der [Assistent zum Synchronisieren von Benutzern](https://aka.ms/directorysyncsetup) aus dem Verzeichnis ihrer Organisation führt Sie durch das Aktivieren der Verzeichnissynchronisierung. Dadurch werden Ihre lokalen und cloudbasierten Identitäten für einen einfacheren Zugriff und eine vereinfachte Verwaltung miteinander vereint. Entsperren Sie neue Funktionen, z. B. einmaliges Anmelden, Self-Service-Optionen, automatische Kontobereitstellung, Steuerelemente für bedingten Zugriff und Compliancerichtlinien. Dadurch wird sichergestellt, dass Ihre Benutzer von überall aus Zugriff auf die benötigten Ressourcen haben.

### <a name="plan-your-passwordless-deployment"></a>Planen Sie Ihre kennwortlose Bereitstellung

Upgrade auf einen alternativen Anmeldeansatz, mit dem Benutzer sicher auf ihre Geräte zugreifen können, mit einer der folgenden kennwortlosen Authentifizierungsmethoden: 

- Windows Hello for Business
- Der Microsoft Authenticator-App
- Sicherheitsschlüssel 

Verwenden Sie [den Assistenten zum Planen der kennwortlosen](https://aka.ms/passwordlesssetup) Bereitstellung, um die besten kennwortlosen Authentifizierungsmethoden zu ermitteln und Anleitungen zur Bereitstellung zu erhalten. 

### <a name="plan-your-self-service-password-reset-sspr-deployment"></a>Planen der Self-Service Password Reset (SSPR)-Bereitstellung

Geben Sie Benutzern die Möglichkeit, ihr Kennwort unabhängig zu ändern oder zurückzusetzen, wenn ihr Konto gesperrt ist oder sie ihr Kennwort vergessen, ohne sich an einen Helpdesktechniker wenden zu müssen. 

Verwenden Sie den Bereitstellungs-Assistenten für die Self-Service-Kennwortzurücksetzung, um relevante Artikel und Anweisungen zum Konfigurieren der entsprechenden [Azure-Portaloptionen](https://aka.ms/SSPRSetupGuide) zu erhalten, die Ihnen bei der Bereitstellung von SSPR in Ihrer Umgebung helfen.

### <a name="active-directory-federation-services-ad-fs-deployment-advisor"></a>Bereitstellungsratgeber für Active Directory Federation Services (AD FS)

Der AD FS-Bereitstellungsratgeber bietet Ihnen schrittweise Anleitungen zum Bereitstellen einer lokalen AD FS-Infrastruktur, die Benutzer für Microsoft 365 und Office 365 authentifiziert. [](https://aka.ms/adfsguidance) Mit diesem Handbuch kann Ihre Organisation ad FS-Komponenten und -Anforderungen überprüfen, für die Bereitstellung erforderliche SSL-Zertifikate erwerben und installieren und einen erforderlichen Webanwendungsproxyserver installieren. 

## <a name="guides-for-security-and-compliance"></a>Anleitungen für Sicherheit und Compliance

Für die vollständige Kontrolle über Unternehmensgeräte verwenden Sie die Mobile Device Management (MDM)-Features von Intune. Zum Verwalten der Daten Ihrer Organisation auf gemeinsam genutzten und persönlichen Geräten können Sie die Mobile Application Management (MAM)-Features von Intune verwenden. 

Mit dem [Microsoft Intune richten](https://aka.ms/intunesetupguide)Sie Geräte- und App-Kompatibilitätsrichtlinien ein, weisen App-Schutzrichtlinien zu und überwachen den Geräte- und App-Schutzstatus. 

### <a name="microsoft-defender-for-endpoint-advisor"></a>Microsoft Defender for Endpoint Advisor

Erstellen Sie eine fundierte Bewertung der Sicherheitslücke Ihrer Organisation, und entscheiden Sie, welches Bereitstellungspaket und welche Konfigurationsmethoden am besten sind.

### <a name="exchange-online-protection-setup-guide"></a>Exchange Online Protection-Setupanleitung

Verwenden Sie [das Exchange Online Protection-Setuphandbuch](https://aka.ms/EOPguidance) zum Einrichten von EOP, indem Sie auswählen, welches der drei Bereitstellungsszenarien lokale Postfächer, hybride (Mischung aus lokalen und Cloud)-Postfächern oder alle Cloudpostfächer für Ihre Organisation geeignet &mdash; &mdash; ist. Das Handbuch enthält Informationen und Ressourcen zum Einrichten und Überprüfen der Lizenzierung Ihres Benutzers, zum Zuweisen von Berechtigungen im Microsoft 365 Admin Center und zum Konfigurieren der Anti-Malware- und Spamrichtlinien Ihrer Organisation im Security & Compliance Center. 

### <a name="microsoft-defender-for-office-365-advisor"></a>Microsoft Defender for Office 365 Advisor

Der [Microsoft Defender for Office 365-Ratgeber](https://aka.ms/oatpsetup) schützt Ihre Organisation vor böswilligen Bedrohungen, auf die Ihre Umgebung möglicherweise über E-Mail-Nachrichten, Links und Tools für die Zusammenarbeit von Drittanbietern trifft. In diesem Handbuch finden Sie die Ressourcen und Informationen, mit deren Hilfe Sie den Defender for Office 365 vorbereiten und identifizieren können, der den Anforderungen Ihrer Organisation entspricht.

Verwenden Sie einen vierstufigen Lebenszyklusansatz zum Ermitteln, Klassifizieren, Schützen und Überwachen von vertraulichen Informationen. Der [Microsoft Information Protection-Einrichtungsleitfaden](https://aka.ms/mipsetupguide) enthält Anleitungen zum Durchführen jeder dieser Stufen.

In diesem Leitfaden erfahren Sie, wie Sie Bezeichnungen, Bezeichnungsrichtlinien und Aufbewahrungsrichtlinien erstellen, automatisch anwenden oder veröffentlichen, die auf die wiederverwendbaren Inhalts- und Compliancedatensätze Ihrer Organisation angewendet werden. ### <a name="integrate-a-third-party-cloud-app-with-azure-ad"></a>Integrieren einer Cloud-App eines Drittanbieters in Azure AD

Mit dieser End-to-End-Erfahrung können Sie die meisten Konfigurationen direkt über diesen Assistenten ausführen. Gegebenenfalls leiten wir Sie zur entsprechenden Konfigurationsseite um.

Verwenden Sie [den Assistenten zum Integrieren von Drittanbieter-Cloud-Apps,](https://admin.microsoft.com/Adminportal/Home?#/azureadappintegration) um eine Cloud-App eines Drittanbieters in Azure AD zu integrieren.
