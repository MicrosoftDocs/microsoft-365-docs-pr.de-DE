## <a name="why-you-cannot-load-test-sharepoint-online"></a><span data-ttu-id="51519-120">Gründe für das Laden von test SharePoint Online</span><span class="sxs-lookup"><span data-stu-id="51519-120">Why you cannot load test SharePoint Online</span></span>
<span data-ttu-id="51519-121">In lokalen Umgebungen werden Auslastungstests verwendet, um die Skalierungsannahme zu überprüfen und letztendlich den Ausgangspunkt einer Farm zu finden. durch saturieren sie mit Load.</span><span class="sxs-lookup"><span data-stu-id="51519-121">With on-premises environments, load testing is used to validate scale assumption and ultimately find the breaking point of a farm; by saturating it with load.</span></span>
