# <a name="sharepoint-storage-quotas-in-multi-geo-environments"></a><span data-ttu-id="ed03e-103">SharePoint Speicherkontingente in Multi-Geo-Umgebungen</span><span class="sxs-lookup"><span data-stu-id="ed03e-103">SharePoint storage quotas in multi-geo environments</span></span>

<span data-ttu-id="ed03e-104">Alle geografischen Standorte einer Multi-Geo-Umgebung teilen sich das Speicherkontingent des Mandanten.</span><span class="sxs-lookup"><span data-stu-id="ed03e-104">By default, all geo locations of a multi-geo environment share the available tenant storage quota.</span></span>

## <a name="configure-a-storage-quota-for-a-geo-location"></a><span data-ttu-id="ed03e-110">Ein Speicherkontingent für einen Geo-Standort konfigurieren</span><span class="sxs-lookup"><span data-stu-id="ed03e-110">Configure a storage quota for a geo location</span></span>

<span data-ttu-id="ed03e-111">Verwenden Sie das [Microsoft SharePoint Online-Modul](https://www.microsoft.com/download/details.aspx?id=35588 ) und stellen Sie eine Verbindung zum zentralen Standort her, um das Speicherkontingent für einen geographischen Standort zuzuweisen.</span><span class="sxs-lookup"><span data-stu-id="ed03e-111">Use the [Microsoft SharePoint Online Module](https://www.microsoft.com/download/details.aspx?id=35588 ) and connect to the central location to allocate the storage quota for a geo location.</span></span>

`Set-SPOGeoStorageQuota -GeoLocation <geolocationcode> -StorageQuotaMB <value>`

<span data-ttu-id="ed03e-113">Führen Sie Folgendes aus, um das Speicherkontingent des aktuellen geografischen Standorts anzuzeigen:</span><span class="sxs-lookup"><span data-stu-id="ed03e-113">To view Storage Quota for the current geo location, run:</span></span>

`Get-SPOGeoStorageQuota`

`Get-SPOGeoStorageQuota -AllLocations`
