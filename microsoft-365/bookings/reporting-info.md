# <a name="reporting-info-for-bookings"></a>Melden von Informationen für Buchungen

Sie können jetzt eine 120-Tage-Ansicht Ihres Buchungs Kalenders in einer TSV-Datei sehen. TSV-Dateien sind Tab-getrennte Wert Dateien. Sie können eine Datei wie diese mit einem beliebigen Text-Editor oder Tabellen Kalkulationsprogramm wie Excel anzeigen oder bearbeiten.

## <a name="see-120-days-booking-activity"></a>Siehe 120 Tage buchungsaktivität

1. Wählen Sie im Dashboard Buchungskalender die Option Weitere Daten als TSV exportieren aus.

:::image type="content" source="../media/bookings-activity.png" alt-text="Screenshot: 30 Tage buchungsaktivitäten":::

1. Speichern Sie die Datei unter einem neuen Namen, und geben Sie das XLS-oder XLSX-Format an.

1. Öffnen Sie die Datei, um die 120 Tagesansicht Ihres Buchungs Kalenders anzuzeigen.

Die Datei enthält eine Zeile für jedes Ereignis im Buchungskalender für die vorherigen 60 Tage und die bevorstehenden 60 Tage. Jede Zeile enthält das Datum und die Uhrzeitinformationen für das Ereignis, Kundenname, Kunden-e-Mail, Kundentelefonnummer, Kundenadresse, Mitarbeiter zugewiesen, Dienst ausgewählt, Standort und Termin Dauer.
