# <a name="help-your-users-install-office-on-windows-10-devices"></a>Hilfe bei der Installation von Office auf Windows 10-Geräten

[![Hinweis, der Sie darüber informiert, dass sich das Admin Center ändert und Sie unter "aka.ms/aboutM365preview" weitere Details finden.](../media/m365admincenterchanging.png)](/office365/admin/microsoft-365-admin-center-preview)

Sie können Office schnell und einfach über das Microsoft 365 Admin Center auf Windows 10-PCs installieren.
  
Um zu verstehen, wie dies bei zuvor installierten Office-Apps funktioniert, lesen Sie [Vorbereiten der Office-Clientinstallation](prepare-for-office-client-deployment.md), bevor Sie mit diesem Vorgang beginnen.

Sehen Sie sich ein kurzes Video zum Installieren von Office-Apps an.<br><br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/acce002c-0756-4b64-ac5d-2198ee96a9b1] 

Wenn Sie dieses Video hilfreich fanden, sehen Sie sich bitte die [komplette Schulungsserie für kleine Unternehmen und diejenigen, für die Microsoft 365 neu ist](https://support.microsoft.com/office/6ab4bbcd-79cf-4000-a0bd-d42ce4d12816) an.

## <a name="manage-office-deployments"></a>Verwalten von Office-Bereitstellungen

1. Wechseln Sie zum Admin Center unter <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> , und melden Sie sich mit globalen Administratoranmeldeinformationen an. 

2. Wechseln Sie **im** linken Navigationsbereich zu Setup, und scrollen Sie auf der **Seite Setup** zu Apps **und Updates**.
    > [!NOTE]
    > Diese Karte wird möglicherweise nicht angezeigt, wenn alle Benutzer Office-Apps installiert haben.
  
3. Wählen Sie **auf der Hilfe zum Installieren ihrer Office-Apps-Karte** die Option **Ansicht** und dann Erste **Schritte aus.**
    
4. Wählen Sie im Bereich E-Mail-Benutzer einen Link **zum Herunterladen** von Office aus, wählen Sie die Benutzer aus, die Sie per E-Mail senden möchten, und wählen Sie dann Ausgewählte Benutzer per E-Mail **aus.**

   ![Wählen Sie Benutzer aus, die E-Mails mit dem Office-Downloadlink senden möchten.](../media/sendemailtousers.png)
