# <a name="sign-up-for-microsoft-365-business"></a>Registrieren für Microsoft 365 Business

Wenn Sie Partner sind, finden Sie unter [Get microsoft 365 Business from Microsoft Partner Center](get-microsoft-365-business.md#get-microsoft-365-business-from-microsoft-partner-center).

Gehen Sie zu [Discover microsoft 365 Business Advantage](https://www.microsoft.com/microsoft-365/business#pmg-cmp-desktop) , und wählen Sie **Jetzt kaufen** unter Microsoft 365 Business aus.

1. Auf der **Willkommensseite, Let es Get to Know You** Page geben Sie die erforderlichen Informationen ein. Beachten Sie, dass 
 
    -  Das von Ihnen ausgewählte **Land** bestimmt den Dienst, den Sie einholen, und kann später nicht mehr geändert werden.
    - Die **geschäftliche e-Mail-Adresse** ist eine beliebige E-Mail, die von Microsoft zum Senden von Korrespondenz zu Ihrem Abonnement verwendet wird.
    - Der **Name des Unternehmens** wird im Admin Center angezeigt und für alle SharePoint-Website-URLs verwendet.

2. Geben Sie auf der Seite **Benutzer-ID erstellen** den Alias ein, den Sie für die Anmeldung verwenden möchten, einen Domänennamen nach dem @-Zeichen, und wählen Sie dann eine Domäne aus der Dropdownliste aus. Sie können dies verwenden, um eine Domäne zu erwerben und die Erweiterung aus der Dropdownliste auszuwählen, beispielsweise können Sie *Teresa@ContosoSkis.net* eingeben, und der Assistent ermittelt, ob die Domäne verfügbar ist, und fügen die Kosten zu ihren Microsoft 365 Business-Kosten hinzu. Die neue Domäne wird auch für Sie eingerichtet und kann verwendet werden, wenn Sie Sie hier erwerben.

    Wenn Sie über eine vorhandene Domäne verfügen, die Sie verwenden möchten, wählen Sie *. onmicrosoft.com* aus der Dropdownliste aus, und Sie können Sie später im Admin Center einrichten.
    
    ![Screenshot der Seite "Benutzer-ID erstellen".](media/signinuserid.png)

3. Füllen Sie die restlichen Felder aus, und wählen Sie **mein Konto erstellen**aus.
4. Im **Beweis. Sie sind. Not'A. Robot.** Seite geben Sie eine Telefonnummer ein, um einen Code per Text oder Anruf zu empfangen, wählen Sie **Text Me** oder **rufen Sie mich**an, \> **** und geben Sie dann den Verifizierungscode ein.
5. Geben Sie an der **Stelle, an der diese Seite verwendet wird** , Ihre Geschäftsadresse ein. Diese Informationen werden verwendet, um die Steuern und die Verfügbarkeit von Abonnements zu bestimmen. Klicken Sie auf **weiter**, überarbeiten Sie **** Ihre Bestellung, und wählen Sie dann erneut aus.
6. Geben Sie auf der Seite wie soll die Bezahlung erfolgen, die Zahlungsdetails ein, und **legen**Sie dann die Bestellung **an** .
    Sie können dann zum [Admin Center](https://go.microsoft.com/fwlink/p/?linkid=837890) wechseln, um die [Einrichtung](set-up.md)abzuschließen.

    > [!NOTE]
    > Wenn Sie am Ende des Setups den Link zum Admin Center auswählen, werden Sie direkt zur [Einrichten](set-up.md) -Benutzeroberfläche geleitet.
