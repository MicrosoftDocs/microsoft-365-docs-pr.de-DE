# <a name="add-additional-users-to-microsoft-365-business"></a>Hinzufügen weiterer Benutzer zu Microsoft 365 Business

## <a name="add-new-users"></a>Hinzufügen neuer Benutzer

1. Wechseln Sie zum Admin Center unter <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>. 
2. Wählen Sie **** \> im linken Navigationsbereich **aktive**Benutzer aus.
1. Klicken Sie auf der Seite **aktive Benutzer** auf **Benutzer hinzufügen**.
 4. Geben Sie im Bereich **Neuer Benutzer** die erforderlichen Informationen ein. 
  
    Sie können auch unter **Kontaktinformationen** zusätzliche Informationen eingeben, unter der Einstellung **Kennwort** auswählen, wie Sie das Kennwort festlegen möchten, und unter **Rollen** die jeweilige Rolle zuweisen.
      
    ![Enter user information in the New user card](media/f04d39ca-48be-4868-8330-8552a4754c8b.png)
      
    Setzen Sie im Abschnitt "Produktlizenzen" die Einstellung für die **Microsoft 365 Business**-Produktlizenz auf **Ein**.
      
    ![Set the license setting to On position](media/7404f7f7-93bc-44a3-9ffb-4208b5b17402.png)
  
Siehe auch [Benutzer einzeln oder massenweise hinzufügen](https://docs.microsoft.com/office365/admin/add-users/add-users).
