# <a name="pre-requisites-for-protecting-data-on-devices-with-microsoft-365-business"></a>Voraussetzungen zum Schutz von Daten auf Geräten mit Microsoft 365 Business

[] Der erste Schritt beim Einrichten Ihrer Organisation mit Microsoft 365 Business besteht im Sicherstellen, dass Sie die Voraussetzungen erfüllen können.
  
## <a name="pre-requisites"></a>Voraussetzungen

- Auf Windows-Geräten muss Windows 7 Professional, Windows 8 pro oder Windows 8.1 pro installiert sein.
    
    [Aktualisieren von Windows-Geräten auf Windows Pro Creators Update](upgrade-to-windows-pro-creators-update.md)
    
    Wenn Sie Windows 10 Home durchführen, müssen Sie Windows 10 pro **erwerben** . Anweisungen finden Sie unter [Upgrade von Windows 10 Home to Windows 10 pro](https://support.office.com/article/0aee10c1-4d34-43ee-a325-579c6c2df90e?ui=en-US&rs=en-US&ad=US) . 
    
- Entfernen Sie Geräte aus mobilen Managementlösungen (Mobile Iron, AirWatch usw.). Sie registrieren alle Personen in Ihrer Organisation beim mobilen Microsoft 365 Business-Management.
    
- Apple iOS 8.0 oder höher.
    
    Google Android 4.0 oder höher (einschließlich Samsung KNOX Standard 4.0 oder höher). Weitere Informationen hierzu finden Sie unter [Von Intune unterstützte Geräte](https://go.microsoft.com/fwlink/p/?linkid=852307).
    
- Wenn Sie vorhandene Office-Anwendungen auf Benutzercomputern verwenden, lesen Sie [Vorbereiten der Office-Clientinstallation](prepare-for-office-client-deployment.md), damit Sie die Schritte verstehen, die Sie ggf. ausführen müssen, bevor Sie Microsoft 365 Business für die Installation von Office 2016 auf Benutzercomputern einrichten können.
