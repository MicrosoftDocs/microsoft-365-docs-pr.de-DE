# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>Voraussetzungen für den Schutz von Daten auf Geräten mit Microsoft 365 for Business

Der erste Schritt beim Einrichten Ihrer Organisation mit Microsoft 365 for Business besteht darin, sicherzustellen, dass Sie die Voraussetzungen erfüllen können.
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>Voraussetzungen für die Einrichtung Ihrer Organisation mit Microsoft 365 for Business

- Auf Windows-Geräten muss Windows 7 Professional, Windows 8 pro oder Windows 8.1 pro installiert sein.
    
    [Aktualisieren von Windows-Geräten auf Windows Pro Creators Update](upgrade-to-windows-pro-creators-update.md)
    
    Wenn Sie Windows 10 Home durchführen, müssen Sie Windows 10 pro **erwerben** . Anweisungen finden Sie unter [Upgrade von Windows 10 Home to Windows 10 pro](https://support.office.com/article/0aee10c1-4d34-43ee-a325-579c6c2df90e?ui=en-US&rs=en-US&ad=US) . 
    
- Entfernen Sie Geräte aus mobilen Verwaltungslösungen (mobiles Eisen, luftuhren usw.). Sie registrieren alle Personen in Ihrer Organisation in Microsoft 365 for Business Mobile Management.
    
- Apple iOS 8.0 oder höher.
    
    Google Android 4.0 oder höher (einschließlich Samsung KNOX Standard 4.0 oder höher). Weitere Informationen finden Sie unter von [InTune unterstützte Geräte](https://go.microsoft.com/fwlink/p/?linkid=852307).
    
- Wenn Sie bereits über Office-Anwendungen auf Benutzercomputern verfügen, lesen Sie [Vorbereiten der Office-Clientinstallation](prepare-for-office-client-deployment.md) , um die Schritte zu verstehen, die Sie möglicherweise ausführen müssen, bevor Sie Microsoft 365 for Business zum Installieren von Office 2016 auf Benutzercomputern einrichten können.
