![Banner, auf das https://aka.ms/aboutM365previewverwiesen wird.](media/m365admincenterchanging.png)

<span data-ttu-id="c958e-p101">[] Bei einer Zurücksetzung auf Werkseinstellungen wird ein Gerät auf die Einstellungen zurückgesetzt, die es bei seinem Kauf hatte. Alle Apps und Daten auf dem Gerät, die nach dem Kauf installiert wurden, werden entfernt. Sie können mithilfe von Microsoft 365 Business die Zurücksetzung auf Werkseinstellungen für von Ihnen verwaltete Windows 10-Geräte durchführen.</span><span class="sxs-lookup"><span data-stu-id="c958e-p101">A factory reset reverts a device to the original settings it had when the device was purchased. All apps and data on the device that were installed after purchase are removed. You can use Microsoft 365 Business to factory reset Windows 10 devices you manage.</span></span>
  
1. <span data-ttu-id="c958e-108">Wechseln Sie zum Admin Center unter <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>..</span><span class="sxs-lookup"><span data-stu-id="c958e-108">Go to the admin center at <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>..</span></span> 
    
2. <span data-ttu-id="c958e-109">Wählen Sie im linken Navigationsbereich **Geräte** \> **Verwalten**aus.</span><span class="sxs-lookup"><span data-stu-id="c958e-109">In the left nav, choose **Devices** \> **Manage**.</span></span>

3. <span data-ttu-id="c958e-110">Aktivieren Sie auf der Seite **Manage** das Kontrollkästchen neben dem Gerät, aus dem Sie Daten entfernen möchten, und klicken Sie dann im Dropdownmenü **Verwalten** auf **Factory zurücksetzen**.</span><span class="sxs-lookup"><span data-stu-id="c958e-110">On the **Manage** page, check the checkbox next to the device you want to remove data from and then, in the **Manage** drop-down choose **Factory reset**.</span></span>
    
4. <span data-ttu-id="c958e-111">Wählen Sie im Bereich **Sind Sie sicher, dass Sie die Zurücksetzung auf Werkseinstellungen auf den folgenden Geräten durchführen möchten?** nacheinander **Bestätigen** \> **Schließen** aus.</span><span class="sxs-lookup"><span data-stu-id="c958e-111">On the **Are you sure you want to factory reset the devices below** pane, choose **Confirm** \> **Close**.</span></span>
