## <a name="upgrade-to-windows-10-pro"></a>Upgrade auf Windows 10 Pro

Für das Upgrade auf Windows 10 Pro stehen Ihnen mehrere Optionen zur Verfügung. Sie können:

- Installieren Sie das Upgrade von der [Microsoft Software Download-Website.](https://go.microsoft.com/fwlink/?LinkID=836951)
  - Wählen Sie diese Option aus, wenn sich das Gerät, auf dem Sie angemeldet sind, auf demselben Gerät befindet wie das Gerät, das Sie aktualisieren möchten.
  - Klicken Sie auf der Softwaredownloadwebsite **auf "Aktualisieren",** um mit dem Upgrade des Geräts auf Windows 10 Pro Creators Update zu beginnen.

- Erstellen Sie ein Installationsmedium mithilfe des [Medienerstellungstools.](https://go.microsoft.com/fwlink/?LinkID=836960) &ndash; Wählen Sie diese Option aus, um ein Windows 10 Pro Creators Update-Installationsmedium (USB-Speicherstick oder ISO-Datei) zu erstellen, um Windows 10 auf einem PC zu installieren, der sich von dem pc unterscheidet, den Sie verwenden.
  - Lesen Sie die Anweisungen zur Verwendung des Tools und zum Erstellen des Installationsmediums.

> [!NOTE]
> Wenn Sie über Windows Geräte verfügen, auf denen Windows 7 Pro, Windows 8 Pro oder Windows 8.1 Pro ausgeführt wird, erhält Ihr Microsoft 365 for Business-Abonnement ein Upgrade Windows Pro 10.
