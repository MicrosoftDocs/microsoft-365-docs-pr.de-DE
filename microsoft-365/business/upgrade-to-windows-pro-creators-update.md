# <a name="upgrade-windows-devices-to-windows-pro-creators-update"></a>Aktualisieren von Windows-Geräten auf Windows Pro Creators Update

## <a name="upgrade-to-windows-10-pro-creators-update"></a>Upgrade auf Windows 10 pro Creators Update
  
1. Für ein Upgrade auf Windows 10 pro Creators Update stehen Ihnen mehrere Optionen zur Verfügung. Sie können:
    
    - Installieren des Upgrades von der [Microsoft-Software Download Website](https://go.microsoft.com/fwlink/?LinkID=836951 ) – wählen Sie diese Option aus, wenn sich das Gerät, das Sie gerade angemeldet haben, auf dem Gerät befindet, das Sie aktualisieren möchten.
    
      a. Klicken Sie auf der Software Download-Website auf **Jetzt aktualisieren** , um das Upgrade des Geräts auf Windows 10 pro Creators Update zu starten. 
    
     - Erstellen eines Installationsmediums mithilfe des [Medien Erstellungstools](https://go.microsoft.com/fwlink/?LinkID=836960) – wählen Sie diese Option aus, um ein installationsMedium für Windows 10 pro Creators (USB-Flashlaufwerk oder ISO-Datei) zu erstellen, um Windows 10 auf einem PC zu installieren, der sich von dem von Ihnen verwendeten unterscheidet.
    
        a. Lesen Sie die Anweisungen zur Verwendung des Tools und zum Erstellen des Installationsmediums. 

> [!Note]
> Wenn Sie Windows-Geräte mit Windows 7 pro, Windows 8 pro oder Windows 8,1 pro ausführen, berechtigt Sie Ihr Microsoft 365 Business-Abonnement zu einem Windows 10-Upgrade – Sie benötigen keinen Product Key.
    
Weitere Informationen finden Sie unter [Einrichten von Windows-Geräten für Microsoft 365 Business-Benutzer](set-up-windows-devices.md) , um das Einrichten von Windows 10-Geräten abzuschließen. 
  
Weitere Informationen finden Sie unter [Einrichten von mobilen Geräten für Microsoft 365 Business-Benutzer](set-up-mobile-devices.md) , um die Einrichtung von Android-und IOS-Geräten abzuschließen.
