# <a name="compliance-manager-templates-list"></a>Compliance-Manager-Vorlagenliste

**In diesem Artikel:** Zeigen Sie die umfassende Liste der **Vorlagen an,** die zum Erstellen von Bewertungen im Compliance-Manager verfügbar sind.

> [!IMPORTANT]
> Die Bewertungsvorlagen, die Für Ihre Organisation verfügbar sind, hängen von Ihrem Lizenzvertrag ab. [Überprüfen Sie die Details](/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-tenantlevel-services-licensing-guidance/microsoft-365-security-compliance-licensing-guidance).

## <a name="overview"></a>Übersicht

[Microsoft Compliance Manager](compliance-manager.md) bietet eine umfassende Reihe von Vorlagen zum Erstellen von Bewertungen. Diese Vorlagen können Ihre Organisation dabei unterstützen, nationale, regionale und branchenspezifische Anforderungen für die Erfassung und Verwendung von Daten zu erfüllen.

Vorlagen werden dem Compliance Manager hinzugefügt, wenn neue Gesetze und Vorschriften erlassen werden. Compliance Manager aktualisiert seine Vorlagen, wenn sich die zugrunde liegenden Gesetze oder Vorschriften ändern. Erfahren Sie mehr darüber, [wie Sie Updates überprüfen und akzeptieren.](compliance-manager-assessments.md#accepting-updates-to-assessments)

Hier finden Sie detaillierte Anleitungen [zum Arbeiten mit Vorlagen](compliance-manager-templates.md) zum Erstellen Ihrer Bewertungen.

## <a name="list-of-templates-and-where-to-find-them"></a>Liste der Vorlagen und wo sie zu finden sind

Unten finden Sie die vollständige Liste der Vorlagen im Compliance-Manager. Jede Seite mit Vorlagendetails enthält Informationen über die Verordnung oder den Standard, auf die sie angewendet wird. Über die Links in den folgenden Vorlagennamen erhalten Sie eine entsprechende Dokumentation zu diesem Standard, dieser Verordnung oder diesem Recht.

**Wo finden Sie Ihre Vorlagen**

Wechseln Sie im Compliance-Manager zur Seite **Bewertungsvorlagen.** Es wird eine Liste aller Vorlagen angezeigt, die Für Ihre Organisation verfügbar sind. Es ist wichtig, den Unterschied zwischen eingeschlossenen und Premiumvorlagen zu verstehen:

- **Enthaltene Vorlagen** sind Vorlagen, die im Lizenzvertrag Ihrer Organisation enthalten sind.
 - **Premium werden zusätzliche** Vorlagen angezeigt, die Ihre Organisation möglicherweise abrufen möchte (siehe [die Dienstbedingungen](/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-tenantlevel-services-licensing-guidance/microsoft-365-security-compliance-licensing-guidance)).

Weitere Informationen zum [Anzeigen und Verwalten Ihrer Vorlagen](compliance-manager-templates.md#viewing-and-managing-templates-from-the-assessment-templates-page)finden Sie unter .

> [!NOTE]
> Weitere Informationen zu eingeschlossenen und premium-Vorlagen sowie Informationen zu den verfügbaren Vorlagen für Ihre Organisation finden Sie unter [Vorlagentypen.](compliance-manager-templates.md#template-types-included-and-premium-active-and-inactive)
