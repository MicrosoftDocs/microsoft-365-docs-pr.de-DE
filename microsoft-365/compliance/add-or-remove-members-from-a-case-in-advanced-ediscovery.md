# <a name="add-or-remove-members-from-a-case"></a><span data-ttu-id="6d12b-103">Hinzufügen oder Entfernen von Mitgliedern aus einem Fall</span><span class="sxs-lookup"><span data-stu-id="6d12b-103">Add or remove members from a case</span></span>

<span data-ttu-id="6d12b-104">Sie können Mitglieder hinzufügen oder entfernen, um die Benutzer zu verwalten, die auf den Fall zugreifen können.</span><span class="sxs-lookup"><span data-stu-id="6d12b-104">You can add or remove members to manage who can access the case.</span></span> <span data-ttu-id="6d12b-105">Bevor ein Mitglied jedoch auf einen Advanced eDiscovery-Fall zugreifen kann (und in diesem Fall Aufgaben ausführen  kann), müssen Sie den Benutzer der Rollengruppe eDiscovery-Manager auf der Seite Berechtigungen im Security and Compliance Center hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="6d12b-105">However, before a member can access a Advanced eDiscovery case (and perform tasks in the case), you must add the user to the eDiscovery Manager role group on the **Permissions** page in the security and compliance center.</span></span> <span data-ttu-id="6d12b-106">Weitere Informationen finden Sie unter [Assign eDiscovery permissions in the Security & Compliance Center](https://docs.microsoft.com/microsoft-365/compliance/assign-ediscovery-permissions).</span><span class="sxs-lookup"><span data-stu-id="6d12b-106">For more information, see [Assign eDiscovery permissions in the Security & Compliance Center](https://docs.microsoft.com/microsoft-365/compliance/assign-ediscovery-permissions).</span></span>

1. <span data-ttu-id="6d12b-107">Gehen Sie auf der Seite **Advanced eDiscovery** zu dem Fall, dem Sie ein Mitglied hinzufügen möchten.</span><span class="sxs-lookup"><span data-stu-id="6d12b-107">On the **Advanced eDiscovery** page, go to the case that you want to add a member to.</span></span>

2. <span data-ttu-id="6d12b-108">Klicken Sie auf die Registerkarte **Einstellungen**, und klicken Sie dann auf der Kachel **Zugriff & Berechtigungen** auf **Auswählen**.</span><span class="sxs-lookup"><span data-stu-id="6d12b-108">Click the **Settings** tab and then click **Select** in the **Access & permissions** tile.</span></span>

3. <span data-ttu-id="6d12b-109">Klicken Sie auf **Aktualisieren**.</span><span class="sxs-lookup"><span data-stu-id="6d12b-109">Click **Update**.</span></span>

4. <span data-ttu-id="6d12b-110">Klicken Sie unter **Fallbeteiligte verwalten** auf **Hinzufügen**, um Fallbeteiligte hinzuzufügen.</span><span class="sxs-lookup"><span data-stu-id="6d12b-110">Under **Manage members**, click **Add** to add members to the case.</span></span> <span data-ttu-id="6d12b-111">Sie können dem Fall auch eine Rollengruppe hinzufügen, indem Sie unter Verwalten **von** Rollengruppen auf Hinzufügen **klicken.**</span><span class="sxs-lookup"><span data-stu-id="6d12b-111">You can also choose to add a role group to the case by clicking  **Add** under **Manage role groups**.</span></span>

5. <span data-ttu-id="6d12b-112">Aktivieren Sie in der Liste der Personen oder Rollengruppen, die als Fallmitglieder hinzugefügt werden können, das Kontrollkästchen neben den Namen der Personen oder Rollengruppen, die Sie hinzufügen möchten.</span><span class="sxs-lookup"><span data-stu-id="6d12b-112">In the list of people or role groups that can be added as members of the case, select the check box next to the names of the people or role groups that you want to add.</span></span>

   > [!NOTE]
   > <span data-ttu-id="6d12b-113">Beim Hinzufügen einer Rollengruppe zu einem Fall können Sie nur die Rollengruppen hinzufügen, in denen Sie Mitglied sind.</span><span class="sxs-lookup"><span data-stu-id="6d12b-113">When adding a role group to a case, you can only add the role groups that you are a member of.</span></span>

6. <span data-ttu-id="6d12b-114">Nachdem Sie die Personen oder Rollengruppen ausgewählt haben, die als Mitglieder des Falls hinzugefügt werden, klicken Sie auf **Hinzufügen**.</span><span class="sxs-lookup"><span data-stu-id="6d12b-114">After you've selected the people or role groups to add as members of the case, click **Add**.</span></span>

7. <span data-ttu-id="6d12b-115">Klicken Sie auf der Flyout-Seite **Fall verwalten** auf **Speichern**, um die neue Liste der Fallbeteiligten zu speichern.</span><span class="sxs-lookup"><span data-stu-id="6d12b-115">In the **Manage this case** flyout page, click **Save** to save the new list of case members.</span></span>
