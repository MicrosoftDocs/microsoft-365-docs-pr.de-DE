# <a name="add-or-remove-members-from-a-case"></a>Hinzufügen oder Entfernen von Mitgliedern aus einem Fall

Sie können Mitglieder hinzufügen oder entfernen, um die Benutzer zu verwalten, die auf den Fall zugreifen können. Bevor ein Mitglied jedoch auf einen Advanced eDiscovery-Fall zugreifen kann (und in diesem Fall Aufgaben ausführen  kann), müssen Sie den Benutzer der Rollengruppe eDiscovery-Manager auf der Seite Berechtigungen im Security and Compliance Center hinzufügen. Weitere Informationen finden Sie unter [Zuweisen von eDiscovery-Berechtigungen im Security & Compliance Center](./assign-ediscovery-permissions.md).

1. Gehen Sie auf der Seite **Advanced eDiscovery** zu dem Fall, dem Sie ein Mitglied hinzufügen möchten.

2. Klicken Sie auf die Registerkarte **Einstellungen**, und klicken Sie dann auf der Kachel **Zugriff & Berechtigungen** auf **Auswählen**.

3. Klicken Sie auf **Aktualisieren**.

4. Klicken Sie unter **Fallbeteiligte verwalten** auf **Hinzufügen**, um Fallbeteiligte hinzuzufügen. Sie können dem Fall auch eine Rollengruppe hinzufügen, indem Sie unter Verwalten **von** Rollengruppen auf Hinzufügen **klicken.**

5. Aktivieren Sie in der Liste der Personen oder Rollengruppen, die als Fallmitglieder hinzugefügt werden können, das Kontrollkästchen neben den Namen der Personen oder Rollengruppen, die Sie hinzufügen möchten.

   > [!NOTE]
   > Beim Hinzufügen einer Rollengruppe zu einem Fall können Sie nur die Rollengruppen hinzufügen, in denen Sie Mitglied sind.

6. Nachdem Sie die Personen oder Rollengruppen ausgewählt haben, die als Mitglieder des Falls hinzugefügt werden, klicken Sie auf **Hinzufügen**.

7. Klicken Sie auf der Flyout-Seite **Fall verwalten** auf **Speichern**, um die neue Liste der Fallbeteiligten zu speichern.
