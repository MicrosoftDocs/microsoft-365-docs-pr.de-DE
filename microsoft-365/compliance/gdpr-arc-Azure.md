# <a name="azure-accountability-readiness-checklist-for-the-gdpr"></a>Azure-Prüfliste zu den Verantwortlichkeiten für die DSGVO

## <a name="1-introduction"></a>1. Einführung
Diese Prüfliste zu den Verantwortlichkeiten ist eine bequeme Möglichkeit, um auf Informationen zuzugreifen, die Sie bei Verwendung von Microsoft Azure zur Unterstützung der DSGVO unter Umständen benötigen. Die Prüfliste ist basierend auf den Titeln und der Referenznummer (für jedes Prüflistenthema in Klammern) einer Gruppe von Datenschutz- und Sicherheitskontrollen für Verarbeiter von personenbezogenen Daten aufgebaut (gemäß ISO/IEC CD 27552 Information technology – Security techniques – Enhancement to ISO/IEC 27001 for privacy management – Requirements). 

Sie können die Elemente dieser Prüfliste mit dem Compliance-Manager [16] verwalten, indem Sie auf der DSGVO-Kachel unter den vom Kunden verwalteten Kontrollen die Kontroll-ID und den Kontrolltitel verwenden. Diese Kontrollstruktur wird auch genutzt, um die Darstellung der internen Steuerelemente zu organisieren, die von Microsoft Azure zur Unterstützung der DSGVO implementiert werden. Den dazugehörigen Download finden Sie unter https://servicetrust.microsoft.com/ViewPage/TrustDocuments.

Diese Steuerelementstruktur wird auch verwendet, um die Darstellung der internen Steuerelemente zu organisieren, die Microsoft Dynamics365 zur Unterstützung der DSGVO implementiert, die Sie hier herunterladen können:<br>Eine Kopie des vollständigen Entwurfs des ISO-Standards finden Sie unter https://shop.bsigroup.com/ProductDetail?pid=000000000030379002. 

Weitere Dokumentationen im Zusammenhang mit der DSGVO finden Sie unter https://aka.ms/gdprgetstarted.
