# <a name="legacy-ediscovery-tools-being-retired"></a>Ältere eDiscovery-Tools werden zurückgezogen

In der ersten Hälfte von 2020 werden zwei Legacy-eDiscovery-Tools in den Ruhestand gebracht: Advanced eDiscovery v 1.0 und Compliance-eDiscovery und-Aufbewahrung.

Advanced eDiscovery v1 (auch als Office 365 Advanced eDiscovery bezeichnet) ist die Version von Advanced eDiscovery, auf die durch Klicken auf "wechseln zu Advanced eDiscovery" in einem eDiscovery-Fall im Security #a0 Compliance Center zugegriffen wird.  Die Pensionierung von Advance eDiscovery v 1.0 wirkt sich nicht auf die neueste Version von Advanced eDiscovery (auch als Advanced eDiscovery v2 oder Advanced eDiscovery in Microsoft 365) aus, auf die im Security #a1 Compliance Center auf der Seite eDiscovery #a0 Advanced eDiscovery zugegriffen wird.

Das Tool für die Compliance-eDiscovery und-Aufbewahrung im Exchange Admin Center wird ebenfalls zurückgezogen. Dieses Tool wird zum Suchen, halten und Exportieren von Postfachinhalten in Exchange Online verwendet. Ähnliche Funktionen sind im Microsoft 365 Compliance Center verfügbar.

Weitere Informationen zum Ruhestand dieser Legacy-eDiscovery-Tools finden Sie in Kürze.
