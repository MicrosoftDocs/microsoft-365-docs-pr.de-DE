# <a name="how-to-submit-feedback-on-app-governance"></a>So übermitteln Sie Feedback zur App-Governance 

Um Feedback zum App-Governance-Add-On für Microsoft Cloud App Security zu übermitteln:

- Klicken Sie in der unteren rechten Ecke einer beliebigen Portalseite, für die Sie Feedback übermitteln wollen, auf die schwarze Feedback-Schaltfläche.

- Wenn Sie Feedback über Features übermitteln:
  1. Geben Sie eine Sternbewertung und Kommentare in das Textfeld ein.  
  1. Wenn Sie die Option **Diesen Screenshot einschließen** auswählen, wird ein Screenshot der Seite, auf der Sie sich befinden, geteilt werden.  
  1. Wenn Sie die Option **Sie können mich zu diesem Feedback kontaktieren** auswählen, wird Ihre E-Mail-Adresse mit Microsoft geteilt.
  1. Wählen Sie **Übermitteln** aus, um Ihr Feedback zu senden.

- Wenn Sie Feedback über bösartige App-Aktivitäten übermitteln:

  1. Geben Sie eine Fünf-Sterne-Bewertung ab (dies ist ein Pflichtfeld).
  1. Geben Sie im Textfeld den Namen und die ID der App ein, die Sie als bösartig markieren, sowie alle unterstützenden Details, die Sie bereitstellen können.
  1. Wenn Sie die Option **Diesen Screenshot einschließen** auswählen, wird ein Screenshot der Seite, auf der Sie sich befinden, geteilt werden.  
  1. Wenn Sie die Option **Sie können mich zu diesem Feedback kontaktieren** auswählen, wird Ihre E-Mail-Adresse mit Microsoft geteilt.
  1. Wählen Sie **Übermitteln** aus, um Ihr Feedback zu senden.
