# <a name="ediscovery-solutions-in-microsoft-365"></a>eDiscovery-Lösungen in Microsoft 365

Als eDiscovery (electronic discovery) wird das Identifizieren und Übermitteln elektronischer Informationen bezeichnet, die als Indizien in einem Rechtsstreit verwendet werden können. Sie können eDiscovery-Tools in Microsoft 365 verwenden, um nach Inhalten in Exchange Online-Postfächern, Microsoft 365-Gruppen, Microsoft Teams, SharePoint Online- und OneDrive for #A0 sowie Skype for #A1 und Yammer zu suchen. Sie können Postfächer und Websites in derselben eDiscovery-Suche mithilfe des Tools für die Inhaltssuche durchsuchen. Und Sie können Core eDiscovery-Fälle verwenden, um Inhalte in Postfächern und Websites zu identifizieren, zu archivieren und zu exportieren. Wenn Ihre Organisation über ein Office 365 E5- oder Microsoft 365 E5-Abonnement (oder verwandte E5-Add-On-Abonnements) verfügt, können Sie Wahrer weiter verwalten und Inhalte mithilfe der Advanced eDiscovery-Lösung in Microsoft 365 analysieren.
  
Microsoft 365 bietet die folgenden eDiscovery-Tools:
  
- [Inhaltssuche](#content-search)

- [Core eDiscovery](#core-ediscovery)

- [Erweiterte eDiscovery](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>Advanced eDiscovery

Die Advanced eDiscovery-Lösung in Microsoft 365 (auch advanced *eDiscovery v2.0* genannt) basiert auf den vorhandenen eDiscovery- und Analysefunktionen in Microsoft 365. Diese eDiscovery-Lösung bietet einen End-to-End-Workflow zum Beibehalten, Sammeln, Überprüfen, Analysieren und Exportieren von Inhalten, die auf interne und externe Untersuchungen Ihrer Organisation reagieren. Außerdem können Rechtsteams Verwahrer und den gesamten Benachrichtigungsworkflow für die gesetzliche Benachrichtigung verwalten, um mit den an einem Fall beteiligten Verwahrern zu kommunizieren.
