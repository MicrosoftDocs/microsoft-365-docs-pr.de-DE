# <a name="ediscovery-solutions-in-microsoft-365"></a>eDiscovery-Lösungen in Microsoft 365

Als eDiscovery (electronic discovery) wird das Identifizieren und Übermitteln elektronischer Informationen bezeichnet, die als Indizien in einem Rechtsstreit verwendet werden können. Sie können eDiscovery-Tools in Microsoft 365 verwenden, um nach Inhalten in Exchange Online-Postfächern, Microsoft 365-Gruppen, Microsoft Teams-, SharePoint Online- und OneDrive for Business-Websites sowie Skype for Business-Unterhaltungen und Yammer-Teams zu suchen. Sie können Postfächer und Websites in derselben eDiscovery-Suche mithilfe des Inhaltssuchtools durchsuchen. Und Sie können Core eDiscovery-Fälle verwenden, um Inhalte in Postfächern und Websites zu identifizieren, zu archivieren und zu exportieren. Wenn Ihre Organisation über ein Office 365 E5- oder Microsoft 365 E5-Abonnement (oder zugehörige E5-Add-On-Abonnements) verfügt, können Sie Custodians weiter verwalten und Inhalte mithilfe der Advanced eDiscovery-Lösung in Microsoft 365.
  
Microsoft 365 bietet die folgenden eDiscovery-Tools:
  
- [Inhaltssuche](#content-search)

- [Core eDiscovery](#core-ediscovery)

- [Erweiterte eDiscovery](#advanced-ediscovery)

## <a name="advanced-ediscovery"></a>Advanced eDiscovery

Die Advanced eDiscovery lösung in Microsoft 365 (auch *als Advanced eDiscovery v2.0* bezeichnet) baut auf den vorhandenen eDiscovery- und Analysefunktionen in Microsoft 365. Diese eDiscovery-Lösung bietet einen End-to-End-Workflow zum Beibehalten, Sammeln, Überprüfen, Analysieren und Exportieren von Inhalten, die auf interne und externe Untersuchungen Ihrer Organisation reagieren. Außerdem können Rechtsteams Verwahrer und den gesamten Workflow für Benachrichtigungen über gesetzliche Benachrichtigungen verwalten, um mit verwahrern zu kommunizieren, die an einem Fall beteiligt sind.
