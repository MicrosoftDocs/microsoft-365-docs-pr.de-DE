# <a name="recent-updates-to-microsoft-365-compliance-content"></a>Neueste Aktualisierungen für Microsoft 365 Compliance-Inhalte

Ähnlich wie bei den Features des Microsoft 365 Compliance Centers werden auch unsere Hilfeinhalte ständig weiterentwickelt. Wir erstellen kontinuierlich neue Artikel, aktualisieren bestehende und nehmen daran Änderungen basierend auf Ihrem Feedback vor. Sehen Sie sich die Neuerungen und Updates in diesem Monat an.

> [!TIP]
> Die neuesten Feature-Updates im Microsoft 365 Compliance Center finden Sie unter [Neuerungen im Microsoft 365 Compliance Center](whats-new.md).

## <a name="february-2020"></a>Februar 2020

### <a name="auditing"></a>Überwachung

[Erweiterte Überwachung in Microsoft 365](advanced-audit.md) (neu)<br>Die erweiterte Überwachung ist für Organisationen mit einem Office 365 E5- oder Microsoft 365 Enterprise E5-Abonnement verfügbar. Sie erweitert vorhandene Überwachungsfunktionen durch Features wie lange Aufbewahrungszeiträume für Überwachungsprotokolle, neue Aufbewahrungsrichtlinien für Überwachungsprotokolle und eine neue Postfachüberwachungsaktion zum Nachverfolgen von E-Mail-Lesevorgängen.

[Verwalten von Aufbewahrungsrichtlinien für Überwachungsprotokolle](audit-log-retention-policies.md) (neu)<br>Details zur Verwaltung von Aufbewahrungsrichtlinien für Überwachungsprotokolle, dem neuen erweiterten Überwachungsfeature, das es Ihnen ermöglicht, Überwachungsdatensätze aus unterschiedlichen Diensten für bis zu ein Jahr zu speichern.

[Verwalten der Postfachüberwachung](enable-mailbox-auditing.md#logon-types-and-mailbox-actions) (aktualisiert)<br>Es wurden Informationen zur neuen MailItemsAccessed-Postfachaktion hinzugefügt, die mit der erweiterten Überwachung eingeführt wurde.
