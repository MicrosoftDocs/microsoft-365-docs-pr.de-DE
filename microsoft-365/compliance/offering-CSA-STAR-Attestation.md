## <a name="csa-star-attestation-overview"></a><span data-ttu-id="94587-105">Übersicht über den CSA STAR-Nachweis</span><span class="sxs-lookup"><span data-stu-id="94587-105">CSA STAR self-assessment overview</span></span>

<span data-ttu-id="94587-106">Die Cloud Security Alliance (CSA) verwaltet das Security, Trust & Assurance Registry (STAR)-Programm, ein kostenloses, öffentliches Verzeichnis, in dem Clouddienstanbieter (Cloud Service Provider, CSPs) ihre CSA-bezogenen Bewertungen veröffentlichen können.</span><span class="sxs-lookup"><span data-stu-id="94587-106">The Cloud Security Alliance (CSA) maintains the Security, Trust & Assurance Registry (STAR), a free, publicly accessible registry where cloud service providers (CSPs) can publish their CSA-related assessments.</span></span> <span data-ttu-id="94587-107">STAR umfasst drei Assurance-Stufen, die den Kontrollvorgaben der CSA Cloud Controls Matrix (CCM) entsprechen.</span><span class="sxs-lookup"><span data-stu-id="94587-107">STAR consists of three levels of assurance aligned with control objectives in the CSA Cloud Controls Matrix (CCM).</span></span> <span data-ttu-id="94587-108">(Die CCM deckt grundlegende Sicherheitsprinzipien in 16 Bereichen ab und unterstützt Cloudkunden bei der Bewertung des Gesamtsicherheitsrisikos eines Cloud Services.):</span><span class="sxs-lookup"><span data-stu-id="94587-108">(The CCM covers fundamental security principles across 16 domains to help cloud customers assess the overall security risk of a cloud service.):</span></span>

- <span data-ttu-id="94587-109">Stufe 1: STAR-Selbstbewertung</span><span class="sxs-lookup"><span data-stu-id="94587-109">Level 1: STAR Self-Assessment</span></span>
- <span data-ttu-id="94587-110">Stufe 2: STAR-Nachweis, STAR-Zertifizierung und C-STAR-Bewertung (die auf Prüfungen durch Dritte basieren)</span><span class="sxs-lookup"><span data-stu-id="94587-110">Level 2: STAR Attestation, STAR Certification, and C-STAR Assessment (which are based on audits by third parties)</span></span>
- <span data-ttu-id="94587-111">Stufe 3: STAR – Kontinuierliche Überwachung (Programmanforderungen zurzeit noch in der Entwicklung durch die CSA)</span><span class="sxs-lookup"><span data-stu-id="94587-111">Level 3: STAR Continuous Monitoring (program requirements are still under development by CSA)</span></span>

<span data-ttu-id="94587-112">Für den STAR-Nachweis ist eine strenge, unabhängige Prüfung der Sicherheitslage eines Cloudanbieters basierend auf einer SOC 2 Typ 2-Prüfung mit den CCM-Kriterien erforderlich.</span><span class="sxs-lookup"><span data-stu-id="94587-112">STAR Attestation involves a rigorous independent audit of a cloud provider’s security posture based on a SOC 2 Type 2 audit with CCM criteria.</span></span> <span data-ttu-id="94587-113">Der unabhängige Prüfer, der die Angebote eines Cloudanbieters im Hinblick auf den STAR-Nachweis bewertet, muss ein Certified Public Accountant (CPA) sein und mit dem CSA Certificate in Cloud Security Knowledge (CCSK) ausgezeichnet sein.</span><span class="sxs-lookup"><span data-stu-id="94587-113">The independent auditor that evaluates a cloud provider’s offerings for STAR Attestation must be a certified public accountant (CPA) and is required to have the CSA Certificate in Cloud Security Knowledge (CCSK).</span></span>  
  
<span data-ttu-id="94587-114">Eine SOC 2 Typ 2-Prüfung basiert auf den American Institute of Certified Public Accountants (AICPA) Trust Services Principles and Criteria, einschließlich Sicherheit, Verfügbarkeit, Vertraulichkeit und Integritätsverarbeitung, sowie auf den Kriterien der CCM.</span><span class="sxs-lookup"><span data-stu-id="94587-114">A SOC 2 Type 2 audit is based on American Institute of Certified Public Accountants (AICPA) Trust Services Principles and Criteria, including security, availability, confidentiality, and processing integrity, and the criteria in the CCM.</span></span> <span data-ttu-id="94587-115">Der STAR-Nachweis stellt die Ergebnisse eines Prüfers im Hinblick auf die Designeignung und Betriebseffektivität der SOC 2-Kontrollen in Microsoft Cloud Services bereit.</span><span class="sxs-lookup"><span data-stu-id="94587-115">STAR Attestation provides an auditor’s findings on the design suitability and operating effectiveness of SOC 2 controls in Microsoft cloud services.</span></span> <span data-ttu-id="94587-116">Das Ziel besteht darin, sowohl die oben beschriebenen AICPA-Kriterien als auch die CCM-Anforderungen zu erfüllen.</span><span class="sxs-lookup"><span data-stu-id="94587-116">The objective is to meet both the AICPA criteria mentioned above and requirements set forth in the CCM.</span></span>
